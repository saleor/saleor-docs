Look up a shipping zone by ID.

Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
shippingZone(
  id: ID!
  channel: String
): ShippingZone
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>shippingZone.<b>id</b></code>](#)<Bullet />[`ID!`](../../../storefront/miscellaneous/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> ID of the shipping zone.

#### [<code style={{ fontWeight: 'normal' }}>shippingZone.<b>channel</b></code>](#)<Bullet />[`String`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of a channel for which the data should be returned.

### Type

#### [`ShippingZone`](../../../storefront/shipping/objects/shipping-zone) <Badge class="secondary" text="object"/>

> Represents a shipping zone in the shop. Zones are the concept used only for grouping shipping methods in the dashboard, and are never exposed to the customers directly.
