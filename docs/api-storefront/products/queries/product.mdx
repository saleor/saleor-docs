Look up a product by ID. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
product(
  id: ID
  slug: String
  externalReference: String
  channel: String
): Product
```
