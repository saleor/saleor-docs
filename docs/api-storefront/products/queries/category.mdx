Look up a category by ID or slug.

```graphql
category(
  id: ID
  slug: String
): Category
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>category.<b>id</b></code>](#)<Bullet />[`ID`](../../../storefront/miscellaneous/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of the category.

#### [<code style={{ fontWeight: 'normal' }}>category.<b>slug</b></code>](#)<Bullet />[`String`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of the category

### Type

#### [`Category`](../../../storefront/products/objects/category) <Badge class="secondary" text="object"/>

> Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.
