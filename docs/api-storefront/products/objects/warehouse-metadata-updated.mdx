Added in Saleor 3.8.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

```graphql
type WarehouseMetadataUpdated implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  warehouse: Warehouse
}
```
