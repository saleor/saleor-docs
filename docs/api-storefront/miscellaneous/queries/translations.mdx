Returns a list of all translatable items of a given kind.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
translations(
  kind: TranslatableKinds!
  before: String
  after: String
  first: Int
  last: Int
): TranslatableItemConnection
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>translations.<b>kind</b></code>](#)<Bullet />[`TranslatableKinds!`](../../../storefront/miscellaneous/enums/translatable-kinds) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="enum"/>

> Kind of objects to retrieve.

#### [<code style={{ fontWeight: 'normal' }}>translations.<b>before</b></code>](#)<Bullet />[`String`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/>

> Return the elements in the list that come before the specified cursor.

#### [<code style={{ fontWeight: 'normal' }}>translations.<b>after</b></code>](#)<Bullet />[`String`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/>

> Return the elements in the list that come after the specified cursor.

#### [<code style={{ fontWeight: 'normal' }}>translations.<b>first</b></code>](#)<Bullet />[`Int`](../../../storefront/miscellaneous/scalars/int) <Badge class="secondary" text="scalar"/>

> Retrieve the first n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

#### [<code style={{ fontWeight: 'normal' }}>translations.<b>last</b></code>](#)<Bullet />[`Int`](../../../storefront/miscellaneous/scalars/int) <Badge class="secondary" text="scalar"/>

> Retrieve the last n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.
