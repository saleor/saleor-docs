Initializes a transaction session. It triggers the webhook `TRANSACTION_INITIALIZE_SESSION`, to the requested `paymentGateways`.

```graphql
transactionInitialize(
  action: TransactionFlowStrategyEnum
  amount: PositiveDecimal
  customerIpAddress: String
  id: ID!
  idempotencyKey: String
  paymentGateway: PaymentGatewayToInitialize!
): TransactionInitialize
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>action</b></code>](#)<Bullet />[`TransactionFlowStrategyEnum`](../../../api-storefront/payments/enums/transaction-flow-strategy-enum) <Badge class="badge badge--secondary" text="enum"/> <Badge class="badge badge--secondary" text="payments"/>

> The expected action called for the transaction. By default, the `channel.paymentSettings.defaultTransactionFlowStrategy` will be used.The field can be used only by app that has `HANDLE_PAYMENTS` permission.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>amount</b></code>](#)<Bullet />[`PositiveDecimal`](../../../api-storefront/miscellaneous/scalars/positive-decimal) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

> The amount requested for initializing the payment gateway. If not provided, the difference between checkout.total - transactions that are already processed will be send.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>customerIpAddress</b></code>](#)<Bullet />[`String`](../../../api-storefront/miscellaneous/scalars/string) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

> The customer's IP address. If not provided Saleor will try to determine the customer's IP address on its own. The customer's IP address will be passed to the payment app. The IP should be in ipv4 or ipv6 format. The field can be used only by an app that has `HANDLE_PAYMENTS` permission.
>
> <Badge text="Added in Saleor 3.16" class="secondary margin-bottom--sm" />

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>id</b></code>](#)<Bullet />[`ID!`](../../../api-storefront/miscellaneous/scalars/id) <Badge class="badge badge--secondary" text="non-null"/> <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

> The ID of the checkout or order.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>idempotencyKey</b></code>](#)<Bullet />[`String`](../../../api-storefront/miscellaneous/scalars/string) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

> The idempotency key assigned to the action. It will be passed to the payment app to discover potential duplicate actions. If not provided, the default one will be generated. If empty string provided, INVALID error code will be raised.
>
> <Badge text="Added in Saleor 3.14" class="secondary margin-bottom--sm" />

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>paymentGateway</b></code>](#)<Bullet />[`PaymentGatewayToInitialize!`](../../../api-storefront/payments/inputs/payment-gateway-to-initialize) <Badge class="badge badge--secondary" text="non-null"/> <Badge class="badge badge--secondary" text="input"/> <Badge class="badge badge--secondary" text="payments"/>

> Payment gateway used to initialize the transaction.
