Look up a transaction by ID.

Added in Saleor 3.6.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: HANDLE_PAYMENTS.

```graphql
transaction(
  id: ID!
): TransactionItem
```

#### [<code style={{ fontWeight: 'normal' }}>transaction.<b>id</b></code>](#)<Bullet />[`ID!`](../../../storefront/miscellaneous/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> ID of a transaction.

### Type
