Sends an email with the account password modification link.

```graphql
requestPasswordReset(
  channel: String
  email: String!
  redirectUrl: String!
): RequestPasswordReset
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>requestPasswordReset.<b>channel</b></code>](#)<Bullet />[`String`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of a channel which will be used for notify user. Optional when only one channel exists.

#### [<code style={{ fontWeight: 'normal' }}>requestPasswordReset.<b>email</b></code>](#)<Bullet />[`String!`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> Email of the user that will be used for password recovery.

#### [<code style={{ fontWeight: 'normal' }}>requestPasswordReset.<b>redirectUrl</b></code>](#)<Bullet />[`String!`](../../../storefront/miscellaneous/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> URL of a view where users should be redirected to reset the password. URL in RFC 1808 format.

### Type

#### [`RequestPasswordReset`](../../../storefront/users/objects/request-password-reset) <Badge class="secondary" text="object"/>

> Sends an email with the account password modification link.
