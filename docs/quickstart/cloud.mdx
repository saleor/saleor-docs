This guide will help you set up a Saleor storefront in your local environment.

<S>
    <S.Step>
        <S.Details>
            ### Get backend instance
            To quickly get started with the backend, use a free developer account at Saleor Cloud.
            Alternatively you can run Saleor locally using docker.
        </S.Details>

    </S.Step>
    <S.Step>
        <S.Details>
            ### Install dependencies
            Install or update to the latest version of the Saleor CLI by running the following command:

        </S.Details>
        <S.Code>
            ```js
            npm i -g @saleor/cli@latest
            ```
        </S.Code>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Copy variables
            Copy environment variables from .env.example to .env:
        </S.Details>
        <S.Code>
            ```bash
            cp .env.example .env
            ```
        </S.Code>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Add backend address

            Make sure to add trailing slash to the backend address:
        </S.Details>
        <S.Code>
            ```bash
            # Example
            NEXT_PUBLIC_SALEOR_API_URL=http://localhost:8000/graphql/
            ```
        </S.Code>
    </S.Step>

</S>
