import S from "@site/components/Steps/";

<S.Column>
  Saleor Cloud offers free sandbox environments for developers to explore the
  Saleor platform without needing credit card information or trial periods.
</S.Column>

<S>
    <S.Step>
        <S.Details>
            ### Signup
            Create an account at [Saleor Cloud](https://cloud.saleor.io/signup).
        </S.Details>

    </S.Step>
    <S.Step>
        <S.Details>
            ### Create a sandbox
            Create a store closest to the location of your project.
        </S.Details>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Explore the dashboard
            The dashboard will be populated with sample datasets to make it easier to explore the features.
            At this stage, you have a ready-to-use Saleor store that you can use to follow the other quickstart guides.
        </S.Details>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Create a snapshot
            1. Try to modify any data (for example, deleting or editing a product) and create a snapshot of the store to save the current state of the store.
            2. In the sandbox details page, use **Create snapshot**.
            3. Give the snapshot a name **Staging snapshot**.
        </S.Details>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Clear data
            We can completely erase data from the DB to start from scratch.

            In the sandbox **Danger** settings, use **Clear database**.

            This will remove all data from the database, making the dashboard empty as a result.
        </S.Details>
    </S.Step>

    <S.Step>
        <S.Details>
            ### Use snapshot
            Use **Restore from snapshot** to back the data we saved in the previous step.

            Snapshots can be used across different sandboxes. You can now also create a new sandbox using the snapshot.

            Go to **Project** -> **New sandbox** -> **Create from snapshot**.

        </S.Details>
    </S.Step>

</S>

### Next steps
