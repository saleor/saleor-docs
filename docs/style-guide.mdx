## Code formatting

### GraphQL

Lines from 2 to 5 should be highlighted.

```graphql {2-5}
mutation {
  checkoutShippingMethodUpdate(
    checkoutId: "Q2hlY2tvdXQ6ZmE5ZjBkMjYtMWM3NC00MDgyLTk3MzktYTIxOGE2NzVjMDZk"
    shippingMethodId: "U2hpcHBpbmdNZXRob2Q6MTM="
  ) {
    checkout {
      id
      shippingMethod {
        name
      }
      totalPrice {
        gross {
          amount
          currency
        }
      }
    }
    checkoutErrors {
      field
      message
    }
  }
}
```

| Tables        |      Are      |   Cool |
