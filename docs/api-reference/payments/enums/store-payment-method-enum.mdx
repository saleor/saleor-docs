Enum representing the type of a payment storage in a gateway.

```graphql
enum StorePaymentMethodEnum {
  ON_SESSION
  OFF_SESSION
  NONE
}
```

### Values

#### [<code style={{ fontWeight: 'normal' }}>StorePaymentMethodEnum.<b>ON_SESSION</b></code>](#)

> On session storage type. The payment is stored only to be reused when the customer is present in the checkout flow.

#### [<code style={{ fontWeight: 'normal' }}>StorePaymentMethodEnum.<b>OFF_SESSION</b></code>](#)

> Off session storage type. The payment is stored to be reused even if the customer is absent.

#### [<code style={{ fontWeight: 'normal' }}>StorePaymentMethodEnum.<b>NONE</b></code>](#)

> Storage is disabled. The payment is not stored.
