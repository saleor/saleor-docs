Determine the transaction flow strategy.

    AUTHORIZATION - the processed transaction should be only authorized
    CHARGE - the processed transaction should be charged.

```graphql
enum TransactionFlowStrategyEnum {
  AUTHORIZATION
  CHARGE
}
```
