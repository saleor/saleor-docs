### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of a transaction. Either it or token is required to fetch the transaction data.

#### [<code>token</code>](#)<Bullet />[<code>UUID</code>](/api-reference/miscellaneous/scalars/uuid.mdx)

Token of a transaction. Either it or ID is required to fetch the transaction data.

### Type

#### [<code>TransactionItem</code>](/api-reference/payments/objects/transaction-item.mdx)

Represents a payment transaction.
