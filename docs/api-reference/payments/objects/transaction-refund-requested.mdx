Event sent when transaction refund is requested.

```graphql
type TransactionRefundRequested implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  transaction: TransactionItem
  action: TransactionAction!
  grantedRefund: OrderGrantedRefund
}
```

#### [`issuedAt`](#)<Bullet />[`DateTime`](/api-reference/miscellaneous/scalars/date-time.mdx)

Time of the event.

#### [`version`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Saleor version that triggered the event.

#### [`issuingPrincipal`](#)<Bullet />[`IssuingPrincipal`](/api-reference/miscellaneous/unions/issuing-principal.mdx)
