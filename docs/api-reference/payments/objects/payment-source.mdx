Represents a payment source stored for user in payment gateway, such as credit card.

```graphql
type PaymentSource {
  gateway: String!
  paymentMethodId: String
  creditCardInfo: CreditCard
  metadata: [MetadataItem!]!
}
```

### Fields

#### [`gateway`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Payment gateway name.

#### [`paymentMethodId`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

ID of stored payment method.

#### [`creditCardInfo`](#)<Bullet />[`CreditCard`](/api-reference/payments/objects/credit-card.mdx)

Stored credit card details if available.
