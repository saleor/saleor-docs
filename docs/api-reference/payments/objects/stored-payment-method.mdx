Represents a payment method stored for user (tokenized) in payment gateway.

```graphql
type StoredPaymentMethod {
  id: ID!
  gateway: PaymentGateway!
  paymentMethodId: String!
  creditCardInfo: CreditCard
  supportedPaymentFlows: [TokenizedPaymentFlowEnum!]
  type: String!
  name: String
  data: JSON
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

Stored payment method ID.

#### [`gateway`](#)<Bullet />[`PaymentGateway!`](/api-reference/payments/objects/payment-gateway.mdx)

Payment gateway that stores this payment method.

#### [`paymentMethodId`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

ID of stored payment method used to make payment actions. Note: method ID is unique only within the payment gateway.

#### [`creditCardInfo`](#)<Bullet />[`CreditCard`](/api-reference/payments/objects/credit-card.mdx)

Stored credit card details if available.

#### [`supportedPaymentFlows`](#)<Bullet />[`[TokenizedPaymentFlowEnum!]`](/api-reference/payments/enums/tokenized-payment-flow-enum.mdx)

#### [`type`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Type of the payment method. Example: credit card, wallet, etc.

#### [`name`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Payment method name. Example: last 4 digits of credit card, obfuscated email, etc.

#### [`data`](#)<Bullet />[`JSON`](/api-reference/miscellaneous/scalars/json.mdx)

JSON data returned by Payment Provider app for this payment method.
