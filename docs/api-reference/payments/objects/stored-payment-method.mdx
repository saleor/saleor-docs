Represents a payment method stored for user (tokenized) in payment gateway.

```graphql
type StoredPaymentMethod {
  id: ID!
  gateway: PaymentGateway!
  paymentMethodId: String!
  creditCardInfo: CreditCard
  supportedPaymentFlows: [TokenizedPaymentFlowEnum!]
  type: String!
  name: String
  data: JSON
}
```

### Fields

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

Stored payment method ID.

#### [<code>gateway</code>](#)<Bullet />[<code>PaymentGateway!</code>](/api-reference/payments/objects/payment-gateway.mdx)

Payment gateway that stores this payment method.

#### [<code>paymentMethodId</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

ID of stored payment method used to make payment actions. Note: method ID is unique only within the payment gateway.

#### [<code>creditCardInfo</code>](#)<Bullet />[<code>CreditCard</code>](/api-reference/payments/objects/credit-card.mdx)

Stored credit card details if available.

#### [<code>supportedPaymentFlows</code>](#)<Bullet />[<code>[TokenizedPaymentFlowEnum!]</code>](/api-reference/payments/enums/tokenized-payment-flow-enum.mdx)

#### [<code>type</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

Type of the payment method. Example: credit card, wallet, etc.

#### [<code>name</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Payment method name. Example: last 4 digits of credit card, obfuscated email, etc.

#### [<code>data</code>](#)<Bullet />[<code>JSON</code>](/api-reference/miscellaneous/scalars/json.mdx)

JSON data returned by Payment Provider app for this payment method.
