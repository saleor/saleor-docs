```graphql
transactionInitialize(
  action: TransactionFlowStrategyEnum
  amount: PositiveDecimal
  id: ID!
  paymentGateway: PaymentGatewayToInitialize!
): TransactionInitialize
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>action</b></code>](#)<Bullet />[`TransactionFlowStrategyEnum`](../../../api-reference/payments/enums/transaction-flow-strategy-enum) <Badge class="secondary" text="enum"/>

> The expected action called for the transaction. By default, the `channel.defaultTransactionFlowStrategy` will be used. The field can be used only by app that has `HANDLE_PAYMENTS` permission.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>amount</b></code>](#)<Bullet />[`PositiveDecimal`](../../../api-reference/miscellaneous/scalars/positive-decimal) <Badge class="secondary" text="scalar"/>

> The amount requested for initializing the payment gateway. If not provided, the difference between checkout.total - transactions that are already processed will be send.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>id</b></code>](#)<Bullet />[`ID!`](../../../api-reference/miscellaneous/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> The ID of the checkout or order.

#### [<code style={{ fontWeight: 'normal' }}>transactionInitialize.<b>paymentGateway</b></code>](#)<Bullet />[`PaymentGatewayToInitialize!`](../../../api-reference/payments/inputs/payment-gateway-to-initialize) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>

> Payment gateway used to initialize the transaction.
