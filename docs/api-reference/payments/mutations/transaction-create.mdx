Creates transaction for checkout or order.

Requires one of the following permissions: HANDLE_PAYMENTS.

```graphql
transactionCreate(
  id: ID!
  transaction: TransactionCreateInput!
  transactionEvent: TransactionEventInput
): TransactionCreate
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the checkout or order.

#### [`transaction`](#)<Bullet />[`TransactionCreateInput!`](/api-reference/payments/inputs/transaction-create-input.mdx)

Input data required to create a new transaction object.

#### [`transactionEvent`](#)<Bullet />[`TransactionEventInput`](/api-reference/payments/inputs/transaction-event-input.mdx)

Data that defines a transaction event.
