Initializes payment process when it is required by gateway.

```graphql
paymentInitialize(
  channel: String
  gateway: String!
  paymentData: JSONString
): PaymentInitialize
```

### Arguments

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [`gateway`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

A gateway name used to initialize the payment.

#### [`paymentData`](#)<Bullet />[`JSONString`](/api-reference/miscellaneous/scalars/jsonstring.mdx)

Client-side generated data required to initialize the payment.

### Type

#### [`PaymentInitialize`](/api-reference/payments/objects/payment-initialize.mdx)

Initializes payment process when it is required by gateway.
