Initializes payment process when it is required by gateway.

```graphql
paymentInitialize(
  channel: String
  gateway: String!
  paymentData: JSONString
): PaymentInitialize
```

### Arguments

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [<code>gateway</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

A gateway name used to initialize the payment.

#### [<code>paymentData</code>](#)<Bullet />[<code>JSONString</code>](/api-reference/miscellaneous/scalars/jsonstring.mdx)

Client-side generated data required to initialize the payment.

### Type

#### [<code>PaymentInitialize</code>](/api-reference/payments/objects/payment-initialize.mdx)

Initializes payment process when it is required by gateway.
