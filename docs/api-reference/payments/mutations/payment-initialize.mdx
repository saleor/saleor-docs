Initializes payment process when it is required by gateway.

```graphql

  channel: String
  gateway: String!
  paymentData: JSONString
): PaymentInitialize
```

#### [<code>channel</code>](#channel)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#channel\}

A gateway name used to initialize the payment.
