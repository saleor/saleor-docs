### Arguments

#### [<code>actionType</code>](#)<Bullet />[<code>TransactionActionEnum!</code>](/api-reference/payments/enums/transaction-action-enum.mdx)

Determines the action type.

#### [<code>amount</code>](#)<Bullet />[<code>PositiveDecimal</code>](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Transaction request amount. If empty, maximal possible amount will be used.

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the transaction. One of field id or token is required.

#### [<code>refundReason</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Reason of the refund
