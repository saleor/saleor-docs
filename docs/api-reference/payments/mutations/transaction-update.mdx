### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the transaction. One of field id or token is required.

#### [<code>token</code>](#)<Bullet />[<code>UUID</code>](/api-reference/miscellaneous/scalars/uuid.mdx)

The token of the transaction. One of field id or token is required.

#### [<code>transaction</code>](#)<Bullet />[<code>TransactionUpdateInput</code>](/api-reference/payments/inputs/transaction-update-input.mdx)

Input data required to create a new transaction object.

#### [<code>transactionEvent</code>](#)<Bullet />[<code>TransactionEventInput</code>](/api-reference/payments/inputs/transaction-event-input.mdx)

Data that defines a transaction transaction.
