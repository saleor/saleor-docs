### Fields

#### [<code>gateway</code>](#gateway)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#gateway\}

A gateway to use with that payment.

#### [<code>token</code>](#token)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#token\}

Client-side generated payment token, representing customer's billing data in a secure manner.

#### [<code>amount</code>](#amount)<Bullet />[<code>PositiveDecimal</code>](/api-reference/miscellaneous/scalars/positive-decimal.mdx) \{#amount\}

Total amount of the transaction, including all taxes and discounts. If no amount is provided, the checkout total will be used.

#### [<code>returnUrl</code>](#return-url)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#return-url\}

URL of a storefront view where user should be redirected after requiring additional actions. Payment with additional actions will not be finished if this field is not provided.

#### [<code>storePaymentMethod</code>](#store-payment-method)<Bullet />[<code>StorePaymentMethodEnum</code>](/api-reference/payments/enums/store-payment-method-enum.mdx) \{#store-payment-method\}

Payment store type.

#### [<code>metadata</code>](#metadata)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx) \{#metadata\}

User public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
