### Fields

#### [`gateway`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

A gateway to use with that payment.

#### [`token`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Client-side generated payment token, representing customer's billing data in a secure manner.

#### [`amount`](#)<Bullet />[`PositiveDecimal`](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Total amount of the transaction, including all taxes and discounts. If no amount is provided, the checkout total will be used.

#### [`returnUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

URL of a storefront view where user should be redirected after requiring additional actions. Payment with additional actions will not be finished if this field is not provided.

#### [`storePaymentMethod`](#)<Bullet />[`StorePaymentMethodEnum`](/api-reference/payments/enums/store-payment-method-enum.mdx)

Payment store type.

#### [`metadata`](#)<Bullet />[`[MetadataInput!]`](/api-reference/miscellaneous/inputs/metadata-input.mdx)

User public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
