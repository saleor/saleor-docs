```graphql
input PaymentInput {
  gateway: String!
  token: String
  amount: PositiveDecimal
  returnUrl: String
  storePaymentMethod: StorePaymentMethodEnum
  metadata: [MetadataInput!]
}
```

### Fields

#### [<code>gateway</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

A gateway to use with that payment.

#### [<code>token</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Client-side generated payment token, representing customer's billing data in a secure manner.

#### [<code>amount</code>](#)<Bullet />[<code>PositiveDecimal</code>](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Total amount of the transaction, including all taxes and discounts. If no amount is provided, the checkout total will be used.

#### [<code>returnUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

URL of a storefront view where user should be redirected after requiring additional actions. Payment with additional actions will not be finished if this field is not provided.

#### [<code>storePaymentMethod</code>](#)<Bullet />[<code>StorePaymentMethodEnum</code>](/api-reference/payments/enums/store-payment-method-enum.mdx)

Payment store type.

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

User public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
