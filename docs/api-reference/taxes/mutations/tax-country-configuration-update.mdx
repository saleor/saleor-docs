```graphql
taxCountryConfigurationUpdate(
  countryCode: CountryCode!
  updateTaxClassRates: [TaxClassRateInput!]!
): TaxCountryConfigurationUpdate
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>taxCountryConfigurationUpdate.<b>countryCode</b></code>](#)<Bullet />[`CountryCode!`](../../../api-reference/miscellaneous/enums/country-code) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="enum"/>

> Country in which to update the tax class rates.

#### [<code style={{ fontWeight: 'normal' }}>taxCountryConfigurationUpdate.<b>updateTaxClassRates</b></code>](#)<Bullet />[`[TaxClassRateInput!]!`](../../../api-reference/taxes/inputs/tax-class-rate-input) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>

> List of tax rates per tax class to update. When `&#x007B;taxClass: id, rate: null`&#x007D; is passed, it deletes the rate object for given taxClass ID. When `&#x007B;rate: Int&#x007D;` is passed without a tax class, it updates the default tax class for this country.
