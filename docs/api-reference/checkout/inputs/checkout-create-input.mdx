### Fields

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel in which to create a checkout.

#### [<code>lines</code>](#)<Bullet />[<code>[CheckoutLineInput!]!</code>](/api-reference/checkout/inputs/checkout-line-input.mdx)

A list of checkout lines, each containing information about an item in the checkout.

#### [<code>email</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The customer's email address.

#### [<code>saveShippingAddress</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Indicates whether the shipping address should be saved to the user’s address book upon checkout completion.Can only be set when a shipping address is provided. If not specified along with the address, the default behavior is to save the address.

<Badge
  text="Added in Saleor 3.21"
  class="badge badge--secondary margin-bottom--sm"
/>

#### [<code>shippingAddress</code>](#)<Bullet />[<code>AddressInput</code>](/api-reference/miscellaneous/inputs/address-input.mdx)

The mailing address to where the checkout will be shipped. Note: the address will be ignored if the checkout doesn't contain shippable items. `skipValidation` requires HANDLE_CHECKOUTS and AUTHENTICATED_APP permissions.

#### [<code>saveBillingAddress</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Indicates whether the billing address should be saved to the user’s address book upon checkout completion. Can only be set when a billing address is provided. If not specified along with the address, the default behavior is to save the address.

<Badge
  text="Added in Saleor 3.21"
  class="badge badge--secondary margin-bottom--sm"
/>

#### [<code>billingAddress</code>](#)<Bullet />[<code>AddressInput</code>](/api-reference/miscellaneous/inputs/address-input.mdx)

Billing address of the customer. `skipValidation` requires HANDLE_CHECKOUTS and AUTHENTICATED_APP permissions.

#### [<code>languageCode</code>](#)<Bullet />[<code>LanguageCodeEnum</code>](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Checkout language code.

#### [<code>validationRules</code>](#)<Bullet />[<code>CheckoutValidationRules</code>](/api-reference/checkout/inputs/checkout-validation-rules.mdx)

The checkout validation rules that can be changed.

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Checkout public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

<Badge
  text="Added in Saleor 3.21"
  class="badge badge--secondary margin-bottom--sm"
/>

#### [<code>privateMetadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Checkout private metadata. Requires one of the following permissions: MANAGE_CHECKOUTS, HANDLE_CHECKOUTS

Requires permissions to modify and to read the metadata of the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
