### Fields

#### [<code>quantity</code>](#)<Bullet />[<code>Int!</code>](/api-reference/miscellaneous/scalars/int.mdx)

The number of items purchased.

#### [<code>variantId</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of the product variant.

#### [<code>price</code>](#)<Bullet />[<code>PositiveDecimal</code>](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Custom price of the item. Can be set only by apps with `HANDLE_CHECKOUTS` permission. When the line with the same variant will be provided multiple times, the last price will be used.

#### [<code>forceNewLine</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Flag that allow force splitting the same variant into multiple lines by skipping the matching logic.

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the object's metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
