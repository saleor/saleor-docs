Updates customer note in the existing checkout object.

Added in Saleor 3.21.

Triggers the following webhook events:

- CHECKOUT_UPDATED (async): A checkout was updated.

```graphql
checkoutCustomerNoteUpdate(
  customerNote: String!
  id: ID!
): CheckoutCustomerNoteUpdate
```
