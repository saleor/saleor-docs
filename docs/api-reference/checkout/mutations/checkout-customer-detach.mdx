Removes the user assigned as the owner of the checkout.

Requires one of the following permissions: AUTHENTICATED_APP, AUTHENTICATED_USER.

```graphql
checkoutCustomerDetach(
  checkoutId: ID
  id: ID
  token: UUID
): CheckoutCustomerDetach
```
