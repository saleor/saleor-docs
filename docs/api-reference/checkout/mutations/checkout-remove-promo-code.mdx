Remove a gift card or a voucher from a checkout.

Triggers the following webhook events:

- CHECKOUT_UPDATED (async): A checkout was updated.

```graphql

  checkoutId: ID
  id: ID
  promoCode: String
  promoCodeId: ID
  token: UUID
): CheckoutRemovePromoCode
```
