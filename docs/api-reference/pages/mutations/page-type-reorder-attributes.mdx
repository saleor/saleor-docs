Reorder the attributes of a page type.

Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
pageTypeReorderAttributes(
  moves: [ReorderInput!]!
  pageTypeId: ID!
): PageTypeReorderAttributes
```

### Arguments

#### [`moves`](#)<Bullet />[`[ReorderInput!]!`](/api-reference/miscellaneous/inputs/reorder-input.mdx)

The list of attribute reordering operations.

#### [`pageTypeId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a page type.
