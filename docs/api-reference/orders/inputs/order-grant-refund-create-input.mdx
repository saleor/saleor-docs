### Fields

#### [<code>amount</code>](#amount)<Bullet />[<code>Decimal</code>](/api-reference/miscellaneous/scalars/decimal.mdx) \{#amount\}

Amount of the granted refund. If not provided, the amount will be calculated automatically based on provided `lines` and `grantRefundForShipping`.

#### [<code>reason</code>](#reason)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#reason\}

Reason of the granted refund.

#### [<code>reasonReference</code>](#reason-reference)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx) \{#reason-reference\}

ID of a `Page` (Model) to reference in reason.

#### [<code>lines</code>](#lines)<Bullet />[<code>[OrderGrantRefundCreateLineInput!]</code>](/api-reference/orders/inputs/order-grant-refund-create-line-input.mdx) \{#lines\}

Lines to assign to granted refund.

#### [<code>grantRefundForShipping</code>](#grant-refund-for-shipping)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx) \{#grant-refund-for-shipping\}

Determine if granted refund should include shipping costs.

#### [<code>transactionId</code>](#transaction-id)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx) \{#transaction-id\}

The ID of the transaction item related to the granted refund. If `amount` provided in the input, the transaction.chargedAmount needs to be equal or greater than provided `amount`.If `amount` is not provided in the input and calculated automatically by Saleor, the `min(calculatedAmount, transaction.chargedAmount)` will be used. Field required starting from Saleor 3.21.
