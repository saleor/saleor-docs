### Fields

#### [<code>amount</code>](#)<Bullet />[<code>Decimal</code>](/api-reference/miscellaneous/scalars/decimal.mdx)

Amount of the granted refund. if not provided and `addLines` or `removeLines` or `grantRefundForShipping` is provided, amount will be calculated automatically.

#### [<code>reason</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Reason of the granted refund.

#### [<code>reasonReference</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of a `Page` (Model) to reference in reason.

<Badge
  text="Added in Saleor 3.22"
  class="badge badge--secondary margin-bottom--sm"
/>

#### [<code>addLines</code>](#)<Bullet />[<code>[OrderGrantRefundUpdateLineAddInput!]</code>](/api-reference/orders/inputs/order-grant-refund-update-line-add-input.mdx)

Lines to assign to granted refund.

#### [<code>removeLines</code>](#)<Bullet />[<code>[ID!]</code>](/api-reference/miscellaneous/scalars/id.mdx)

Lines to remove from granted refund.

#### [<code>grantRefundForShipping</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Determine if granted refund should include shipping costs.

#### [<code>transactionId</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the transaction item related to the granted refund. If `amount` provided in the input, the transaction.chargedAmount needs to be equal or greater than provided `amount`.If `amount` is not provided in the input and calculated automatically by Saleor, the `min(calculatedAmount, transaction.chargedAmount)` will be used.Field will be required starting from Saleor 3.21.
