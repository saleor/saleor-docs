### Fields

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>automaticallyConfirmAllNewOrders</b></code>](#)<Bullet />[`Boolean`](../../../api-reference/miscellaneous/scalars/boolean.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

When disabled, all new orders from checkout will be marked as unconfirmed. When enabled orders from checkout will become unfulfilled immediately. By default set to True

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>automaticallyFulfillNonShippableGiftCard</b></code>](#)<Bullet />[`Boolean`](../../../api-reference/miscellaneous/scalars/boolean.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

When enabled, all non-shippable gift card orders will be fulfilled automatically. By default set to True.

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>expireOrdersAfter</b></code>](#)<Bullet />[`Minute`](../../../api-reference/miscellaneous/scalars/minute.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

Expiration time in minutes. Default null - means do not expire any orders. Enter 0 or null to disable.

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>deleteExpiredOrdersAfter</b></code>](#)<Bullet />[`Day`](../../../api-reference/miscellaneous/scalars/day.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

The time in days after expired orders will be deleted.Allowed range is from 1 to 120.

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>markAsPaidStrategy</b></code>](#)<Bullet />[`MarkAsPaidStrategyEnum`](../../../api-reference/channels/enums/mark-as-paid-strategy-enum.mdx) <Badge class="badge badge--secondary" text="enum"/> <Badge class="badge badge--secondary" text="channels"/>

Determine what strategy will be used to mark the order as paid. Based on the chosen option, the proper object will be created and attached to the order when it's manually marked as paid.
`PAYMENT_FLOW` - [default option] creates the `Payment` object.
`TRANSACTION_FLOW` - creates the `TransactionItem` object.

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>allowUnpaidOrders</b></code>](#)<Bullet />[`Boolean`](../../../api-reference/miscellaneous/scalars/boolean.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

Determine if it is possible to place unpaid order by calling `checkoutComplete` mutation.

#### [<code style={{ fontWeight: 'normal' }}>OrderSettingsInput.<b>includeDraftOrderInVoucherUsage</b></code>](#)<Bullet />[`Boolean`](../../../api-reference/miscellaneous/scalars/boolean.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

Specify whether a coupon applied to draft orders will count toward voucher usage.

Warning: when switching this setting from `false` to `true`, the vouchers will be disconnected from all draft orders.
