SKU of the product variant.

#### [`productVariantId`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

##### [`size`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Desired longest side the image in pixels. Defaults to 4096. Images are never cropped. Pass 0 to retrieve the original size (not recommended).

##### [`format`](#)<Bullet />[`ThumbnailFormatEnum`](/api-reference/miscellaneous/enums/thumbnail-format-enum.mdx)

The format of the image. When not provided, format of the original image will be used.

#### [`unitPrice`](#)<Bullet />[`TaxedMoney!`](/api-reference/miscellaneous/objects/taxed-money.mdx)

Price of the single item in the order line with all the line-level discounts and order-level discount portions applied.

#### [`undiscountedUnitPrice`](#)<Bullet />[`TaxedMoney!`](/api-reference/miscellaneous/objects/taxed-money.mdx)

Price of the single item in the order line without any discount applied.

#### [`unitDiscount`](#)<Bullet />[`Money!`](/api-reference/miscellaneous/objects/money.mdx)

Sum of the line-level discounts applied to the order line. Order-level discounts which affect the line are not visible in this field. For order-level discount portion (if any), please query `order.discounts` field.

#### [`unitDiscountReason`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Reason for line-level discounts applied on the order line. Order-level discounts which affect the line are not visible in this field. For order-level discount reason (if any), please query `order.discounts` field.

#### [`unitDiscountValue`](#)<Bullet />[`PositiveDecimal!`](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Value of the discount. Can store fixed value or percent value. This field shouldn't be used when multiple discounts affect the line. There is a limitation, that after running `checkoutComplete` mutation the field always stores fixed value.

#### [`unitDiscountType`](#)<Bullet />[`DiscountValueTypeEnum`](/api-reference/discounts/enums/discount-value-type-enum.mdx)

Type of the discount: `fixed` or `percent`. This field shouldn't be used when multiple discounts affect the line. There is a limitation, that after running `checkoutComplete` mutation the field is always set to `fixed`.

#### [`totalPrice`](#)<Bullet />[`TaxedMoney!`](/api-reference/miscellaneous/objects/taxed-money.mdx)

Price of the order line.

#### [`undiscountedTotalPrice`](#)<Bullet />[`TaxedMoney!`](/api-reference/miscellaneous/objects/taxed-money.mdx)

Price of the order line without discounts.

#### [`isPriceOverridden`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Returns True, if the line unit price was overridden.

#### [`variant`](#)<Bullet />[`ProductVariant`](/api-reference/products/objects/product-variant.mdx)

A purchased product variant. Note: this field may be null if the variant has been removed from stock at all. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

#### [`translatedProductName`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Product name in the customer's language

#### [`translatedVariantName`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Variant name in the customer's language

#### [`allocations`](#)<Bullet />[`[Allocation!]`](/api-reference/products/objects/allocation.mdx)

List of allocations across warehouses.

Requires one of the following permissions: MANAGE_PRODUCTS, MANAGE_ORDERS.

#### [`saleId`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

Denormalized sale ID, set when order line is created for a product variant that is on sale.

#### [`quantityToFulfill`](#)<Bullet />[`Int!`](/api-reference/miscellaneous/scalars/int.mdx)

A quantity of items remaining to be fulfilled.

#### [`taxClass`](#)<Bullet />[`TaxClass`](/api-reference/taxes/objects/tax-class.mdx)

Denormalized tax class of the product in this order line.

Requires one of the following permissions: AUTHENTICATED_STAFF_USER, AUTHENTICATED_APP.

#### [`taxClassName`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Denormalized name of the tax class.

#### [`taxClassMetadata`](#)<Bullet />[`[MetadataItem!]!`](/api-reference/miscellaneous/objects/metadata-item.mdx)

Denormalized public metadata of the tax class.

#### [`taxClassPrivateMetadata`](#)<Bullet />[`[MetadataItem!]!`](/api-reference/miscellaneous/objects/metadata-item.mdx)

Denormalized private metadata of the tax class. Requires staff permissions to access.

#### [`voucherCode`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Voucher code that was used for this order line.

#### [`isGift`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Determine if the line is a gift.

Added in Saleor 3.19.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

#### [`discounts`](#)<Bullet />[`[OrderLineDiscount!]`](/api-reference/orders/objects/order-line-discount.mdx)

List of applied discounts

Added in Saleor 3.21.
