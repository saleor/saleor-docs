### Values

#### [<code>NUMBER</code>](#number) \{#number\}

Sort orders by number.

#### [<code>RANK</code>](#rank) \{#rank\}

Sort orders by rank. Note: This option is available only with the `search` filter.

#### [<code>CREATED_AT</code>](#created-at) \{#created-at\}

Sort orders by creation date.

#### [<code>LAST_MODIFIED_AT</code>](#last-modified-at) \{#last-modified-at\}

Sort orders by last modified date.

#### [<code>CUSTOMER</code>](#customer) \{#customer\}

Sort orders by customer.

#### [<code>PAYMENT</code>](#payment) \{#payment\}

Sort orders by payment status.

#### [<code>STATUS</code>](#status) \{#status\}

Sort orders by order status.
