Approve existing fulfillment.

Requires one of the following permissions: MANAGE_ORDERS.

Triggers the following webhook events:

- FULFILLMENT_APPROVED (async): Fulfillment is approved.

```graphql
orderFulfillmentApprove(
  allowStockToBeExceeded: Boolean = false
  id: ID!
  notifyCustomer: Boolean!
): FulfillmentApprove
```

### Arguments

#### [`allowStockToBeExceeded`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

True if stock could be exceeded.

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a fulfillment to approve.

#### [`notifyCustomer`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

True if confirmation email should be send.
