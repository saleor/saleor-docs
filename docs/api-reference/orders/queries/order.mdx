Look up an order by ID or external reference.

```graphql
order(
  id: ID
  externalReference: String
): Order
```

### Arguments

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of an order.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of an order.

Requires one of the following permissions: MANAGE_ORDERS.

### Type

#### [`Order`](/api-reference/orders/objects/order.mdx)

Represents an order in the shop.
