Return the total sales amount from a specific period.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
ordersTotal(
  period: ReportingPeriod
  channel: String
): TaxedMoney
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>ordersTotal.<b>period</b></code>](#)<Bullet />[`ReportingPeriod`](../../../api-reference/miscellaneous/enums/reporting-period) <Badge class="secondary" text="enum"/> <Badge class="secondary" text="miscellaneous"/>

> A period of time.

#### [<code style={{ fontWeight: 'normal' }}>ordersTotal.<b>channel</b></code>](#)<Bullet />[`String`](../../../api-reference/miscellaneous/scalars/string) <Badge class="secondary" text="scalar"/> <Badge class="secondary" text="miscellaneous"/>

> Slug of a channel for which the data should be returned.

### Type

#### [`TaxedMoney`](../../../api-reference/miscellaneous/objects/taxed-money) <Badge class="secondary" text="object"/> <Badge class="secondary" text="miscellaneous"/>

> Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.
