Look up an app by ID. If ID is not provided, return the currently authenticated app.

<Permissions
  permissions={"AUTHENTICATED_STAFF_USER AUTHENTICATED_APP"}
  text={"Requires one of the following permissions"}
/>
The authenticated app has access to its resources. Fetching different apps
requires MANAGE&#x005F;APPS permission.

```graphql
app(
  id: ID
): App
```
