Look up an app by ID. If ID is not provided, return the currently authenticated app.

<Permissions permissions={"AUTHENTICATED_STAFF_USER AUTHENTICATED_APP"} />
The authenticated app has access to its resources. Fetching different apps requires
MANAGE_APPS permission.

```graphql
app(
  id: ID
): App
```
