Input for dismissing app problems. Only one can be specified.

```graphql
input AppProblemDismissInput {
  byApp: AppProblemDismissByAppInput
  byStaffWithIds: AppProblemDismissByStaffWithIdsInput
  byStaffWithKeys: AppProblemDismissByStaffWithKeysInput
}
```

### Fields

#### [`byApp`](#)<Bullet />[`AppProblemDismissByAppInput`](/api-reference/apps/inputs/app-problem-dismiss-by-app-input.mdx)

For app callers only - dismiss own problems.

#### [`byStaffWithIds`](#)<Bullet />[`AppProblemDismissByStaffWithIdsInput`](/api-reference/apps/inputs/app-problem-dismiss-by-staff-with-ids-input.mdx)

For staff callers - dismiss problems by IDs.

#### [`byStaffWithKeys`](#)<Bullet />[`AppProblemDismissByStaffWithKeysInput`](/api-reference/apps/inputs/app-problem-dismiss-by-staff-with-keys-input.mdx)
