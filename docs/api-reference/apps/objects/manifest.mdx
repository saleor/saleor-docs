#### [`identifier`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The identifier of the manifest for the app.

#### [`version`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The version of the manifest for the app.

#### [`name`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The name of the manifest for the app .

#### [`about`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Description of the app displayed in the dashboard.

#### [`permissions`](#)<Bullet />[`[Permission!]`](/api-reference/authentication/objects/permission.mdx)

The array permissions required for the app.

#### [`appUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

App website rendered in the dashboard.

#### [`tokenTargetUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Endpoint used during process of app installation, [see installing an app.](https://docs.saleor.io/developer/extending/apps/installing-apps#installing-an-app)

#### [`dataPrivacyUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

URL to the full privacy policy.

#### [`homepageUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External URL to the app homepage.

#### [`supportUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External URL to the page where app users can find support.

#### [`extensions`](#)<Bullet />[`[AppManifestExtension!]!`](/api-reference/apps/objects/app-manifest-extension.mdx)

List of extensions that will be mounted in Saleor's dashboard. For details, please [see the extension section.](https://docs.saleor.io/developer/extending/apps/extending-dashboard-with-apps#key-concepts)

#### [`webhooks`](#)<Bullet />[`[AppManifestWebhook!]!`](/api-reference/apps/objects/app-manifest-webhook.mdx)

List of the app's webhooks.

#### [`audience`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

The audience that will be included in all JWT tokens for the app.

#### [`requiredSaleorVersion`](#)<Bullet />[`AppManifestRequiredSaleorVersion`](/api-reference/apps/objects/app-manifest-required-saleor-version.mdx)

Determines the app's required Saleor version as semver range.

#### [`author`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

The App's author name.

#### [`brand`](#)<Bullet />[`AppManifestBrand`](/api-reference/apps/objects/app-manifest-brand.mdx)

App's brand data.
