The manifest definition.

```graphql
type Manifest {
  identifier: String!
  version: String!
  name: String!
  about: String
  permissions: [Permission!]
  appUrl: String
  configurationUrl: String @deprecated
  tokenTargetUrl: String
  dataPrivacy: String @deprecated
  dataPrivacyUrl: String
  homepageUrl: String
  supportUrl: String
  extensions: [AppManifestExtension!]!
  webhooks: [AppManifestWebhook!]!
  audience: String
  requiredSaleorVersion: AppManifestRequiredSaleorVersion
  author: String
  brand: AppManifestBrand
}
```

#### [<code>identifier</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

The identifier of the manifest for the app.

#### [<code>version</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

The version of the manifest for the app.

#### [<code>name</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

The name of the manifest for the app .

#### [<code>about</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Description of the app displayed in the dashboard.

#### [<code>permissions</code>](#)<Bullet />[<code>[Permission!]</code>](/api-reference/authentication/objects/permission.mdx)

The array permissions required for the app.

#### [<code>appUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

App website rendered in the dashboard.

#### [<code>tokenTargetUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Endpoint used during process of app installation, [see installing an app.](https://docs.saleor.io/developer/extending/apps/installing-apps#installing-an-app)

#### [<code>dataPrivacyUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

URL to the full privacy policy.

#### [<code>homepageUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

External URL to the app homepage.

#### [<code>supportUrl</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

External URL to the page where app users can find support.

#### [<code>extensions</code>](#)<Bullet />[<code>[AppManifestExtension!]!</code>](/api-reference/apps/objects/app-manifest-extension.mdx)

List of extensions that will be mounted in Saleor's dashboard. For details, please [see the extension section.](https://docs.saleor.io/developer/extending/apps/extending-dashboard-with-apps#key-concepts)

#### [<code>webhooks</code>](#)<Bullet />[<code>[AppManifestWebhook!]!</code>](/api-reference/apps/objects/app-manifest-webhook.mdx)

List of the app's webhooks.

#### [<code>audience</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The audience that will be included in all JWT tokens for the app.

#### [<code>requiredSaleorVersion</code>](#)<Bullet />[<code>AppManifestRequiredSaleorVersion</code>](/api-reference/apps/objects/app-manifest-required-saleor-version.mdx)

Determines the app's required Saleor version as semver range.

#### [<code>author</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The App's author name.

#### [<code>brand</code>](#)<Bullet />[<code>AppManifestBrand</code>](/api-reference/apps/objects/app-manifest-brand.mdx)

App's brand data.
