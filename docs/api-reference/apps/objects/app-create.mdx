Creates a new app. Requires the following permissions: AUTHENTICATED_STAFF_USER and MANAGE_APPS.

Triggers the following webhook events:

- APP_INSTALLED (async): An app was installed.

```graphql
type AppCreate {
  authToken: String
  appErrors: [AppError!]! @deprecated
  errors: [AppError!]!
  app: App
}
```
