Represents app data.

```graphql
type AppExtension implements Node {
  id: ID!
  permissions: [Permission!]!
  label: String!
  url: String!
  mountName: String!
  targetName: String!
  settings: JSON!
  app: App!
  accessToken: String
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the app extension.

#### [`permissions`](#)<Bullet />[`[Permission!]!`](/api-reference/authentication/objects/permission.mdx)

List of the app extension's permissions.

#### [`label`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Label of the extension to show in the dashboard.

#### [`url`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

URL of a view where extension's iframe is placed.

#### [`mountName`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Name of the extension mount point in the dashboard. Value returned in UPPERCASE.

Added in Saleor 3.22.

#### [`targetName`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Name of the extension target in the dashboard. Value returned in UPPERCASE.

Added in Saleor 3.22.

#### [`settings`](#)<Bullet />[`JSON!`](/api-reference/miscellaneous/scalars/json.mdx)

App extension settings.

Added in Saleor 3.22.

#### [`app`](#)<Bullet />[`App!`](/api-reference/apps/objects/app.mdx)

The app assigned to app extension.

#### [`accessToken`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

JWT token used to authenticate by third-party app extension.
