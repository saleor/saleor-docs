Represents a problem associated with an app.

Added in Saleor 3.22.

```graphql
type AppProblem implements Node {
  id: ID!
  createdAt: DateTime!
  updatedAt: DateTime!
  count: Int!
  isCritical: Boolean!
  dismissed: AppProblemDismissed
  message: String!
  key: String!
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the app problem.

Added in Saleor 3.22.

#### [`createdAt`](#)<Bullet />[`DateTime!`](/api-reference/miscellaneous/scalars/date-time.mdx)

The date and time when the problem was created.

Added in Saleor 3.22.

#### [`updatedAt`](#)<Bullet />[`DateTime!`](/api-reference/miscellaneous/scalars/date-time.mdx)

The date and time when the problem was last updated.

Added in Saleor 3.22.

#### [`count`](#)<Bullet />[`Int!`](/api-reference/miscellaneous/scalars/int.mdx)

Number of occurrences.

Added in Saleor 3.22.

#### [`isCritical`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

Whether the problem has reached critical threshold.

Added in Saleor 3.22.

#### [`dismissed`](#)<Bullet />[`AppProblemDismissed`](/api-reference/miscellaneous/objects/app-problem-dismissed.mdx)

Dismissal information. Null if the problem has not been dismissed.

Added in Saleor 3.22.

Requires one of the following permissions: AUTHENTICATED_APP, MANAGE_APPS.

#### [`message`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The problem message.

Added in Saleor 3.22.

#### [`key`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Key identifying the type of problem.

Added in Saleor 3.22.
