Updates stocks for product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantStocksUpdate(
  sku: String
  stocks: [StockInput!]!
  variantId: ID
): ProductVariantStocksUpdate
```

### Arguments

#### [`sku`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

SKU of product variant for which stocks will be updated.

#### [`stocks`](#)<Bullet />[`[StockInput!]!`](/api-reference/products/inputs/stock-input.mdx)

Input list of stocks to create or update.

#### [`variantId`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a product variant for which stocks will be updated.
