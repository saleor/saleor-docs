Reorder product attribute values.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput!]!
  productId: ID!
): ProductReorderAttributeValues
```

### Arguments

#### [`attributeId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of an attribute.

#### [`moves`](#)<Bullet />[`[ReorderInput!]!`](/api-reference/miscellaneous/inputs/reorder-input.mdx)

The list of reordering operations for given attribute values.

#### [`productId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a product.

### Type

#### [`ProductReorderAttributeValues`](/api-reference/products/objects/product-reorder-attribute-values.mdx)

Reorder product attribute values.

Requires one of the following permissions: MANAGE_PRODUCTS.
