Un-assign attributes from a given product type.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productAttributeUnassign(
  attributeIds: [ID!]!
  productTypeId: ID!
): ProductAttributeUnassign
```

### Arguments

#### [`attributeIds`](#)<Bullet />[`[ID!]!`](/api-reference/miscellaneous/scalars/id.mdx)

The IDs of the attributes to unassign.

#### [`productTypeId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the product type from which the attributes should be unassigned.

### Type

#### [`ProductAttributeUnassign`](/api-reference/products/objects/product-attribute-unassign.mdx)

Un-assign attributes from a given product type.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
