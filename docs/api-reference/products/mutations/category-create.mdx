Creates a new category.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
categoryCreate(
  input: CategoryInput!
  parent: ID
): CategoryCreate
```

### Arguments

#### [`input`](#)<Bullet />[`CategoryInput!`](/api-reference/products/inputs/category-input.mdx)

Fields required to create a category.

#### [`parent`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the parent category. If empty, category will be top level category.

#### [`CategoryCreate`](/api-reference/products/objects/category-create.mdx)
