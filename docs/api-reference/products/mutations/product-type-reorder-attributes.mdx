Reorder the attributes of a product type.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productTypeReorderAttributes(
  moves: [ReorderInput!]!
  productTypeId: ID!
  type: ProductAttributeType!
): ProductTypeReorderAttributes
```

### Arguments

#### [`moves`](#)<Bullet />[`[ReorderInput!]!`](/api-reference/miscellaneous/inputs/reorder-input.mdx)

The list of attribute reordering operations.

#### [`productTypeId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a product type.

#### [`type`](#)<Bullet />[`ProductAttributeType!`](/api-reference/products/enums/product-attribute-type.mdx)

The attribute type to reorder.
