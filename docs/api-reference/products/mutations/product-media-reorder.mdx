Changes ordering of the product media.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productMediaReorder(
  mediaIds: [ID!]!
  productId: ID!
): ProductMediaReorder
```

### Arguments

#### [`mediaIds`](#)<Bullet />[`[ID!]!`](/api-reference/miscellaneous/scalars/id.mdx)

IDs of a product media in the desired order.

#### [`productId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of product that media order will be altered.

### Type

#### [`ProductMediaReorder`](/api-reference/products/objects/product-media-reorder.mdx)

Changes ordering of the product media.

Requires one of the following permissions: MANAGE_PRODUCTS.
