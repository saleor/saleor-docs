Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantReorder(
  moves: [ReorderInput!]!
  productId: ID!
): ProductVariantReorder
```

### Arguments

#### [`moves`](#)<Bullet />[`[ReorderInput!]!`](/api-reference/miscellaneous/inputs/reorder-input.mdx)

The list of variant reordering operations.

#### [`productId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

Id of product that variants order will be altered.

### Type

#### [`ProductVariantReorder`](/api-reference/products/objects/product-variant-reorder.mdx)

Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook.

Requires one of the following permissions: MANAGE_PRODUCTS.
