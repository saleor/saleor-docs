Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantSetDefault(
  productId: ID!
  variantId: ID!
): ProductVariantSetDefault
```

### Arguments

#### [`productId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

Id of a product that will have the default variant set.

#### [`variantId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

Id of a variant that will be set as default.

### Type

#### [`ProductVariantSetDefault`](/api-reference/products/objects/product-variant-set-default.mdx)

Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.

Requires one of the following permissions: MANAGE_PRODUCTS.
