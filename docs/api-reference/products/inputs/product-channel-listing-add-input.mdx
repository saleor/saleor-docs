#### [<code>isPublished</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

#### [<code>publishedAt</code>](#)<Bullet />[<code>DateTime</code>](/api-reference/miscellaneous/scalars/date-time.mdx)

Publication date time. ISO 8601 standard.

#### [<code>visibleInListings</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Determines if product is visible in product listings (doesn't apply to product collections).

#### [<code>isAvailableForPurchase</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Determines if product should be available for purchase in this channel. This does not guarantee the availability of stock. When set to `False`, this product is still visible to customers, but it cannot be purchased.

#### [<code>availableForPurchaseAt</code>](#)<Bullet />[<code>DateTime</code>](/api-reference/miscellaneous/scalars/date-time.mdx)

A start date time from which a product will be available for purchase. When not set and `isAvailable` is set to True, the current day is assumed.

#### [<code>addVariants</code>](#)<Bullet />[<code>[ID!]</code>](/api-reference/miscellaneous/scalars/id.mdx)

List of variants to which the channel should be assigned.

#### [<code>removeVariants</code>](#)<Bullet />[<code>[ID!]</code>](/api-reference/miscellaneous/scalars/id.mdx)

List of variants from which the channel should be unassigned.
