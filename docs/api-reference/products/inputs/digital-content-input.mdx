### Fields

#### [<code>useDefaultSettings</code>](#)<Bullet />[<code>Boolean!</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Use default digital content settings for this product.

#### [<code>maxDownloads</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Determines how many times a download link can be accessed by a customer.

#### [<code>urlValidDays</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Determines for how many days a download link is active since it was generated.

#### [<code>automaticFulfillment</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Overwrite default automatic_fulfillment setting for variant.

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the digital content metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [<code>privateMetadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the digital content private metadata. Requires permissions to modify and to read the metadata of the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.
