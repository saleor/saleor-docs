### Fields

#### [`attributes`](#)<Bullet />[`[AttributeValueInput!]`](/api-reference/attributes/inputs/attribute-value-input.mdx)

List of attributes specific to this variant.

#### [`sku`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Stock keeping unit.

#### [`name`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Variant name.

#### [`trackInventory`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Determines if the inventory of this variant should be tracked. If false, the quantity won't change when customers buy this item. If the field is not provided, `Shop.trackInventoryByDefault` will be used.

#### [`weight`](#)<Bullet />[`WeightScalar`](/api-reference/miscellaneous/scalars/weight-scalar.mdx)

Weight of the Product Variant.

#### [`preorder`](#)<Bullet />[`PreorderSettingsInput`](/api-reference/products/inputs/preorder-settings-input.mdx)

Determines if variant is in preorder.

#### [`quantityLimitPerCustomer`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Determines maximum quantity of `ProductVariant`,that can be bought in a single checkout.

#### [`metadata`](#)<Bullet />[`[MetadataInput!]`](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the product variant metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [`privateMetadata`](#)<Bullet />[`[MetadataInput!]`](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the product variant private metadata. Requires permissions to modify and to read the metadata of the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of this product variant.
