Look up a product variant by ID or SKU. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
productVariant(
  id: ID
  sku: String
  externalReference: String
  channel: String
): ProductVariant
```

### Arguments

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the product variant.

#### [`sku`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

SKU of the product variant.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of the product.

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

### Type

#### [`ProductVariant`](/api-reference/products/objects/product-variant.mdx)

Represents a version of a product such as different size or color.
