Look up a product variant by ID or SKU.

<Permissions permissions={"MANAGE_ORDERS,MANAGE_DISCOUNTS,MANAGE_PRODUCTS"} />

```graphql
productVariant(
  id: ID
  sku: String
  externalReference: String
  channel: String
): ProductVariant
```
