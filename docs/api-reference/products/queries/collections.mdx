List of the shop's collections. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
collections(
  filter: CollectionFilterInput
  where: CollectionWhereInput
  sortBy: CollectionSortingInput
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): CollectionCountableConnection
```

### Arguments

#### [`filter`](#)<Bullet />[`CollectionFilterInput`](/api-reference/products/inputs/collection-filter-input.mdx)

Filtering options for collections.

#### [`where`](#)<Bullet />[`CollectionWhereInput`](/api-reference/products/inputs/collection-where-input.mdx)

Where filtering options for collections.

#### [`sortBy`](#)<Bullet />[`CollectionSortingInput`](/api-reference/products/inputs/collection-sorting-input.mdx)

Sort collections.

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [`before`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come before the specified cursor.

#### [`after`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come after the specified cursor.

#### [`first`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the first n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

#### [`last`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the last n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

### Type

#### [`CollectionCountableConnection`](/api-reference/products/objects/collection-countable-connection.mdx)

Represents a connection to a list of collections.
