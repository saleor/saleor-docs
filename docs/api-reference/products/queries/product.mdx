### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of the product.

#### [<code>slug</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of the product.

#### [<code>slugLanguageCode</code>](#)<Bullet />[<code>LanguageCodeEnum</code>](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Language code of the product slug, omit to use primary slug.
