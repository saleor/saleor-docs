Look up a category by ID or slug.

```graphql
category(
  id: ID
  slug: String
  slugLanguageCode: LanguageCodeEnum
): Category
```

### Arguments

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the category.

#### [`slug`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of the category

#### [`slugLanguageCode`](#)<Bullet />[`LanguageCodeEnum`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Language code of the category slug, omit to use primary slug.

Added in Saleor 3.21.

### Type

#### [`Category`](/api-reference/products/objects/category.mdx)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.
