### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of the category.

#### [<code>slug</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of the category

#### [<code>slugLanguageCode</code>](#)<Bullet />[<code>LanguageCodeEnum</code>](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Language code of the category slug, omit to use primary slug.

### Type

#### [<code>Category</code>](/api-reference/products/objects/category.mdx)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.
