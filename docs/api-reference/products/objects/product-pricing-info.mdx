Represents availability of a product in the storefront.

```graphql
type ProductPricingInfo {
  onSale: Boolean
  discount: TaxedMoney
  discountPrior: TaxedMoney
  discountLocalCurrency: TaxedMoney @deprecated
  displayGrossPrices: Boolean!
  priceRange: TaxedMoneyRange
  priceRangeUndiscounted: TaxedMoneyRange
  priceRangePrior: TaxedMoneyRange
  priceRangeLocalCurrency: TaxedMoneyRange @deprecated
}
```

### Fields

#### [`onSale`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Whether it is in sale or not.

#### [`discount`](#)<Bullet />[`TaxedMoney`](/api-reference/miscellaneous/objects/taxed-money.mdx)

The discount amount if in sale (null otherwise).

#### [`discountPrior`](#)<Bullet />[`TaxedMoney`](/api-reference/miscellaneous/objects/taxed-money.mdx)

The discount amount compared to prior price. Null if product is not on sale or prior price was not provided in VariantChannelListing

Added in Saleor 3.21.

#### [`displayGrossPrices`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

Determines whether displayed prices should include taxes.

#### [`priceRange`](#)<Bullet />[`TaxedMoneyRange`](/api-reference/miscellaneous/objects/taxed-money-range.mdx)

The discounted price range of the product variants.

#### [`priceRangeUndiscounted`](#)<Bullet />[`TaxedMoneyRange`](/api-reference/miscellaneous/objects/taxed-money-range.mdx)

The undiscounted price range of the product variants.

#### [`priceRangePrior`](#)<Bullet />[`TaxedMoneyRange`](/api-reference/miscellaneous/objects/taxed-money-range.mdx)

The prior price range of the product variants.

Added in Saleor 3.21.
