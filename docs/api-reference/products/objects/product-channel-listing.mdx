Represents product channel listing.

```graphql
type ProductChannelListing implements Node {
  id: ID!
  publicationDate: Date @deprecated
  publishedAt: DateTime
  isPublished: Boolean!
  channel: Channel!
  visibleInListings: Boolean!
  availableForPurchase: Date @deprecated
  availableForPurchaseAt: DateTime
  discountedPrice: Money
  purchaseCost: MoneyRange
  margin: Margin
  isAvailableForPurchase: Boolean
  pricing(address: AddressInput): ProductPricingInfo
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the product channel listing.

#### [`publishedAt`](#)<Bullet />[`DateTime`](/api-reference/miscellaneous/scalars/date-time.mdx)

The product publication date time.

#### [`isPublished`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

Indicates if the product is published in the channel.

#### [`channel`](#)<Bullet />[`Channel!`](/api-reference/channels/objects/channel.mdx)

The channel in which the product is listed.

#### [`visibleInListings`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

Indicates product visibility in the channel listings.

#### [`availableForPurchaseAt`](#)<Bullet />[`DateTime`](/api-reference/miscellaneous/scalars/date-time.mdx)

The product available for purchase date time.

#### [`discountedPrice`](#)<Bullet />[`Money`](/api-reference/miscellaneous/objects/money.mdx)

The price of the cheapest variant (including discounts).

#### [`purchaseCost`](#)<Bullet />[`MoneyRange`](/api-reference/miscellaneous/objects/money-range.mdx)

Purchase cost of product.

Requires one of the following permissions: MANAGE_PRODUCTS.

#### [`margin`](#)<Bullet />[`Margin`](/api-reference/products/objects/margin.mdx)

Range of margin percentage value.

Requires one of the following permissions: MANAGE_PRODUCTS.

#### [`isAvailableForPurchase`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Refers to a state that can be set by admins to control whether a product is available for purchase in storefronts in this channel. This does not guarantee the availability of stock. When set to `False`, this product is still visible to customers, but it cannot be purchased.

#### [`pricing`](#)<Bullet />[`ProductPricingInfo`](/api-reference/products/objects/product-pricing-info.mdx)

Lists the storefront product's pricing, the current price and discounts, only meant for displaying.

##### [`address`](#)<Bullet />[`AddressInput`](/api-reference/miscellaneous/inputs/address-input.mdx)

Destination address used to find warehouses where stock availability for this product is checked. If address is empty, uses `Shop.companyAddress` or fallbacks to server's `settings.DEFAULT_COUNTRY` configuration.
