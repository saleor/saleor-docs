Represents a product image.

```graphql
type ProductImage {
  id: ID!
  alt: String
  sortOrder: Int
  url(size: Int, format: ThumbnailFormatEnum = ORIGINAL): String!
}
```

### Fields

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the image.

#### [<code>alt</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The alt text of the image.

#### [<code>sortOrder</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

The new relative sorting position of the item (from -inf to +inf). 1 moves the item one position forward, -1 moves the item one position backward, 0 leaves the item unchanged.

#### [<code>url</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

The URL of the image.

##### [<code>size</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Desired longest side the image in pixels. Defaults to 4096. Images are never cropped. Pass 0 to retrieve the original size (not recommended).

##### [<code>format</code>](#)<Bullet />[<code>ThumbnailFormatEnum</code>](/api-reference/miscellaneous/enums/thumbnail-format-enum.mdx)

The format of the image. When not provided, format of the original image will be used.
