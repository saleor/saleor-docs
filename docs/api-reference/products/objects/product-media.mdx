Represents a product media.

```graphql
type ProductMedia implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  privateMetafield(
    key: String!
  ): String
  privateMetafields(
    keys: [String!]
  ): Metadata
  metadata: [MetadataItem!]!
  metafield(
    key: String!
  ): String
  metafields(
    keys: [String!]
  ): Metadata
  sortOrder: Int
  alt: String!
  type: ProductMediaType!
  oembedData: JSONString!
  url(
    size: Int
    format: ThumbnailFormatEnum = ORIGINAL
  ): String!
  productId: ID
}
```

##### [<code>size</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Desired longest side the image in pixels. Defaults to 4096. Images are never cropped. Pass 0 to retrieve the original size (not recommended).

##### [<code>format</code>](#)<Bullet />[<code>ThumbnailFormatEnum</code>](/api-reference/miscellaneous/enums/thumbnail-format-enum.mdx)

The format of the image. When not provided, format of the original image will be used.

Product id the media refers to.
