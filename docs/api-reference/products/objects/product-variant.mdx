Represents a version of a product such as different size or color.

```graphql
type ProductVariant implements Node, ObjectWithMetadata, ObjectWithAttributes {
  id: ID!
  privateMetadata: [MetadataItem!]!
  privateMetafield(
    key: String!
  ): String
  privateMetafields(
    keys: [String!]
  ): Metadata
  metadata: [MetadataItem!]!
  metafield(
    key: String!
  ): String
  metafields(
    keys: [String!]
  ): Metadata
  assignedAttribute(
    slug: String!
  ): AssignedAttribute
  assignedAttributes(
    limit: PositiveInt = 100
  ): [AssignedAttribute!]!
  name: String!
  sku: String
  product: Product!
  trackInventory: Boolean!
  quantityLimitPerCustomer: Int
  weight: Weight
  channel: String
  channelListings: [ProductVariantChannelListing!]
  pricing(
    address: AddressInput
  ): VariantPricingInfo
  attributes(
    variantSelection: VariantAttributeScope
  ): [SelectedAttribute!]! @deprecated
  margin: Int
  quantityOrdered: Int
  revenue(
    period: ReportingPeriod
  ): TaxedMoney
  images: [ProductImage!] @deprecated
  media: [ProductMedia!]
  translation(
    languageCode: LanguageCodeEnum!
  ): ProductVariantTranslation
  stocks(
    address: AddressInput
    countryCode: CountryCode
  ): [Stock!]
  quantityAvailable(
    address: AddressInput
    countryCode: CountryCode
  ): Int
  preorder: PreorderData
  created: DateTime!
  updatedAt: DateTime!
  externalReference: String
}
```

List of price information in channels for the product.

#### [`pricing`](#)<Bullet />[`VariantPricingInfo`](/api-reference/products/objects/variant-pricing-info.mdx)

Lists the storefront variant's pricing, the current price and discounts, only meant for displaying.

##### [`address`](#)<Bullet />[`AddressInput`](/api-reference/miscellaneous/inputs/address-input.mdx)

Destination address used to find warehouses where stock availability for this product is checked. If address is empty, uses `Shop.companyAddress` or fallbacks to server's `settings.DEFAULT_COUNTRY` configuration.

#### [`margin`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Gross margin percentage value.

#### [`quantityOrdered`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Total quantity ordered.

#### [`revenue`](#)<Bullet />[`TaxedMoney`](/api-reference/miscellaneous/objects/taxed-money.mdx)

Total revenue generated by a variant in given period of time. Note: this field should be queried using `reportProductSales` query as it uses optimizations suitable for such calculations.

Requires one of the following permissions: MANAGE_PRODUCTS.

##### [`address`](#)<Bullet />[`AddressInput`](/api-reference/miscellaneous/inputs/address-input.mdx)

#### [`quantityAvailable`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Quantity of a product available for sale in one checkout. Field value will be `null` when no `limitQuantityPerCheckout` in global settings has been set, and `productVariant` stocks are not tracked.

##### [`address`](#)<Bullet />[`AddressInput`](/api-reference/miscellaneous/inputs/address-input.mdx)

Destination address used to find warehouses where stock availability for this product is checked. If address is empty, uses `Shop.companyAddress` or fallbacks to server's `settings.DEFAULT_COUNTRY` configuration.

##### [`countryCode`](#)<Bullet />[`CountryCode`](/api-reference/miscellaneous/enums/country-code.mdx)

:::warning[DEPRECATED]

Use `address` argument instead.

:::

Two-letter ISO 3166-1 country code. When provided, the exact quantity from a warehouse operating in shipping zones that contain this country will be returned. Otherwise, it will return the maximum quantity from all shipping zones.

Preorder data for product variant.

The date and time when the product variant was last updated.
