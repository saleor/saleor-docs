Voucher should be applied once per customer.

#### [`onlyForStaff`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Voucher can be used only by staff user.

#### [`singleUse`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

When set to 'True', each voucher code can be used only once; otherwise, codes can be used multiple times depending on `usageLimit`.

The option can only be changed if none of the voucher codes have been used.

Added in Saleor 3.18.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

#### [`usageLimit`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Limit number of times this voucher can be used in total.
