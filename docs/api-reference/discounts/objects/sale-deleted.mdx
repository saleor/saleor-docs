Added in Saleor 3.2.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

```graphql
type SaleDeleted implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  sale(channel: String): Sale
```
