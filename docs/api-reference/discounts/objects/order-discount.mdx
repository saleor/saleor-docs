Contains all details related to the applied discount to the order.

```graphql
type OrderDiscount implements Node {
  id: ID!
  type: OrderDiscountType!
  name: String
  translatedName: String
  valueType: DiscountValueTypeEnum!
  value: PositiveDecimal!
  reason: String
  amount: Money! @deprecated
  total: Money!
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of discount applied.

#### [`type`](#)<Bullet />[`OrderDiscountType!`](/api-reference/discounts/enums/order-discount-type.mdx)

The type of applied discount: Sale, Voucher or Manual.

#### [`name`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

The name of applied discount.

#### [`translatedName`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Translated name of the applied discount.

#### [`valueType`](#)<Bullet />[`DiscountValueTypeEnum!`](/api-reference/discounts/enums/discount-value-type-enum.mdx)

Type of the discount: fixed or percent

#### [`value`](#)<Bullet />[`PositiveDecimal!`](/api-reference/miscellaneous/scalars/positive-decimal.mdx)

Value of the discount. Can store fixed value or percent value

#### [`reason`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Explanation for the applied discount.

Requires one of the following permissions: MANAGE_ORDERS.

#### [`total`](#)<Bullet />[`Money!`](/api-reference/miscellaneous/objects/money.mdx)

The amount of discount applied to the order.

Added in Saleor 3.21.
