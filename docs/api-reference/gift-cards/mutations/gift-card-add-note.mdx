Adds note to the gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_UPDATED (async): A gift card was updated.

```graphql
giftCardAddNote(
  id: ID!
  input: GiftCardAddNoteInput!
): GiftCardAddNote
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the gift card to add a note for.

#### [`input`](#)<Bullet />[`GiftCardAddNoteInput!`](/api-reference/gift-cards/inputs/gift-card-add-note-input.mdx)

Fields required to create a note for the gift card.
