Creates/updates translations for a shipping method.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
shippingPriceTranslate(
  id: ID!
  input: ShippingPriceTranslationInput!
  languageCode: LanguageCodeEnum!
): ShippingPriceTranslate
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ShippingMethodType ID or ShippingMethodTranslatableContent ID.

#### [`input`](#)<Bullet />[`ShippingPriceTranslationInput!`](/api-reference/miscellaneous/inputs/shipping-price-translation-input.mdx)

Fields required to update shipping price translations.

#### [`languageCode`](#)<Bullet />[`LanguageCodeEnum!`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Translation language code.
