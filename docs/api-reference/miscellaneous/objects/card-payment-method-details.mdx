Represents a card payment method used for a transaction.

Added in Saleor 3.22.

```graphql
type CardPaymentMethodDetails implements PaymentMethodDetails {
  name: String!
  brand: String
  firstDigits: String
  lastDigits: String
  expMonth: Int
  expYear: Int
}
```
