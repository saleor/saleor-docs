Creates/updates translations for shop settings.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
type ShopSettingsTranslate {
  shop: Shop
  translationErrors: [TranslationError!]!
  errors: [TranslationError!]!
}
```
