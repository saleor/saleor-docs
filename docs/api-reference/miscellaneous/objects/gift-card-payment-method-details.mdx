Represents a gift card payment method used for a transaction.

Added in Saleor 3.23.

```graphql
type GiftCardPaymentMethodDetails implements PaymentMethodDetails {
  name: String!
  brand: String
  lastChars: String
  isSaleorGiftcard: Boolean!
}
```
