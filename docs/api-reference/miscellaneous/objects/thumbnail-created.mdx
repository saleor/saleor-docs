Event sent when thumbnail is created.

Added in Saleor 3.12.

```graphql
type ThumbnailCreated implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  id: ID
  url: String
  objectId: ID
  mediaUrl: String
}
```
