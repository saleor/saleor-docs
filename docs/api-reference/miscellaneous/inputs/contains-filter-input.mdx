Define the filtering options for fields that can contain multiple values.

```graphql
input ContainsFilterInput {
  containsAny: [String!]

}
```

### Fields

#### [<code>containsAny</code>](#)<Bullet />[<code>[String!]</code>](/api-reference/miscellaneous/scalars/string.mdx)
