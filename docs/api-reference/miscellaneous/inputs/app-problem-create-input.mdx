#### [<code style={{ fontWeight: 'normal' }}>AppProblemCreateInput.<b>message</b></code>](#)<Bullet />[`String!`](../../../api-reference/miscellaneous/scalars/string.mdx) <Badge class="badge badge--secondary" text="non-null"/> <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

The problem message to display. Must be at least 3 characters. Messages longer than 2048 characters will be truncated to 2048 characters with '...' suffix.

#### [<code style={{ fontWeight: 'normal' }}>AppProblemCreateInput.<b>key</b></code>](#)<Bullet />[`String!`](../../../api-reference/miscellaneous/scalars/string.mdx) <Badge class="badge badge--secondary" text="non-null"/> <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

Key identifying the type of problem. App can add multiple problems under the same key, to merge them together or delete them in batch. Must be between 3 and 128 characters.

#### [<code style={{ fontWeight: 'normal' }}>AppProblemCreateInput.<b>criticalThreshold</b></code>](#)<Bullet />[`PositiveInt`](../../../api-reference/miscellaneous/scalars/positive-int.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

If set, the problem becomes critical when count reaches this value. If sent again with higher value than already counted, problem can be de-escalated.

#### [<code style={{ fontWeight: 'normal' }}>AppProblemCreateInput.<b>aggregationPeriod</b></code>](#)<Bullet />[`Minute`](../../../api-reference/miscellaneous/scalars/minute.mdx) <Badge class="badge badge--secondary" text="scalar"/> <Badge class="badge badge--secondary" text="miscellaneous"/>

Time window in minutes for aggregating problems with the same key. Defaults to 60. If 0, a new problem is always created.
