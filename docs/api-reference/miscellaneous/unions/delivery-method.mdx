Represents a delivery method chosen for the checkout. `Warehouse` type is used when checkout is marked as "click and collect" and `ShippingMethod` otherwise.

<Badge text="Added in Saleor 3.1" class="secondary margin-bottom--sm" />

<FeaturePreview />

```graphql
union DeliveryMethod = Warehouse | ShippingMethod
```

### Possible types

#### [`Warehouse`](../../../api-reference/products/objects/warehouse) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents warehouse.

#### [`ShippingMethod`](../../../api-reference/shipping/objects/shipping-method) <Badge class="secondary" text="object"/> <Badge class="secondary" text="shipping"/>

> Shipping methods that can be used as means of shipping for orders and checkouts.
