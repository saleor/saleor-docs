Represents an problem in the checkout.

```graphql
union CheckoutProblem =
  | CheckoutProblemDeliveryMethodStale
  | CheckoutProblemDeliveryMethodInvalid
  | CheckoutLineProblemInsufficientStock
  | CheckoutLineProblemVariantNotAvailable
```

#### [<code style={{ fontWeight: 'normal' }}>CheckoutProblem.<b>CheckoutLineProblemInsufficientStock</b></code>](../../../api-reference/checkout/objects/checkout-line-problem-insufficient-stock.mdx) <Badge class="badge badge--secondary" text="object"/> <Badge class="badge badge--secondary" text="checkout"/>
