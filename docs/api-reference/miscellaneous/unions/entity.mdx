\_Entity union as defined by Federation spec.

```graphql
union _Entity =
  | App
  | PageType
  | Address
  | User
  | Group
  | ProductVariant
  | Product
  | ProductType
  | ProductMedia
  | Category
  | Collection
  | Order
```

### Possible types

#### [<code>App</code>](/api-reference/apps/objects/app.mdx)

Represents app data.

#### [<code>PageType</code>](/api-reference/pages/objects/page-type.mdx)

Represents a type of page. It defines what attributes are available to pages of this type.

#### [<code>Address</code>](/api-reference/users/objects/address.mdx)

Represents user address data.

#### [<code>User</code>](/api-reference/users/objects/user.mdx)

Represents user data.

#### [<code>Group</code>](/api-reference/users/objects/group.mdx)

Represents permission group data.

#### [<code>ProductVariant</code>](/api-reference/products/objects/product-variant.mdx)

Represents a version of a product such as different size or color.

#### [<code>Product</code>](/api-reference/products/objects/product.mdx)

Represents an individual item for sale in the storefront.

#### [<code>ProductType</code>](/api-reference/products/objects/product-type.mdx)

Represents a type of product. It defines what attributes are available to products of this type.

#### [<code>ProductMedia</code>](/api-reference/products/objects/product-media.mdx)

Represents a product media.

#### [<code>Category</code>](/api-reference/products/objects/category.mdx)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.

#### [<code>Collection</code>](/api-reference/products/objects/collection.mdx)

Represents a collection of products.

#### [<code>Order</code>](/api-reference/orders/objects/order.mdx)

Represents an order in the shop.
