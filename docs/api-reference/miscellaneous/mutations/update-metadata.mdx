Updates metadata of an object.Requires permissions to modify and to read the metadata of the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

```graphql
updateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdateMetadata
```

### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID or token (for Order and Checkout) of an object to update.

#### [<code>input</code>](#)<Bullet />[<code>[MetadataInput!]!</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the object's metadata.
