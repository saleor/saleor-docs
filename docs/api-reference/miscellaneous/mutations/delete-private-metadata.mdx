Delete object's private metadata. To use it, you need to be an authenticated staff user or an app and have access to the modified object.

```graphql
deletePrivateMetadata(
  id: ID!
  keys: [String!]!
): DeletePrivateMetadata
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID or token (for Order and Checkout) of an object to update.

#### [`keys`](#)<Bullet />[`[String!]!`](/api-reference/miscellaneous/scalars/string.mdx)

Metadata keys to delete.

### Type

#### [`DeletePrivateMetadata`](/api-reference/miscellaneous/objects/delete-private-metadata.mdx)

Delete object's private metadata. To use it, you need to be an authenticated staff user or an app and have access to the modified object.
