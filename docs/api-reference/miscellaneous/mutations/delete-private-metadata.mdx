Delete object's private metadata. To use it, you need to be an authenticated staff user or an app and have access to the modified object.

```graphql
deletePrivateMetadata(
  id: ID!
  keys: [String!]!
): DeletePrivateMetadata
```

### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID or token (for Order and Checkout) of an object to update.

#### [<code>keys</code>](#)<Bullet />[<code>[String!]!</code>](/api-reference/miscellaneous/scalars/string.mdx)

Metadata keys to delete.

### Type

#### [<code>DeletePrivateMetadata</code>](/api-reference/miscellaneous/objects/delete-private-metadata.mdx)
