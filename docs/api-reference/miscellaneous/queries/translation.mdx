Lookup a translatable item by ID.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
translation(
  id: ID!
  kind: TranslatableKinds!
): TranslatableItem
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the object to retrieve.

#### [`kind`](#)<Bullet />[`TranslatableKinds!`](/api-reference/miscellaneous/enums/translatable-kinds.mdx)

Kind of the object to retrieve.
