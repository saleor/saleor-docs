Represents a single item of the related menu. Can store categories, collection or pages.

```graphql
type MenuItem implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  privateMetafield(
    key: String!
  ): String
  privateMetafields(
    keys: [String!]
  ): Metadata
  metadata: [MetadataItem!]!
  metafield(
    key: String!
  ): String
  metafields(
    keys: [String!]
  ): Metadata
  name: String!
  menu: Menu!
  parent: MenuItem
  category: Category
  collection: Collection
  page: Page
  level: Int!
  children: [MenuItem!]
  url: String
  translation(
    languageCode: LanguageCodeEnum!
  ): MenuItemTranslation
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the menu item.

List of private metadata items. Requires staff permissions to access.

#### [`privateMetafield`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

A single key from private metadata. Requires staff permissions to access.

##### [`key`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

#### [`privateMetafields`](#)<Bullet />[`Metadata`](/api-reference/miscellaneous/scalars/metadata.mdx)

Private metadata. Requires staff permissions to access. Use `keys` to control which fields you want to include. The default is to include everything.

##### [`keys`](#)<Bullet />[`[String!]`](/api-reference/miscellaneous/scalars/string.mdx)

#### [`metadata`](#)<Bullet />[`[MetadataItem!]!`](/api-reference/miscellaneous/objects/metadata-item.mdx)

List of public metadata items. Can be accessed without permissions.

#### [`metafield`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

A single key from public metadata.

Tip: Use GraphQL aliases to fetch multiple keys.

##### [`key`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

#### [`metafields`](#)<Bullet />[`Metadata`](/api-reference/miscellaneous/scalars/metadata.mdx)

Public metadata. Use `keys` to control which fields you want to include. The default is to include everything.

##### [`keys`](#)<Bullet />[`[String!]`](/api-reference/miscellaneous/scalars/string.mdx)

#### [`name`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The name of the menu item.

#### [`menu`](#)<Bullet />[`Menu!`](/api-reference/menu/objects/menu.mdx)

Represents the menu to which the menu item belongs.

#### [`parent`](#)<Bullet />[`MenuItem`](/api-reference/menu/objects/menu-item.mdx)

ID of parent menu item. If empty, menu will be top level menu.

#### [`category`](#)<Bullet />[`Category`](/api-reference/products/objects/category.mdx)

Category associated with the menu item.

#### [`collection`](#)<Bullet />[`Collection`](/api-reference/products/objects/collection.mdx)

A collection associated with this menu item. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

#### [`page`](#)<Bullet />[`Page`](/api-reference/pages/objects/page.mdx)

A page associated with this menu item. Requires one of the following permissions to include unpublished items: MANAGE_PAGES.

#### [`level`](#)<Bullet />[`Int!`](/api-reference/miscellaneous/scalars/int.mdx)

Indicates the position of the menu item within the menu structure.

#### [`children`](#)<Bullet />[`[MenuItem!]`](/api-reference/menu/objects/menu-item.mdx)

Represents the child items of the current menu item.

#### [`url`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

URL to the menu item.

#### [`translation`](#)<Bullet />[`MenuItemTranslation`](/api-reference/menu/objects/menu-item-translation.mdx)

Returns translated menu item fields for the given language code.

##### [`languageCode`](#)<Bullet />[`LanguageCodeEnum!`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

A language code to return the translation for menu item.
