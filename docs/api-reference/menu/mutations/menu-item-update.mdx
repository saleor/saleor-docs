Updates a menu item.

<Permissions
  permissions={"MANAGE_MENUS"}
  text={"Requires one of the following permissions"}
/>

Triggers the following webhook events:

- MENU_ITEM_UPDATED (async): A menu item was updated.

```graphql
menuItemUpdate(
  id: ID!
  input: MenuItemInput!
): MenuItemUpdate
```

### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of a menu item to update.

#### [<code>input</code>](#)<Bullet />[<code>MenuItemInput!</code>](/api-reference/miscellaneous/inputs/menu-item-input.mdx)

Fields required to update a menu item. Only one of `url`, `category`, `page`, `collection` is allowed per item.
