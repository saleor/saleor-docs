List of the storefronts's menu items.

```graphql
menuItems(
  channel: String
  sortBy: MenuItemSortingInput
  filter: MenuItemFilterInput
  before: String
  after: String
  first: Int
  last: Int
): MenuItemCountableConnection
```

### Arguments

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [`sortBy`](#)<Bullet />[`MenuItemSortingInput`](/api-reference/miscellaneous/inputs/menu-item-sorting-input.mdx)

Sort menus items.

#### [`filter`](#)<Bullet />[`MenuItemFilterInput`](/api-reference/miscellaneous/inputs/menu-item-filter-input.mdx)

Filtering options for menu items.

#### [`before`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come before the specified cursor.

#### [`after`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come after the specified cursor.

#### [`first`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the first n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

#### [`last`](#)<Bullet />[`Int`](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the last n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

### Type

#### [`MenuItemCountableConnection`](/api-reference/menu/objects/menu-item-countable-connection.mdx)
