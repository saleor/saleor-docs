Look up a menu item by ID.

```graphql
menuItem(
  id: ID!
  channel: String
): MenuItem
```

### Arguments

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of the menu item.

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

### Type

#### [<code>MenuItem</code>](/api-reference/menu/objects/menu-item.mdx)

Represents a single item of the related menu. Can store categories, collection or pages.
