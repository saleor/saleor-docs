List of the storefront's menus.

```graphql
menus(
  channel: String
  sortBy: MenuSortingInput
  filter: MenuFilterInput
  before: String
  after: String
  first: Int
  last: Int
): MenuCountableConnection
```

### Arguments

#### [<code>channel</code>](#channel)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#channel\}

Slug of a channel for which the data should be returned.

#### [<code>sortBy</code>](#sort-by)<Bullet />[<code>MenuSortingInput</code>](/api-reference/miscellaneous/inputs/menu-sorting-input.mdx) \{#sort-by\}

Sort menus.

#### [<code>filter</code>](#filter)<Bullet />[<code>MenuFilterInput</code>](/api-reference/miscellaneous/inputs/menu-filter-input.mdx) \{#filter\}

Filtering options for menus.

`slug`: Use `slugs` instead.

#### [<code>before</code>](#before)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#before\}

Return the elements in the list that come before the specified cursor.

#### [<code>after</code>](#after)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx) \{#after\}

Return the elements in the list that come after the specified cursor.

#### [<code>first</code>](#first)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx) \{#first\}

Retrieve the first n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

#### [<code>last</code>](#last)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx) \{#last\}

Retrieve the last n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.
