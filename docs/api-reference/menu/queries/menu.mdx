Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID
  name: String
  slug: String
): Menu
```

### Arguments

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the menu.

#### [`name`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

The menu's name.

#### [`slug`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

The menu's slug.

### Type

#### [`Menu`](/api-reference/menu/objects/menu.mdx)

Represents a single menu - an object that is used to help navigate through the store.
