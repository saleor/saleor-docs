Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID
  name: String
  slug: String
): Menu
```

### Arguments

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [<code>id</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

ID of the menu.

#### [<code>name</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The menu's name.

#### [<code>slug</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The menu's slug.

### Type

#### [<code>Menu</code>](/api-reference/menu/objects/menu.mdx)

Represents a single menu - an object that is used to help navigate through the store.
