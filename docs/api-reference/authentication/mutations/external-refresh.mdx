Refresh user's access by custom plugin.

```graphql
externalRefresh(
  input: JSONString!
  pluginId: String!
): ExternalRefresh
```

### Arguments

#### [<code>input</code>](#)<Bullet />[<code>JSONString!</code>](/api-reference/miscellaneous/scalars/jsonstring.mdx)

#### [<code>pluginId</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

The ID of the authentication plugin.

#### [<code>ExternalRefresh</code>](/api-reference/authentication/objects/external-refresh.mdx)
