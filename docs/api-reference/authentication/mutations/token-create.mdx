### Arguments

#### [<code>audience</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The audience that will be included to JWT tokens with prefix `custom:`.

#### [<code>email</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

Email of a user.

#### [<code>password</code>](#)<Bullet />[<code>String!</code>](/api-reference/miscellaneous/scalars/string.mdx)

Password of a user.
