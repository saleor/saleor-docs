### Arguments

#### [<code>period</code>](#)<Bullet />[<code>ReportingPeriod</code>](/api-reference/miscellaneous/enums/reporting-period.mdx)

A period of time.

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

### Type

#### [<code>TaxedMoney</code>](/api-reference/miscellaneous/objects/taxed-money.mdx)

Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.
