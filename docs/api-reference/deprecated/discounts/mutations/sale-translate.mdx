Creates/updates translations for a sale.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
saleTranslate(
  id: ID!
  input: NameTranslationInput!
  languageCode: LanguageCodeEnum!
): SaleTranslate @deprecated
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

Sale ID or SaleTranslatableContent ID.

#### [`input`](#)<Bullet />[`NameTranslationInput!`](/api-reference/miscellaneous/inputs/name-translation-input.mdx)

Fields required to update sale translations.

#### [`languageCode`](#)<Bullet />[`LanguageCodeEnum!`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

Translation language code.

### Type

#### [`SaleTranslate`](/api-reference/discounts/objects/sale-translate.mdx)
