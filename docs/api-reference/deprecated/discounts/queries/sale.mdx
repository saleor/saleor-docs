Look up a sale by ID.

Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
sale(
  id: ID!
  channel: String
): Sale @deprecated
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the sale.

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

### Type

#### [`Sale`](/api-reference/discounts/objects/sale.mdx)

Sales allow creating discounts for categories, collections or products and are visible to all the customers.

DEPRECATED: this type will be removed. Use `Promotion` type instead.
