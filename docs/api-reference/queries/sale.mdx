Look up a sale by ID.

Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
sale(
  id: ID!
  channel: String
): Sale
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>sale.<b>id</b></code>](#)<Bullet />[`ID!`](../../api-reference/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> ID of the sale.

#### [<code style={{ fontWeight: 'normal' }}>sale.<b>channel</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of a channel for which the data should be returned.

### Type

#### [`Sale`](../../api-reference/objects/sale) <Badge class="secondary" text="object"/>

> Sales allow creating discounts for categories, collections or products and are visible to all the customers.
