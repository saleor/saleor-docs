Look up a voucher by ID.

Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
voucher(
  id: ID!
  channel: String
): Voucher
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>voucher.<b>id</b></code>](#)<Bullet />[`ID!`](../../api-reference/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> ID of the voucher.

#### [<code style={{ fontWeight: 'normal' }}>voucher.<b>channel</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of a channel for which the data should be returned.

### Type

#### [`Voucher`](../../api-reference/objects/voucher) <Badge class="secondary" text="object"/>

> Vouchers allow giving discounts to particular customers on categories, collections or specific products. They can be used during checkout by providing valid voucher codes.
