Look up an attribute by ID.

```graphql
attribute(
  id: ID
  slug: String
): Attribute
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>attribute.<b>id</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of the attribute.

#### [<code style={{ fontWeight: 'normal' }}>attribute.<b>slug</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of the attribute.

### Type

#### [`Attribute`](../../api-reference/objects/attribute) <Badge class="secondary" text="object"/>

> Custom attribute of a product. Attributes can be assigned to products and variants at the product type level.
