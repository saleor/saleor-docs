Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID
  name: String
  slug: String
): Menu
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>menu.<b>channel</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> Slug of a channel for which the data should be returned.

#### [<code style={{ fontWeight: 'normal' }}>menu.<b>id</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of the menu.

#### [<code style={{ fontWeight: 'normal' }}>menu.<b>name</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> The menu's name.

#### [<code style={{ fontWeight: 'normal' }}>menu.<b>slug</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> The menu's slug.

### Type

#### [`Menu`](../../api-reference/objects/menu) <Badge class="secondary" text="object"/>

> Represents a single menu - an object that is used to help navigate through the store.
