### Fields

#### [<code>slug</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Internal representation of an attribute name.

#### [<code>value</code>](#)<Bullet />[<code>AssignedAttributeValueInput</code>](/api-reference/miscellaneous/inputs/assigned-attribute-value-input.mdx)

Filter by value of the attribute. Only one value input field is allowed. If provided more than one, the error will be raised. Cannot be combined with deprecated fields of `AttributeInput`.

<Details dataOpen="Hide deprecated" dataClose="Show deprecated">

#### [<code>values</code>](#)<Bullet />[<code>[String!]</code>](/api-reference/miscellaneous/scalars/string.mdx)

:::warning[DEPRECATED]

Use `value` instead.

:::

Slugs identifying the attributeValues associated with the Attribute. When specified, it filters the results to include only records with one of the matching values. Requires `slug` to be provided.

#### [<code>valuesRange</code>](#)<Bullet />[<code>IntRangeInput</code>](/api-reference/miscellaneous/inputs/int-range-input.mdx)

:::warning[DEPRECATED]

Use `value` instead.

:::

The range that the returned values should be in. Requires `slug` to be provided.

#### [<code>dateTime</code>](#)<Bullet />[<code>DateTimeRangeInput</code>](/api-reference/miscellaneous/inputs/date-time-range-input.mdx)

:::warning[DEPRECATED]

Use `value` instead.

:::

The date/time range that the returned values should be in. Requires `slug` to be provided.

#### [<code>date</code>](#)<Bullet />[<code>DateRangeInput</code>](/api-reference/miscellaneous/inputs/date-range-input.mdx)

:::warning[DEPRECATED]

Use `value` instead.

:::

The date range that the returned values should be in. In case of date/time attributes, the UTC midnight of the given date is used. Requires `slug` to be provided.

#### [<code>boolean</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

:::warning[DEPRECATED]

Use `value` instead.

:::

The boolean value of the attribute. Requires `slug` to be provided.
