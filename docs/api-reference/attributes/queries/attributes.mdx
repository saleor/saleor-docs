### Arguments

#### [<code>where</code>](#)<Bullet />[<code>AttributeWhereInput</code>](/api-reference/attributes/inputs/attribute-where-input.mdx)

Where filtering options for attributes.

#### [<code>search</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Search attributes.

#### [<code>sortBy</code>](#)<Bullet />[<code>AttributeSortingInput</code>](/api-reference/attributes/inputs/attribute-sorting-input.mdx)

Sorting options for attributes.

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel for which the data should be returned.

#### [<code>before</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come before the specified cursor.

#### [<code>after</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Return the elements in the list that come after the specified cursor.

#### [<code>first</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the first n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.

#### [<code>last</code>](#)<Bullet />[<code>Int</code>](/api-reference/miscellaneous/scalars/int.mdx)

Retrieve the last n elements from the list. Note that the system only allows fetching a maximum of 100 objects in a single query.
