Look up an attribute by ID, slug or external reference.

```graphql
attribute(
  id: ID
  slug: String
  externalReference: String
): Attribute
```

### Arguments

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the attribute.

#### [`slug`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of the attribute.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of the attribute.

### Type

#### [`Attribute`](/api-reference/attributes/objects/attribute.mdx)

Custom attribute of a product. Attributes can be assigned to products and variants at the product type level.
