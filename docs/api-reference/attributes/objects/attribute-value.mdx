Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  value: String
  translation(languageCode: LanguageCodeEnum!): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  reference: ID
  file: File
  richText: JSONString
  plainText: String
  boolean: Boolean
  date: Date
  dateTime: DateTime
  externalReference: String
}
```

### Fields

#### [<code>id</code>](#)<Bullet />[<code>ID!</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the attribute value.

#### [<code>name</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Name of a value displayed in the interface.

#### [<code>slug</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Internal representation of a value (unique per attribute).

#### [<code>value</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Represent value of the attribute value (e.g. color values for swatch attributes).

#### [<code>translation</code>](#)<Bullet />[<code>AttributeValueTranslation</code>](/api-reference/attributes/objects/attribute-value-translation.mdx)

Returns translated attribute value fields for the given language code.

##### [<code>languageCode</code>](#)<Bullet />[<code>LanguageCodeEnum!</code>](/api-reference/miscellaneous/enums/language-code-enum.mdx)

A language code to return the translation for attribute value.

#### [<code>inputType</code>](#)<Bullet />[<code>AttributeInputTypeEnum</code>](/api-reference/attributes/enums/attribute-input-type-enum.mdx)

The input type to use for entering attribute values in the dashboard.

#### [<code>reference</code>](#)<Bullet />[<code>ID</code>](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the referenced object.

#### [<code>file</code>](#)<Bullet />[<code>File</code>](/api-reference/miscellaneous/objects/file.mdx)

Represents file URL and content type (if attribute value is a file).

#### [<code>richText</code>](#)<Bullet />[<code>JSONString</code>](/api-reference/miscellaneous/scalars/jsonstring.mdx)

Represents the text of the attribute value, includes formatting.

Rich text format. For reference see https://editorjs.io/

#### [<code>plainText</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Represents the text of the attribute value, plain text without formatting.

#### [<code>boolean</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

Represents the boolean value of the attribute value.

#### [<code>date</code>](#)<Bullet />[<code>Date</code>](/api-reference/miscellaneous/scalars/date.mdx)

Represents the date value of the attribute value.

#### [<code>dateTime</code>](#)<Bullet />[<code>DateTime</code>](/api-reference/miscellaneous/scalars/date-time.mdx)

Represents the date/time value of the attribute value.

#### [<code>externalReference</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

External ID of this attribute value.
