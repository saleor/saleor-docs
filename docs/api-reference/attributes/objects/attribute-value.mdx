Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  value: String
  translation(languageCode: LanguageCodeEnum!): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  reference: ID
  file: File
  richText: JSONString
  plainText: String
  boolean: Boolean
  date: Date
  dateTime: DateTime
  externalReference: String
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the attribute value.

#### [`name`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Name of a value displayed in the interface.

#### [`slug`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Internal representation of a value (unique per attribute).

#### [`value`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Represent value of the attribute value (e.g. color values for swatch attributes).

#### [`translation`](#)<Bullet />[`AttributeValueTranslation`](/api-reference/attributes/objects/attribute-value-translation.mdx)

Returns translated attribute value fields for the given language code.

##### [`languageCode`](#)<Bullet />[`LanguageCodeEnum!`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

A language code to return the translation for attribute value.

#### [`inputType`](#)<Bullet />[`AttributeInputTypeEnum`](/api-reference/attributes/enums/attribute-input-type-enum.mdx)

The input type to use for entering attribute values in the dashboard.

#### [`reference`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the referenced object.

#### [`file`](#)<Bullet />[`File`](/api-reference/miscellaneous/objects/file.mdx)

Represents file URL and content type (if attribute value is a file).

#### [`richText`](#)<Bullet />[`JSONString`](/api-reference/miscellaneous/scalars/jsonstring.mdx)

Represents the text of the attribute value, includes formatting.

Rich text format. For reference see https://editorjs.io/

#### [`plainText`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Represents the text of the attribute value, plain text without formatting.

#### [`boolean`](#)<Bullet />[`Boolean`](/api-reference/miscellaneous/scalars/boolean.mdx)

Represents the boolean value of the attribute value.

#### [`date`](#)<Bullet />[`Date`](/api-reference/miscellaneous/scalars/date.mdx)

Represents the date value of the attribute value.

#### [`dateTime`](#)<Bullet />[`DateTime`](/api-reference/miscellaneous/scalars/date-time.mdx)

Represents the date/time value of the attribute value.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of this attribute value.
