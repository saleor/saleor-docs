Represents permission group data.

```graphql
type Group implements Node {
  id: ID!
  name: String!
  users: [User!]
  permissions: [Permission!]
  userCanManage: Boolean!
  accessibleChannels: [Channel!]
  restrictedAccessToChannels: Boolean!
}
```

### Fields

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

The ID of the group.

#### [`name`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The name of the group.

#### [`users`](#)<Bullet />[`[User!]`](/api-reference/users/objects/user.mdx)

List of group users

Requires one of the following permissions: MANAGE_STAFF.

#### [`permissions`](#)<Bullet />[`[Permission!]`](/api-reference/authentication/objects/permission.mdx)

List of group permissions

#### [`userCanManage`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

True, if the currently authenticated user has rights to manage a group.

#### [`accessibleChannels`](#)<Bullet />[`[Channel!]`](/api-reference/channels/objects/channel.mdx)

List of channels the group has access to.

#### [`restrictedAccessToChannels`](#)<Bullet />[`Boolean!`](/api-reference/miscellaneous/scalars/boolean.mdx)

Determine if the group have restricted access to channels.
