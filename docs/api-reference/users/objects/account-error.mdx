Represents errors in account mutations.

```graphql
type AccountError {
  field: String
  message: String
  code: AccountErrorCode!
  addressType: AddressTypeEnum
}
```

### Fields

#### [<code>field</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### [<code>message</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The error message.

#### [<code>code</code>](#)<Bullet />[<code>AccountErrorCode!</code>](/api-reference/users/enums/account-error-code.mdx)

The error code.

#### [<code>addressType</code>](#)<Bullet />[<code>AddressTypeEnum</code>](/api-reference/miscellaneous/enums/address-type-enum.mdx)

A type of address that causes the error.
