Sets the user's password from the token sent by email using the RequestPasswordReset mutation.

```graphql
setPassword(
  email: String!
  password: String!
  token: String!
): SetPassword
```

### Arguments

#### [`email`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Email of a user.

#### [`password`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Password of a user.

#### [`token`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

A one-time token required to set the password.

### Type

#### [`SetPassword`](/api-reference/users/objects/set-password.mdx)

Sets the user's password from the token sent by email using the RequestPasswordReset mutation.
