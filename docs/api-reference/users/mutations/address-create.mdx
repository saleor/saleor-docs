Creates user address.

Requires one of the following permissions: MANAGE_USERS.

Triggers the following webhook events:

- ADDRESS_CREATED (async): A new address was created.

```graphql
addressCreate(
  input: AddressInput!
  userId: ID!
): AddressCreate
```

### Arguments

#### [`input`](#)<Bullet />[`AddressInput!`](/api-reference/miscellaneous/inputs/address-input.mdx)

Fields required to create address.

#### [`userId`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a user to create address for.
