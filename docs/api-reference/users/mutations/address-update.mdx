Updates an address.

Requires one of the following permissions: MANAGE_USERS.

Triggers the following webhook events:

- ADDRESS_UPDATED (async): An address was updated.

```graphql
addressUpdate(
  id: ID!
  input: AddressInput!
): AddressUpdate
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the address to update.

#### [`input`](#)<Bullet />[`AddressInput!`](/api-reference/miscellaneous/inputs/address-input.mdx)

Fields required to update the address.
