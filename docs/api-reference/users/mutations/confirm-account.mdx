Confirm user account with token sent by email during registration.

Triggers the following webhook events:

- ACCOUNT_CONFIRMED (async): Account was confirmed.

```graphql
confirmAccount(
  email: String!
  token: String!
): ConfirmAccount
```

### Arguments

#### [`email`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

E-mail of the user performing account confirmation.

#### [`token`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

A one-time token required to confirm the account.

### Type

#### [`ConfirmAccount`](/api-reference/users/objects/confirm-account.mdx)
