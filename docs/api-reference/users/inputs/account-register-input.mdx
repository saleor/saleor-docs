  metadata: [MetadataInput!]
  channel: String
}

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

User public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [<code>channel</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel which will be used to notify users. Optional when only one channel exists.
