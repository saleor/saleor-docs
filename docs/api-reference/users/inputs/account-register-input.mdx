```graphql
input AccountRegisterInput {
  firstName: String
  lastName: String
  languageCode: LanguageCodeEnum
  email: String!
  password: String!
  redirectUrl: String
  metadata: [MetadataInput!]
  channel: String
}
```

### Fields

#### [`firstName`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Given name.

#### [`lastName`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Family name.

#### [`languageCode`](#)<Bullet />[`LanguageCodeEnum`](/api-reference/miscellaneous/enums/language-code-enum.mdx)

User language code.

#### [`email`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

The email address of the user.

#### [`password`](#)<Bullet />[`String!`](/api-reference/miscellaneous/scalars/string.mdx)

Password.

#### [`redirectUrl`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Base of frontend URL that will be needed to create confirmation URL. Required when account confirmation is enabled.

#### [`metadata`](#)<Bullet />[`[MetadataInput!]`](/api-reference/miscellaneous/inputs/metadata-input.mdx)

User public metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [`channel`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Slug of a channel which will be used to notify users. Optional when only one channel exists.
