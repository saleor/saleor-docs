Fields required to update the user.

```graphql
input AccountInput {
  firstName: String
  lastName: String
  languageCode: LanguageCodeEnum
  defaultBillingAddress: AddressInput
  defaultShippingAddress: AddressInput
  metadata: [MetadataInput!]
}
```
