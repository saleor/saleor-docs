Fields required to update a staff user.

```graphql
input StaffUpdateInput {
  firstName: String
  lastName: String
  email: String
  isActive: Boolean
  note: String
  metadata: [MetadataInput!]
  privateMetadata: [MetadataInput!]
  addGroups: [ID!]
  removeGroups: [ID!]
}
```

### Fields

#### [<code>firstName</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Given name.

#### [<code>lastName</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

Family name.

#### [<code>email</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

The unique email address of the user.

#### [<code>isActive</code>](#)<Bullet />[<code>Boolean</code>](/api-reference/miscellaneous/scalars/boolean.mdx)

User account is active.

#### [<code>note</code>](#)<Bullet />[<code>String</code>](/api-reference/miscellaneous/scalars/string.mdx)

A note about the user.

#### [<code>metadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the user metadata. Can be read by any API client authorized to read the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [<code>privateMetadata</code>](#)<Bullet />[<code>[MetadataInput!]</code>](/api-reference/miscellaneous/inputs/metadata-input.mdx)

Fields required to update the user private metadata. Requires permissions to modify and to read the metadata of the object it's attached to.

Warning: never store sensitive information, including financial data such as credit card details.

#### [<code>addGroups</code>](#)<Bullet />[<code>[ID!]</code>](/api-reference/miscellaneous/scalars/id.mdx)

List of permission group IDs to which user should be assigned.

#### [<code>removeGroups</code>](#)<Bullet />[<code>[ID!]</code>](/api-reference/miscellaneous/scalars/id.mdx)

List of permission group IDs from which user should be unassigned.
