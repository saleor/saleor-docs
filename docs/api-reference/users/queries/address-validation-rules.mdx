Returns address validation rules.

```graphql
addressValidationRules(
  countryCode: CountryCode!
  countryArea: String
  city: String
  cityArea: String
): AddressValidationData
```

### Arguments

#### [`countryCode`](#)<Bullet />[`CountryCode!`](/api-reference/miscellaneous/enums/country-code.mdx)

Two-letter ISO 3166-1 country code.

#### [`countryArea`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Designation of a region, province or state.

#### [`city`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

City or a town name.

#### [`cityArea`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Sublocality like a district.

### Type

#### [`AddressValidationData`](/api-reference/users/objects/address-validation-data.mdx)

Represents address validation rules for a country.
