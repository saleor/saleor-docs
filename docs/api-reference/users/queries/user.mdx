Look up a user by ID or email address.

Requires one of the following permissions: MANAGE_STAFF, MANAGE_USERS, MANAGE_ORDERS.

```graphql
user(
  id: ID
  email: String
  externalReference: String
): User
```

### Arguments

#### [`id`](#)<Bullet />[`ID`](/api-reference/miscellaneous/scalars/id.mdx)

ID of the user.

#### [`email`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

Email address of the user.

#### [`externalReference`](#)<Bullet />[`String`](/api-reference/miscellaneous/scalars/string.mdx)

External ID of the user.

### Type

#### [`User`](/api-reference/users/objects/user.mdx)

Represents user data.
