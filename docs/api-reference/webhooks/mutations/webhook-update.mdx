Updates a webhook subscription.

Requires one of the following permissions: MANAGE_APPS, AUTHENTICATED_APP.

```graphql
webhookUpdate(
  id: ID!
  input: WebhookUpdateInput!
): WebhookUpdate
```

### Arguments

#### [`id`](#)<Bullet />[`ID!`](/api-reference/miscellaneous/scalars/id.mdx)

ID of a webhook to update.

#### [`input`](#)<Bullet />[`WebhookUpdateInput!`](/api-reference/webhooks/inputs/webhook-update-input.mdx)

Fields required to update a webhook.

### Type

#### [`WebhookUpdate`](/api-reference/webhooks/objects/webhook-update.mdx)
