Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productDelete(
  externalReference: String
  id: ID
): ProductDelete
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>productDelete.<b>externalReference</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> External ID of a product to delete.

Added in Saleor 3.10.

>

#### [<code style={{ fontWeight: 'normal' }}>productDelete.<b>id</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of a product to delete.

### Type

#### [`ProductDelete`](../../api-reference/objects/product-delete) <Badge class="secondary" text="object"/>

> Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.
