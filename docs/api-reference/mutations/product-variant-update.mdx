Updates an existing variant for product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantUpdate(
  externalReference: String
  id: ID
  input: ProductVariantInput!
  sku: String
): ProductVariantUpdate
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>productVariantUpdate.<b>externalReference</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> External ID of a product variant to update.

Added in Saleor 3.10.

#### [<code style={{ fontWeight: 'normal' }}>productVariantUpdate.<b>id</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>
