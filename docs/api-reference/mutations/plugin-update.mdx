Update plugin configuration.

Requires one of the following permissions: MANAGE_PLUGINS.

```graphql
pluginUpdate(
  channelId: ID
  id: ID!
  input: PluginUpdateInput!
): PluginUpdate
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>pluginUpdate.<b>channelId</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of a channel for which the data should be modified.

#### [<code style={{ fontWeight: 'normal' }}>pluginUpdate.<b>id</b></code>](#)<Bullet />[`ID!`](../../api-reference/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> ID of plugin to update.

#### [<code style={{ fontWeight: 'normal' }}>pluginUpdate.<b>input</b></code>](#)<Bullet />[`PluginUpdateInput!`](../../api-reference/inputs/plugin-update-input) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>

> Fields required to update a plugin configuration.
