Deletes an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeDelete(
  externalReference: String
  id: ID
): AttributeDelete
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>attributeDelete.<b>externalReference</b></code>](#)<Bullet />[`String`](../../api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> External ID of an attribute to delete.

Added in Saleor 3.10.

>

#### [<code style={{ fontWeight: 'normal' }}>attributeDelete.<b>id</b></code>](#)<Bullet />[`ID`](../../api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> ID of an attribute to delete.

### Type

#### [`AttributeDelete`](../../api-reference/objects/attribute-delete) <Badge class="secondary" text="object"/>

> Deletes an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
