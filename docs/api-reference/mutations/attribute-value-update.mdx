Updates value of an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeValueUpdate(
  externalReference: String
  id: ID
  input: AttributeValueUpdateInput!
): AttributeValueUpdate
```
