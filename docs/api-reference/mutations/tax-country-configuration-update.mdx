Update tax class rates for a specific country.

Added in Saleor 3.9.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: MANAGE_TAXES.

```graphql
taxCountryConfigurationUpdate(
  countryCode: CountryCode!
  updateTaxClassRates: [TaxClassRateInput!]!
): TaxCountryConfigurationUpdate
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>taxCountryConfigurationUpdate.<b>countryCode</b></code>](#)<Bullet />[`CountryCode!`](../../api-reference/enums/country-code) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="enum"/>

> Country in which to update the tax class rates.

#### [<code style={{ fontWeight: 'normal' }}>taxCountryConfigurationUpdate.<b>updateTaxClassRates</b></code>](#)<Bullet />[`[TaxClassRateInput!]!`](../../api-reference/inputs/tax-class-rate-input) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>

> List of tax rates per tax class to update. When `&#x007B;taxClass: id, rate: null`&#x007D; is passed, it deletes the rate object for given taxClass ID. When `&#x007B;rate: Int&#x007D;` is passed without a tax class, it updates the default tax class for this country.
