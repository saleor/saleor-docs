[Saleor App Template](https://github.com/saleor/saleor-app-template) is a minimalistic boilerplate/starter that provides a working example of the Saleor App.

We recommend starting the new App from this repository since it is up-to-date with best practices. You can also start with a specific [example](https://github.com/saleor/app-examples/tree/main/examples).

# Starting app from App Template

## With CLI

Ensure CLI is installed

```shell
npm i -g @saleor/cli
```

Then, run `saleor app create` to create a project or `saleor app create -h` to check available options.

## With App Template Gallery

Open [App Template Gallery](../../../../dashboard/app-templates-gallery.md) in Dashboard, find App Template, and follow the process. It will deploy App Template to Vercel and create a repository for you.

## Git clone

Run `git clone https://github.com/saleor/saleor-app-template.git`
