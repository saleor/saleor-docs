---
title: Admin Emails
---

The plugin uses the same template language, and helpers as described in [`UserEmails` plugin section](user-emails).


:::note

If the email configuration of the plugin on the dashboard side will be empty, the plugin will try to use these variables defined in settings.py file:

- `settings.EMAIL_HOST`
- `settings.EMAIL_PORT`
- `settings.EMAIL_HOST_USER`
- `settings.EMAIL_HOST_PASSWORD`
- `settings.DEFAULT_FROM_EMAIL`
- `settings.EMAIL_USE_TLS`
- `settings.EMAIL_USE_SSL`

:::
