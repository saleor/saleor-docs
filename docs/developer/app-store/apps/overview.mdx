# Apps

:::tip

Do you want to build your own app? Visit our [App Quickstart Guide](../../extending/apps/quickstart/getting-started).

:::

## Built-in apps catalog

Saleor App Store contains the following apps:

<CardGrid>

[**Adyen** Simplify your payment process and offer a seamless online shopping experience with Adyen.](adyen)

[**CRM** Synchronize customers with CRM platforms](crm)

[**Data importer** Import data from CSV to Saleor.](https://github.com/saleor/apps/tree/main/apps/data-importer)

[**Emails and messages** Notifications and email communication with customers.](emails-and-messages/overview)

[**Invoices** Generate invoice PDF for each order.](https://github.com/saleor/apps/tree/main/apps/invoices)

[**Klaviyo** Send Saleor events to Klaviyo, where you can notify the customers.](https://github.com/saleor/apps/tree/main/apps/klaviyo)

[**Search** Connect Saleor with search engines.](https://github.com/saleor/apps/tree/main/apps/search)

[**Slack** Get notifications on Slack channel from Saleor events.](https://github.com/saleor/apps/tree/main/apps/slack)

[**Taxes** Calculate order and checkout taxes using external services.](taxes)

</CardGrid>

## Suggest an app

If you have an idea for an app that you would like to see in the Saleor App Store, please [submit it](https://github.com/saleor/apps/discussions/categories/integrations-features) in the [saleor/apps](https://github.com/saleor/apps) repository.
