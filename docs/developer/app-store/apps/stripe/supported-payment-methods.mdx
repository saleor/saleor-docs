- [Card](https://docs.stripe.com/payments/cards)

- [Klarna](https://docs.stripe.com/payments/klarna)

- [SEPA Direct Debit](https://docs.stripe.com/payments/sepa-debit)
- [Link](https://docs.stripe.com/payments/link)

Ensure your Stripe Dashboard available methods match the list above.

:::warning

If you enable more methods, Stripe UI will show unsupported methods for the customer.
Trying to pay with it will cause App to return an error.

:::
