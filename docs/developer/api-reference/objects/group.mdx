Represents permission group data.

```graphql
type Group implements Node {
  id: ID!
  name: String!
  permissions: [Permission]
  users: [User]
  userCanManage: Boolean!
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `name` ([`String!`](/docs/developer/api-reference/scalars/string))

#### `permissions` ([`[Permission]`](/docs/developer/api-reference/objects/permission))

List of group permissions

#### `users` ([`[User]`](/docs/developer/api-reference/objects/user))

List of group users

#### `userCanManage` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

True, if the currently authenticated user has rights to manage a group.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
