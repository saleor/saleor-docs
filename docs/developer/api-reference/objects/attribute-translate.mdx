Creates/updates translations for an attribute. Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
type AttributeTranslate {
  translationErrors: [TranslationError!]!
  errors: [TranslationError!]!
  attribute: Attribute
}
```
