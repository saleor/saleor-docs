title: PaymentSource
---

Represents a payment source stored for user in payment gateway, such as credit card.

```graphql
type PaymentSource {
  gateway: String!
  paymentMethodId: String
  creditCardInfo: CreditCard
  metadata: [MetadataItem!]!
}
```

### Fields

#### `gateway` ([`String!`](../../../developer/api-reference/scalars/string))

Payment gateway name.

#### `paymentMethodId` ([`String`](../../../developer/api-reference/scalars/string))

ID of stored payment method.

#### `creditCardInfo` ([`CreditCard`](../../../developer/api-reference/objects/credit-card))

Stored credit card details if available.

#### `metadata` ([`[MetadataItem!]!`](../../../developer/api-reference/objects/metadata-item))

New in Saleor 3.1. List of public metadata items. Can be accessed without permissions.
