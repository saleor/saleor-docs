title: PaymentSource
---

Represents a payment source stored for user in payment gateway, such as credit card.

```graphql
type PaymentSource {
  gateway: String!
  paymentMethodId: String
  creditCardInfo: CreditCard
}
```

### Fields

#### `gateway` ([`String!`](/developer/api-reference/scalars/string.mdx))

Payment gateway name.

#### `paymentMethodId` ([`String`](/developer/api-reference/scalars/string.mdx))

ID of stored payment method.

#### `creditCardInfo` ([`CreditCard`](/developer/api-reference/objects/credit-card.mdx))

Stored credit card details if available.
