Contains all details related to the applied discount to the order.

```graphql
type OrderDiscount implements Node {
  id: ID!
  type: OrderDiscountType!
  valueType: DiscountValueTypeEnum!
  value: PositiveDecimal!
  name: String
  translatedName: String
  reason: String
  amount: Money!
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `type` ([`OrderDiscountType!`](/docs/developer/api-reference/enums/order-discount-type))

#### `valueType` ([`DiscountValueTypeEnum!`](/docs/developer/api-reference/enums/discount-value-type-enum))

Type of the discount: fixed or percent

#### `value` ([`PositiveDecimal!`](/docs/developer/api-reference/scalars/positive-decimal))

Value of the discount. Can store fixed value or percent value

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

#### `translatedName` ([`String`](/docs/developer/api-reference/scalars/string))

#### `reason` ([`String`](/docs/developer/api-reference/scalars/string))

Explanation for the applied discount.

#### `amount` ([`Money!`](/docs/developer/api-reference/objects/money))

Returns amount of discount.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
