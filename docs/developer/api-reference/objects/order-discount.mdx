Contains all details related to the applied discount to the order.

```graphql
type OrderDiscount implements Node {
  id: ID!
  type: OrderDiscountType!
  valueType: DiscountValueTypeEnum!
  value: PositiveDecimal!
  name: String
  translatedName: String
  reason: String
  amount: Money!
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `type` ([`OrderDiscountType!`](/developer/api-reference/enums/order-discount-type.mdx))

#### `valueType` ([`DiscountValueTypeEnum!`](/developer/api-reference/enums/discount-value-type-enum.mdx))

Type of the discount: fixed or percent

#### `value` ([`PositiveDecimal!`](/developer/api-reference/scalars/positive-decimal.mdx))

Value of the discount. Can store fixed value or percent value

#### `name` ([`String`](/developer/api-reference/scalars/string.mdx))

#### `translatedName` ([`String`](/developer/api-reference/scalars/string.mdx))

#### `reason` ([`String`](/developer/api-reference/scalars/string.mdx))

Explanation for the applied discount.

#### `amount` ([`Money!`](/developer/api-reference/objects/money.mdx))

Returns amount of discount.

### Interfaces

#### [`Node`](/developer/api-reference/interfaces/node.mdx)

An object with an ID
