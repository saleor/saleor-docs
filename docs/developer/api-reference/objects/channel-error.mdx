```graphql
type ChannelError {
  field: String
  message: String
  code: ChannelErrorCode!
  shippingZones: [ID!]
}
```

### Fields

#### `field` ([`String`](/docs/developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

The error message.

#### `code` ([`ChannelErrorCode!`](/docs/developer/api-reference/enums/channel-error-code))

The error code.

#### `shippingZones` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of shipping zone IDs which causes the error.
