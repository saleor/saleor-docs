title: FulfillmentCancel
---

Cancels existing fulfillment and optionally restocks items.

```graphql
type FulfillmentCancel {
  fulfillment: Fulfillment
  order: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `fulfillment` ([`Fulfillment`](/docs/developer/api-reference/objects/fulfillment))

A canceled fulfillment.

#### `order` ([`Order`](/docs/developer/api-reference/objects/order))

Order which fulfillment was cancelled.

#### `orderErrors` ([`[OrderError!]!`](/docs/developer/api-reference/objects/order-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
