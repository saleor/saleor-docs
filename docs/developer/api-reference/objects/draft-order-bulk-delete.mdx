Deletes draft orders. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type DraftOrderBulkDelete {
  count: Int!
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

Returns how many objects were affected.
