title: FulfillmentReturnProducts
---

Return products. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type FulfillmentReturnProducts {
  returnFulfillment: Fulfillment
  replaceFulfillment: Fulfillment
  order: Order
  replaceOrder: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `returnFulfillment` ([`Fulfillment`](../../../developer/api-reference/objects/fulfillment))

A return fulfillment.

#### `replaceFulfillment` ([`Fulfillment`](../../../developer/api-reference/objects/fulfillment))

A replace fulfillment.

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Order which fulfillment was returned.

#### `replaceOrder` ([`Order`](../../../developer/api-reference/objects/order))

A draft order which was created for products with replace flag.

#### `orderErrors` ([`OrderError`](../../../developer/api-reference/objects/order-error))

#### `errors` ([`OrderError`](../../../developer/api-reference/objects/order-error))
