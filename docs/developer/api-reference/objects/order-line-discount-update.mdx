Update discount for the order line. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderLineDiscountUpdate {
  orderLine: OrderLine
  order: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `orderLine` ([`OrderLine`](../../../developer/api-reference/objects/order-line))

Order line which has been discounted.

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Order which is related to the discounted line.
