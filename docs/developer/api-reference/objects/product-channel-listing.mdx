Represents product channel listing.

```graphql
type ProductChannelListing implements Node {
  id: ID!
  publicationDate: Date
  isPublished: Boolean!
  channel: Channel!
  visibleInListings: Boolean!
  availableForPurchase: Date
  discountedPrice: Money
  purchaseCost: MoneyRange
  margin: Margin
  isAvailableForPurchase: Boolean
  pricing(address: AddressInput): ProductPricingInfo
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `publicationDate` ([`Date`](/docs/developer/api-reference/scalars/date))

#### `isPublished` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

#### `channel` ([`Channel!`](/docs/developer/api-reference/objects/channel))

#### `visibleInListings` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

#### `availableForPurchase` ([`Date`](/docs/developer/api-reference/scalars/date))

#### `discountedPrice` ([`Money`](/docs/developer/api-reference/objects/money))

The price of the cheapest variant (including discounts).

#### `purchaseCost` ([`MoneyRange`](/docs/developer/api-reference/objects/money-range))

Purchase cost of product.

#### `margin` ([`Margin`](/docs/developer/api-reference/objects/margin))

Range of margin percentage value.

#### `isAvailableForPurchase` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the product is available for purchase.

#### `pricing` ([`ProductPricingInfo`](/docs/developer/api-reference/objects/product-pricing-info))

Lists the storefront product's pricing, the current price and discounts, only meant for displaying.

- ##### `address` ([`AddressInput`](/docs/developer/api-reference/inputs/address-input))

Destination address used to find warehouses where stock availability for this product is checked. If address is empty, uses `Shop.companyAddress` or fallbacks to server's `settings.DEFAULT_COUNTRY` configuration.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
