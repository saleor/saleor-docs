Represents app data.

```graphql
type AppExtension implements Node {
  id: ID!
  app: App!
  label: String!
  url: String!
  view: AppExtensionViewEnum!
  type: AppExtensionTypeEnum!
  target: AppExtensionTargetEnum!
  permissions: [Permission!]!
  accessToken: String
}
```

### Fields

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the object.

#### `app` ([`App!`](../../../developer/api-reference/objects/app))

#### `label` ([`String!`](../../../developer/api-reference/scalars/string))

Label of the extension to show in the dashboard.

#### `url` ([`String!`](../../../developer/api-reference/scalars/string))

URL of a view where extension's iframe is placed.

#### `view` ([`AppExtensionViewEnum!`](../../../developer/api-reference/enums/app-extension-view-enum))

Name of a view where extension's iframe will be mounted.

#### `type` ([`AppExtensionTypeEnum!`](../../../developer/api-reference/enums/app-extension-type-enum))

Type of a view where extension's iframe will be mounted.

#### `target` ([`AppExtensionTargetEnum!`](../../../developer/api-reference/enums/app-extension-target-enum))

Place where extension's iframe will be mounted.

#### `permissions` ([`[Permission!]!`](../../../developer/api-reference/objects/permission))

List of the app extension's permissions.

#### `accessToken` ([`String`](../../../developer/api-reference/scalars/string))

JWT token used to authenticate by thridparty app extension.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
