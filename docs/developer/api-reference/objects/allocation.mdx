Represents allocation.

```graphql
type Allocation implements Node {
  id: ID!
  quantity: Int!
  warehouse: Warehouse!
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `quantity` ([`Int!`](/docs/developer/api-reference/scalars/int))

Quantity allocated for orders.

#### `warehouse` ([`Warehouse!`](/docs/developer/api-reference/objects/warehouse))

The warehouse were items were allocated.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
