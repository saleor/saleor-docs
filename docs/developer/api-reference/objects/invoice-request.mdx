Request an invoice for the order using plugin. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type InvoiceRequest {
  order: Order
  invoiceErrors: [InvoiceError!]!
  errors: [InvoiceError!]!
  invoice: Invoice
}
```
