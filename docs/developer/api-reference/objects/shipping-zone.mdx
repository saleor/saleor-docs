---

Represents a shipping zone in the shop. Zones are the concept used only for grouping shipping methods in the dashboard, and are never exposed to the customers directly.

```graphql
type ShippingZone implements Node, ObjectWithMetadata {
  id: ID!
  name: String!
  default: Boolean!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  priceRange: MoneyRange
  countries: [CountryDisplay]
  shippingMethods: [ShippingMethod]
  warehouses: [Warehouse!]!
  channels: [Channel!]!
  description: String
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `name` ([`String!`](/docs/developer/api-reference/scalars/string))

#### `default` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

#### `privateMetadata` ([`[MetadataItem]!`](/docs/developer/api-reference/objects/metadata-item))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/docs/developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.

#### `priceRange` ([`MoneyRange`](/docs/developer/api-reference/objects/money-range))

Lowest and highest prices for the shipping.

#### `countries` ([`[CountryDisplay]`](/docs/developer/api-reference/objects/country-display))

List of countries available for the method.

#### `shippingMethods` ([`[ShippingMethod]`](/docs/developer/api-reference/objects/shipping-method))

List of shipping methods available for orders shipped to countries within this shipping zone.

#### `warehouses` ([`[Warehouse!]!`](/docs/developer/api-reference/objects/warehouse))

List of warehouses for shipping zone.

#### `channels` ([`[Channel!]!`](/docs/developer/api-reference/objects/channel))

List of channels for shipping zone.

#### `description` ([`String`](/docs/developer/api-reference/scalars/string))

Description of a shipping zone.
