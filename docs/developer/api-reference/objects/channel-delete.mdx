title: ChannelDelete

Delete a channel. Orders associated with the deleted channel will be moved to the target channel. Checkouts, product availability, and pricing will be removed.

```graphql
type ChannelDelete {
  channelErrors: [ChannelError!]!
  errors: [ChannelError!]!
  channel: Channel
}
```
