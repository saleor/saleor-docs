Manage sale's availability in channels. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
type SaleChannelListingUpdate {
  sale: Sale
  discountErrors: [DiscountError!]!
  errors: [DiscountError!]!
}
```

### Fields
