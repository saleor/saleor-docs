Deletes an order line from an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderLineDelete {
  order: Order
  orderLine: OrderLine
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```


### Fields

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

A related order.


#### `orderLine` ([`OrderLine`](../../../developer/api-reference/objects/order-line))

An order line that was deleted.
