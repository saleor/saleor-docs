id: shop
title: Shop
---

Represents a shop resource containing general shop data and configuration.

```graphql
type Shop {
  availablePaymentGateways(
    currency: String
    channel: String
  ): [PaymentGateway!]!
  availableExternalAuthentications: [ExternalAuthentication!]!
  availableShippingMethods(
    channel: String!
    address: AddressInput
  ): [ShippingMethod]
  channelCurrencies: [String!]!
  countries(languageCode: LanguageCodeEnum): [CountryDisplay!]!
  defaultCountry: CountryDisplay
  defaultMailSenderName: String
  defaultMailSenderAddress: String
  description: String
  domain: Domain!
  languages: [LanguageDisplay]!
  name: String!
  permissions: [Permission]!
  phonePrefixes: [String]!
  headerText: String
  includeTaxesInPrices: Boolean!
  fulfillmentAutoApprove: Boolean!
  fulfillmentAllowUnpaid: Boolean!
  displayGrossPrices: Boolean!
  chargeTaxesOnShipping: Boolean!
  trackInventoryByDefault: Boolean
  defaultWeightUnit: WeightUnitsEnum
  translation(languageCode: LanguageCodeEnum!): ShopTranslation
  automaticFulfillmentDigitalProducts: Boolean
  defaultDigitalMaxDownloads: Int
  defaultDigitalUrlValidDays: Int
  companyAddress: Address
  customerSetPasswordUrl: String
  staffNotificationRecipients: [StaffNotificationRecipient]
  limits: LimitInfo!
  version: String!
}
```

### Fields

#### `availablePaymentGateways` ([`[PaymentGateway!]!`](/docs/developer/api-reference/objects/payment-gateway))

List of available payment gateways.

- ##### `currency` ([`String`](/docs/developer/api-reference/scalars/string))

A currency for which gateways will be returned.

DEPRECATED: this field will be removed in Saleor 4.0. Use `channel` argument instead.

- ##### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `availableExternalAuthentications` ([`[ExternalAuthentication!]!`](/docs/developer/api-reference/objects/external-authentication))

List of available external authentications.

#### `availableShippingMethods` ([`[ShippingMethod]`](/docs/developer/api-reference/objects/shipping-method))

Shipping methods that are available for the shop.

- ##### `channel` ([`String!`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

- ##### `address` ([`AddressInput`](/docs/developer/api-reference/inputs/address-input))

Address for which available shipping methods should be returned.

#### `channelCurrencies` ([`[String!]!`](/docs/developer/api-reference/scalars/string))

New in Saleor 3.1. List of all currencies supported by shop's channels.

#### `countries` ([`[CountryDisplay!]!`](/docs/developer/api-reference/objects/country-display))

List of countries available in the shop.

- ##### `languageCode` ([`LanguageCodeEnum`](/docs/developer/api-reference/enums/language-code-enum))

A language code to return the translation for.

DEPRECATED: this field will be removed in Saleor 4.0.

#### `defaultCountry` ([`CountryDisplay`](/docs/developer/api-reference/objects/country-display))

Shop's default country.

#### `defaultMailSenderName` ([`String`](/docs/developer/api-reference/scalars/string))

Default shop's email sender's name.

#### `defaultMailSenderAddress` ([`String`](/docs/developer/api-reference/scalars/string))

Default shop's email sender's address.

#### `description` ([`String`](/docs/developer/api-reference/scalars/string))

Shop's description.

#### `domain` ([`Domain!`](/docs/developer/api-reference/objects/domain))

Shop's domain data.

#### `languages` ([`[LanguageDisplay]!`](/docs/developer/api-reference/objects/language-display))

List of the shops's supported languages.

#### `name` ([`String!`](/docs/developer/api-reference/scalars/string))

Shop's name.

#### `permissions` ([`[Permission]!`](/docs/developer/api-reference/objects/permission))

List of available permissions.

#### `phonePrefixes` ([`[String]!`](/docs/developer/api-reference/scalars/string))

List of possible phone prefixes.

#### `headerText` ([`String`](/docs/developer/api-reference/scalars/string))

Header text.

#### `includeTaxesInPrices` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Include taxes in prices.

#### `fulfillmentAutoApprove` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

New in Saleor 3.1. Automatically approve all new fulfillments.

#### `fulfillmentAllowUnpaid` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

New in Saleor 3.1. Allow to approve fulfillments which are unpaid.

#### `displayGrossPrices` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Display prices with tax in store.

#### `chargeTaxesOnShipping` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Charge taxes on shipping.

#### `trackInventoryByDefault` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Enable inventory tracking.

#### `defaultWeightUnit` ([`WeightUnitsEnum`](/docs/developer/api-reference/enums/weight-units-enum))

Default weight unit.

#### `translation` ([`ShopTranslation`](/docs/developer/api-reference/objects/shop-translation))

Returns translated shop fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

A language code to return the translation for shop.

#### `automaticFulfillmentDigitalProducts` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Enable automatic fulfillment for all digital products.

#### `defaultDigitalMaxDownloads` ([`Int`](/docs/developer/api-reference/scalars/int))

Default number of max downloads per digital content URL.

#### `defaultDigitalUrlValidDays` ([`Int`](/docs/developer/api-reference/scalars/int))

Default number of days which digital content URL will be valid.

#### `companyAddress` ([`Address`](/docs/developer/api-reference/objects/address))

Company address.

#### `customerSetPasswordUrl` ([`String`](/docs/developer/api-reference/scalars/string))

URL of a view where customers can set their password.

#### `staffNotificationRecipients` ([`[StaffNotificationRecipient]`](/docs/developer/api-reference/objects/staff-notification-recipient))

List of staff notification recipients.

#### `limits` ([`LimitInfo!`](/docs/developer/api-reference/objects/limit-info))

Resource limitations and current usage if any set for a shop

#### `version` ([`String!`](/docs/developer/api-reference/scalars/string))

Saleor API version.
