Represents a shop resource containing general shop data and configuration.

```graphql
type Shop {
  availablePaymentGateways(
    currency: String
    channel: String
  ): [PaymentGateway!]!
  availableExternalAuthentications: [ExternalAuthentication!]!
  availableShippingMethods(
    channel: String!
    address: AddressInput
  ): [ShippingMethod]
  countries(languageCode: LanguageCodeEnum): [CountryDisplay!]!
  defaultCountry: CountryDisplay
  defaultMailSenderName: String
  defaultMailSenderAddress: String
  description: String
  domain: Domain!
  languages: [LanguageDisplay]!
  name: String!
  permissions: [Permission]!
  phonePrefixes: [String]!
  headerText: String
  includeTaxesInPrices: Boolean!
  displayGrossPrices: Boolean!
  chargeTaxesOnShipping: Boolean!
  trackInventoryByDefault: Boolean
  defaultWeightUnit: WeightUnitsEnum
  translation(languageCode: LanguageCodeEnum!): ShopTranslation
  automaticFulfillmentDigitalProducts: Boolean
  defaultDigitalMaxDownloads: Int
  defaultDigitalUrlValidDays: Int
  companyAddress: Address
  customerSetPasswordUrl: String
  staffNotificationRecipients: [StaffNotificationRecipient]
  limits: LimitInfo!
  version: String!
}
```

### Fields

#### `availablePaymentGateways` ([`[PaymentGateway!]!`](/developer/api-reference/objects/payment-gateway.mdx))

List of available payment gateways.

- ##### `currency` ([`String`](/developer/api-reference/scalars/string.mdx))

DEPRECATED: use `channel` argument instead. This argument will be removed in Saleor 4.0.A currency for which gateways will be returned.

- ##### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `availableExternalAuthentications` ([`[ExternalAuthentication!]!`](/developer/api-reference/objects/external-authentication.mdx))

List of available external authentications.

#### `availableShippingMethods` ([`[ShippingMethod]`](/developer/api-reference/objects/shipping-method.mdx))

Shipping methods that are available for the shop.

- ##### `channel` ([`String!`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

- ##### `address` ([`AddressInput`](/developer/api-reference/inputs/address-input.mdx))

Address for which available shipping methods should be returned.

#### `countries` ([`[CountryDisplay!]!`](/developer/api-reference/objects/country-display.mdx))

List of countries available in the shop.

- ##### `languageCode` ([`LanguageCodeEnum`](/developer/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for.

#### `defaultCountry` ([`CountryDisplay`](/developer/api-reference/objects/country-display.mdx))

#### `defaultMailSenderAddress` ([`String`](/developer/api-reference/scalars/string.mdx))
