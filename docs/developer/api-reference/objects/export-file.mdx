Represents a job data of exported file.

```graphql
type ExportFile implements Node, Job {
  id: ID!
  status: JobStatusEnum!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
  url: String
  events: [ExportEvent!]
  user: User
  app: App
}
```

### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

#### `status` ([`JobStatusEnum`](../../../developer/api-reference/enums/job-status-enum))

Job status.

#### `createdAt` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Created date time of job in ISO 8601 format.

#### `updatedAt` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Date time of job last update in ISO 8601 format.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

Job message.

#### `url` ([`String`](../../../developer/api-reference/scalars/string))

The URL of field to download.

#### `events` ([`ExportEvent`](../../../developer/api-reference/objects/export-event))

List of events associated with the export.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

#### `app` ([`App`](../../../developer/api-reference/objects/app))
