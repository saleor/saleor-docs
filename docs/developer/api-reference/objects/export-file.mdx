Represents a job data of exported file.

```graphql
type ExportFile implements Node, Job {
  id: ID!
  user: User
  app: App
  status: JobStatusEnum!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
  url: String
  events: [ExportEvent!]
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `user` ([`User`](/docs/developer/api-reference/objects/user))

#### `app` ([`App`](/docs/developer/api-reference/objects/app))

#### `status` ([`JobStatusEnum!`](/docs/developer/api-reference/enums/job-status-enum))

Job status.

#### `createdAt` ([`DateTime!`](/docs/developer/api-reference/scalars/date-time))

Created date time of job in ISO 8601 format.

#### `updatedAt` ([`DateTime!`](/docs/developer/api-reference/scalars/date-time))

Date time of job last update in ISO 8601 format.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

Job message.

#### `url` ([`String`](/docs/developer/api-reference/scalars/string))

The URL of field to download.

#### `events` ([`[ExportEvent!]`](/docs/developer/api-reference/objects/export-event))

List of events associated with the export.
