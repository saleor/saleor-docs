Represents a collection of products.

```graphql
type Collection implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  metadata: [MetadataItem!]!
  seoTitle: String
  seoDescription: String
  name: String!
  description: JSONString
  slug: String!
  channel: String
  descriptionJson: JSONString
  products(
    filter: ProductFilterInput
    sortBy: ProductOrder
    before: String
    after: String
    first: Int
    last: Int
  ): ProductCountableConnection
  backgroundImage(size: Int): Image
  translation(languageCode: LanguageCodeEnum!): CollectionTranslation
  channelListings: [CollectionChannelListing!]
}
```

#### `metadata` ([`[MetadataItem!]!`](../../../developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.

#### `products` ([`ProductCountableConnection`](../../../developer/api-reference/objects/product-countable-connection))
