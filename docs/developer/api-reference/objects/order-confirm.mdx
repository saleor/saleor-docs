Confirms an unconfirmed order by changing status to unfulfilled.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderConfirm {
  order: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```
