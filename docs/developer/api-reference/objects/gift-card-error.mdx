```graphql
type GiftCardError {
  field: String
  message: String
  code: GiftCardErrorCode!
  tags: [String!]
}
```


### Fields

#### `field` ([`String`](../../../developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.


#### `message` ([`String`](../../../developer/api-reference/scalars/string))

The error message.


#### `code` ([`GiftCardErrorCode`](../../../developer/api-reference/enums/gift-card-error-code))

The error code.


#### `tags` ([`String`](../../../developer/api-reference/scalars/string))

List of tag values that cause the error.
