title: GiftCardAddNote
---


New in Saleor 3.1. Adds note to the gift card. Note: this feature is in a preview state and can be subject to changes at later point. Requires one of the following permissions: MANAGE_GIFT_CARD.

```graphql
type GiftCardAddNote {
  giftCard: GiftCard
  event: GiftCardEvent
  errors: [GiftCardError!]!
}
```


### Fields

#### `giftCard` ([`GiftCard`](../../../developer/api-reference/objects/gift-card))

Gift card with the note added.


#### `event` ([`GiftCardEvent`](../../../developer/api-reference/objects/gift-card-event))

Gift card note created.


#### `errors` ([`GiftCardError`](../../../developer/api-reference/objects/gift-card-error))
