title: GiftCardAddNote
---

New in Saleor 3.1. Adds note to the gift card.

```graphql
type GiftCardAddNote {
  giftCard: GiftCard
  event: GiftCardEvent
  errors: [GiftCardError!]!
}
```

### Fields

#### `giftCard` ([`GiftCard`](/docs/developer/api-reference/objects/gift-card))

Gift card with the note added.

#### `event` ([`GiftCardEvent`](/docs/developer/api-reference/objects/gift-card-event))

Gift card note created.

#### `errors` ([`[GiftCardError!]!`](/docs/developer/api-reference/objects/gift-card-error))
