Verify external authentication data by plugin.

```graphql
type ExternalVerify {
  user: User
  isValid: Boolean!
  verifyData: JSONString
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

User assigned to data.

#### `isValid` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

Determine if authentication data is valid or not.

#### `verifyData` ([`JSONString`](/developer/api-reference/scalars/json-string.mdx))

External data.
