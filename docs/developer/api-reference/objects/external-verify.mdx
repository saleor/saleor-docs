Verify external authentication data by plugin.

```graphql
type ExternalVerify {
  user: User
  isValid: Boolean!
  verifyData: JSONString
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

### Fields

#### `user` ([`User`](/docs/developer/api-reference/objects/user))

User assigned to data.

#### `isValid` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Determine if authentication data is valid or not.

#### `verifyData` ([`JSONString`](/docs/developer/api-reference/scalars/json-string))

External data.
