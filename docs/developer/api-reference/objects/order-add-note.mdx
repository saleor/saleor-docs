Adds note to the order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderAddNote {
  order: Order
  event: OrderEvent
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Order with the note added.

#### `event` ([`OrderEvent`](../../../developer/api-reference/objects/order-event))

Order note created.
