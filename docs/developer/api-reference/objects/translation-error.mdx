```graphql
type TranslationError {
  field: String
  message: String
  code: TranslationErrorCode!
}
```

### Fields

#### `field` ([`String`](/docs/developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

The error message.

#### `code` ([`TranslationErrorCode!`](/docs/developer/api-reference/enums/translation-error-code))

The error code.
