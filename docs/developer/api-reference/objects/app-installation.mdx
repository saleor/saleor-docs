Represents ongoing installation of app.

```graphql
type AppInstallation implements Node, Job {
  appName: String!
  manifestUrl: String!
  id: ID!
  status: JobStatusEnum!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
}
```

### Fields

#### `manifestUrl` ([`String!`](/docs/developer/api-reference/scalars/string))

The ID of the object.

#### `status` ([`JobStatusEnum!`](/docs/developer/api-reference/enums/job-status-enum))

Job status.

#### `createdAt` ([`DateTime!`](/docs/developer/api-reference/scalars/date-time))

Created date time of job in ISO 8601 format.

#### `updatedAt` ([`DateTime!`](/docs/developer/api-reference/scalars/date-time))

Date time of job last update in ISO 8601 format.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

Job message.
