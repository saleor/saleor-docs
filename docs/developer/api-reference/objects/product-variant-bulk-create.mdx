Creates product variants for a given product. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
type ProductVariantBulkCreate {
  count: Int!
  productVariants: [ProductVariant!]!
  bulkProductErrors: [BulkProductError!]!
  errors: [BulkProductError!]!
}
```


### Fields

#### `count` ([`Int`](../../../developer/api-reference/scalars/int))

Returns how many objects were created.


#### `productVariants` ([`ProductVariant`](../../../developer/api-reference/objects/product-variant))

List of the created variants.
