Creates product variants for a given product.

```graphql
type ProductVariantBulkCreate {
  count: Int!

  bulkProductErrors: [BulkProductError!]!
```

### Fields

#### `count` ([`Int!`](/docs/developer/api-reference/scalars/int))

Returns how many objects were created.

#### `productVariants` ([`[ProductVariant!]!`](/docs/developer/api-reference/objects/product-variant))
