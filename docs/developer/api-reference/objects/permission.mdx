Represents a permission object in a friendly form.

```graphql
type Permission {
  code: PermissionEnum!
  name: String!
}
```

### Fields

#### `code` ([`PermissionEnum!`](/developer/api-reference/enums/permission-enum.mdx))

Internal code for permission.

#### `name` ([`String!`](/developer/api-reference/scalars/string.mdx))

Describe action(s) allowed to do by permission.
