History log of export file.

```graphql
type ExportEvent implements Node {
  id: ID!
  date: DateTime!
  type: ExportEventsEnum!
  user: User
  app: App
  message: String!
}
```

### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the object.

#### `date` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Date when event happened at in ISO 8601 format.

#### `type` ([`ExportEventsEnum`](../../../developer/api-reference/enums/export-events-enum))

Export event type.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

User who performed the action. Requires one of the following permissions: AuthorizationFilters.OWNER, AccountPermissions.MANAGE_STAFF.

#### `app` ([`App`](../../../developer/api-reference/objects/app))

App which performed the action. Requires one of the following permissions: AuthorizationFilters.OWNER, AppPermission.MANAGE_APPS.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

Content of the event.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
