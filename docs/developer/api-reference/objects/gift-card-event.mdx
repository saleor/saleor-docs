History log of the gift card.

```graphql
type GiftCardEvent implements Node {
  id: ID!
  date: DateTime
  type: GiftCardEventsEnum
  user: User
  app: App
  message: String
  email: String
  orderId: ID
  orderNumber: String
  tag: String
  oldTag: String
  balance: GiftCardEventBalance
  expiry: GiftCardEventExpiry
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `date` ([`DateTime`](/developer/api-reference/scalars/date-time.mdx))

Date when event happened at in ISO 8601 format.

#### `type` ([`GiftCardEventsEnum`](/developer/api-reference/enums/gift-card-events-enum.mdx))

Gift card event type.

#### `user` ([`User`](/developer/api-reference/objects/user.mdx))

User who performed the action.

#### `app` ([`App`](/developer/api-reference/objects/app.mdx))

App that performed the action.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

Content of the event.

#### `email` ([`String`](/developer/api-reference/scalars/string.mdx))

Email of the customer.

#### `orderId` ([`ID`](/developer/api-reference/scalars/id.mdx))

The order ID where gift card was used or bought.

#### `orderNumber` ([`String`](/developer/api-reference/scalars/string.mdx))

User-friendly number of an order where gift card was used or bought.

#### `tag` ([`String`](/developer/api-reference/scalars/string.mdx))

The gift card tag.

#### `oldTag` ([`String`](/developer/api-reference/scalars/string.mdx))

Old gift card tag.

#### `balance` ([`GiftCardEventBalance`](/developer/api-reference/objects/gift-card-event-balance.mdx))

The gift card balance.

#### `expiry` ([`GiftCardEventExpiry`](/developer/api-reference/objects/gift-card-event-expiry.mdx))

The gift card expiry settings.
