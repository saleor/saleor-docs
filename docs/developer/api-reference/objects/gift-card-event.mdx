New in Saleor 3.1. History log of the gift card.

```graphql
type GiftCardEvent implements Node {
  id: ID!
  date: DateTime
  type: GiftCardEventsEnum
  user: User
  app: App
  message: String
  email: String
  orderId: ID
  orderNumber: String
  tags: [String!]
  oldTags: [String!]
  balance: GiftCardEventBalance
  expiryDate: Date
  oldExpiryDate: Date
}
```

### Fields

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

#### `date` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Date when event happened at in ISO 8601 format.

#### `type` ([`GiftCardEventsEnum`](../../../developer/api-reference/enums/gift-card-events-enum))

Gift card event type.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

User who performed the action.

#### `app` ([`App`](../../../developer/api-reference/objects/app))

App that performed the action.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

Content of the event.

#### `email` ([`String`](../../../developer/api-reference/scalars/string))

Email of the customer.

#### `orderId` ([`ID`](../../../developer/api-reference/scalars/id))

The order ID where gift card was used or bought.

#### `orderNumber` ([`String`](../../../developer/api-reference/scalars/string))

User-friendly number of an order where gift card was used or bought.

#### `tags` ([`[String!]`](../../../developer/api-reference/scalars/string))

The list of gift card tags.

#### `oldTags` ([`[String!]`](../../../developer/api-reference/scalars/string))

The list of old gift card tags.

#### `balance` ([`GiftCardEventBalance`](../../../developer/api-reference/objects/gift-card-event-balance))

The gift card balance.

#### `expiryDate` ([`Date`](../../../developer/api-reference/scalars/date))

The gift card expiry date.

#### `oldExpiryDate` ([`Date`](../../../developer/api-reference/scalars/date))

Previous gift card expiry date.
