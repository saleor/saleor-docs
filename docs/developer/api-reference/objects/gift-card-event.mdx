New in Saleor 3.1. History log of the gift card.

```graphql
type GiftCardEvent implements Node {
  id: ID!
  date: DateTime
  type: GiftCardEventsEnum
  user: User
  app: App
  message: String
  email: String
  orderId: ID
  orderNumber: String
  tag: String
  oldTag: String
  balance: GiftCardEventBalance
  expiryDate: Date
  oldExpiryDate: Date
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `date` ([`DateTime`](/docs/developer/api-reference/scalars/date-time))

Date when event happened at in ISO 8601 format.

#### `type` ([`GiftCardEventsEnum`](/docs/developer/api-reference/enums/gift-card-events-enum))

Gift card event type.

#### `user` ([`User`](/docs/developer/api-reference/objects/user))

User who performed the action.

#### `app` ([`App`](/docs/developer/api-reference/objects/app))

App that performed the action.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

Content of the event.

#### `email` ([`String`](/docs/developer/api-reference/scalars/string))

Email of the customer.

#### `orderId` ([`ID`](/docs/developer/api-reference/scalars/id))

The order ID where gift card was used or bought.

#### `orderNumber` ([`String`](/docs/developer/api-reference/scalars/string))

User-friendly number of an order where gift card was used or bought.

#### `tag` ([`String`](/docs/developer/api-reference/scalars/string))

The gift card tag.

#### `oldTag` ([`String`](/docs/developer/api-reference/scalars/string))

Old gift card tag.

#### `balance` ([`GiftCardEventBalance`](/docs/developer/api-reference/objects/gift-card-event-balance))

The gift card balance.

#### `expiryDate` ([`Date`](/docs/developer/api-reference/scalars/date))

The gift card expiry date.

#### `oldExpiryDate` ([`Date`](/docs/developer/api-reference/scalars/date))

Previous gift card expiry date.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
