Assigns storefront's navigation menus.

Requires one of the following permissions: MANAGE_MENUS, MANAGE_SETTINGS.

```graphql
type AssignNavigation {
  menu: Menu
  menuErrors: [MenuError!]!
  errors: [MenuError!]!
}
```
