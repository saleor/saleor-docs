Updates an address. Requires one of the following permissions: MANAGE_USERS.

```graphql
type AddressUpdate {
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
  address: Address
}
```

### Fields

#### `user` ([`User`](../../../developer/api-reference/objects/user))

A user object for which the address was edited.
