Represents a reduced VAT rate for a particular type of goods.

```graphql
type ReducedRate {
  rate: Float!
  rateType: String!
}
```

### Fields

#### `rate` ([`Float!`](../../../developer/api-reference/scalars/float))

Reduced VAT rate in percent.

#### `rateType` ([`String!`](../../../developer/api-reference/scalars/string))

A type of goods.
