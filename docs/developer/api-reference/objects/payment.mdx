Represents a payment of a given type.

```graphql
type Payment implements Node, ObjectWithMetadata {
  id: ID!
  gateway: String!
  isActive: Boolean!
  created: DateTime!
  modified: DateTime!
  token: String!
  checkout: Checkout
  order: Order
  paymentMethodType: String!
  customerIpAddress: String
  pspReference: String
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  chargeStatus: PaymentChargeStatusEnum!
  actions: [OrderAction]!
  partial: Boolean
  total: Money
  capturedAmount: Money
  transactions: [Transaction]
  availableCaptureAmount: Money
  availableRefundAmount: Money
  creditCard: CreditCard
  gatewayName: String!
}
```

#### `actions` ([`[OrderAction]!`](../../../developer/api-reference/enums/order-action))

List of actions that can be performed in the current state of a payment.

#### `partial` ([`[Boolean]!`](../../../developer/api-reference/scalars/boolean))

#### `total` ([`Money`](../../../developer/api-reference/objects/money))

#### `capturedAmount` ([`Money`](../../../developer/api-reference/objects/money))

#### `transactions` ([`[Transaction]`](../../../developer/api-reference/objects/transaction))

#### `availableCaptureAmount` ([`Money`](../../../developer/api-reference/objects/money))

#### `availableRefundAmount` ([`Money`](../../../developer/api-reference/objects/money))

#### `creditCard` ([`CreditCard`](../../../developer/api-reference/objects/credit-card))

The details of the card used for this payment.

#### `gatewayName` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.0. A human-readable name of the payment gateway plugin.
