title: VariantPricingInfo
---

Represents availability of a variant in the storefront.

```graphql
type VariantPricingInfo {
  onSale: Boolean
  discount: TaxedMoney
  discountLocalCurrency: TaxedMoney
  price: TaxedMoney
  priceUndiscounted: TaxedMoney
  priceLocalCurrency: TaxedMoney
}
```

### Fields

#### `onSale` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether it is in sale or not.

#### `discount` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The discount amount if in sale (null otherwise).

#### `discountLocalCurrency` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The discount amount in the local currency.

#### `price` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The price, with any discount subtracted.

#### `priceUndiscounted` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The price without any discount.

#### `priceLocalCurrency` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The discounted price in the local currency.
