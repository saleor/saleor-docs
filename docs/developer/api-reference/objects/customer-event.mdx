History log of the customer.

```graphql
type CustomerEvent implements Node {
  id: ID!
  date: DateTime
  type: CustomerEventsEnum
  user: User
  message: String
  count: Int
  order: Order
  orderLine: OrderLine
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `date` ([`DateTime`](/developer/api-reference/scalars/date-time.mdx))

Date when event happened at in ISO 8601 format.

#### `type` ([`CustomerEventsEnum`](/developer/api-reference/enums/customer-events-enum.mdx))

Customer event type.

#### `user` ([`User`](/developer/api-reference/objects/user.mdx))

User who performed the action.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

Content of the event.

#### `count` ([`Int`](/developer/api-reference/scalars/int.mdx))

Number of objects concerned by the event.

#### `order` ([`Order`](/developer/api-reference/objects/order.mdx))

The concerned order.

#### `orderLine` ([`OrderLine`](/developer/api-reference/objects/order-line.mdx))

The concerned order line.

### Interfaces

#### [`Node`](/developer/api-reference/interfaces/node.mdx)

An object with an ID
