History log of the customer.

```graphql
type CustomerEvent implements Node {
  id: ID!
  date: DateTime
  type: CustomerEventsEnum
  user: User
  app: App
  message: String
  count: Int
  order: Order
  orderLine: OrderLine
}
```

### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

#### `date` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Date when event happened at in ISO 8601 format.

#### `type` ([`CustomerEventsEnum`](../../../developer/api-reference/enums/customer-events-enum))

Customer event type.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

User who performed the action.

#### `app` ([`App`](../../../developer/api-reference/objects/app))

App that performed the action.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

Content of the event.

#### `count` ([`Int`](../../../developer/api-reference/scalars/int))

Number of objects concerned by the event.

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

The concerned order.

#### `orderLine` ([`OrderLine`](../../../developer/api-reference/objects/order-line))

The concerned order line.
