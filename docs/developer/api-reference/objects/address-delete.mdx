Deletes an address. Requires one of the following permissions: MANAGE_USERS.

```graphql
type AddressDelete {
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
  address: Address
}
```

### Fields

#### `user` ([`User`](../../../developer/api-reference/objects/user))

A user instance for which the address was deleted.
