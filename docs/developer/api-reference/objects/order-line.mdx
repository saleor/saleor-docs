Represents order line of particular order.

```graphql
type OrderLine implements Node {
  id: ID!
  productName: String!
  variantName: String!
  productSku: String!
  isShippingRequired: Boolean!
  quantity: Int!
  quantityFulfilled: Int!
  unitDiscountReason: String
  taxRate: Float!
  digitalContentUrl: DigitalContentUrl
  thumbnail(size: Int): Image
  unitPrice: TaxedMoney!
  undiscountedUnitPrice: TaxedMoney!
  unitDiscount: Money!
  unitDiscountValue: PositiveDecimal!
  totalPrice: TaxedMoney!
  variant: ProductVariant
  translatedProductName: String!
  translatedVariantName: String!
  allocations: [Allocation!]
  quantityToFulfill: Int!
  unitDiscountType: DiscountValueTypeEnum
}
```

### Fields

#### `productName` ([`String!`](/docs/developer/api-reference/scalars/string))

- ##### `size` ([`Int`](/docs/developer/api-reference/scalars/int))

Size of thumbnail.

#### `unitPrice` ([`TaxedMoney!`](/docs/developer/api-reference/objects/taxed-money))

Price of the single item in the order line.

#### `undiscountedUnitPrice` ([`TaxedMoney!`](/docs/developer/api-reference/objects/taxed-money))

Price of the single item in the order line without applied an order line discount.

#### `unitDiscount` ([`Money!`](/docs/developer/api-reference/objects/money))

The discount applied to the single order line.

#### `unitDiscountValue` ([`PositiveDecimal!`](/docs/developer/api-reference/scalars/positive-decimal))

Value of the discount. Can store fixed value or percent value

#### `totalPrice` ([`TaxedMoney!`](/docs/developer/api-reference/objects/taxed-money))

Price of the order line.

#### `variant` ([`ProductVariant`](/docs/developer/api-reference/objects/product-variant))

A purchased product variant. Note: this field may be null if the variant has been removed from stock at all.

#### `translatedProductName` ([`String!`](/docs/developer/api-reference/scalars/string))

Product name in the customer's language

#### `translatedVariantName` ([`String!`](/docs/developer/api-reference/scalars/string))

Variant name in the customer's language

#### `allocations` ([`[Allocation!]`](/docs/developer/api-reference/objects/allocation))

List of allocations across warehouses.

#### `quantityToFulfill` ([`Int!`](/docs/developer/api-reference/scalars/int))

New in Saleor 3.1. A quantity of items remaining to be fulfilled.

#### `unitDiscountType` ([`DiscountValueTypeEnum`](/docs/developer/api-reference/enums/discount-value-type-enum))

Type of the discount: fixed or percent
