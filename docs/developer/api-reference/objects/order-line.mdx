Represents order line of particular order.

```graphql
type OrderLine implements Node {
  id: ID!
  productName: String!
  variantName: String!
  productSku: String!
  isShippingRequired: Boolean!
  quantity: Int!
  quantityFulfilled: Int!
  unitDiscountReason: String
  taxRate: Float!
  digitalContentUrl: DigitalContentUrl
  thumbnail(size: Int = 0): Image
  unitPrice: TaxedMoney!
  undiscountedUnitPrice: TaxedMoney!
  unitDiscount: Money!
  unitDiscountValue: PositiveDecimal!
  totalPrice: TaxedMoney!
  variant: ProductVariant
  translatedProductName: String!
  translatedVariantName: String!
  allocations: [Allocation!]
  quantityToFulfill: Int!
  unitDiscountType: DiscountValueTypeEnum
}
```

#### `unitPrice` ([`TaxedMoney!`](/developer/api-reference/objects/taxed-money.mdx))

Price of the single item in the order line.

#### `undiscountedUnitPrice` ([`TaxedMoney!`](/developer/api-reference/objects/taxed-money.mdx))

Price of the single item in the order line without applied an order line discount.

#### `unitDiscount` ([`Money!`](/developer/api-reference/objects/money.mdx))

The discount applied to the single order line.

#### `unitDiscountValue` ([`PositiveDecimal!`](/developer/api-reference/scalars/positive-decimal.mdx))

Value of the discount. Can store fixed value or percent value

#### `totalPrice` ([`TaxedMoney!`](/developer/api-reference/objects/taxed-money.mdx))

#### `variant` ([`ProductVariant`](/developer/api-reference/objects/product-variant.mdx))

A purchased product variant. Note: this field may be null if the variant has been removed from stock at all.

#### `translatedProductName` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `translatedVariantName` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `allocations` ([`[Allocation!]`](/developer/api-reference/objects/allocation.mdx))

List of allocations across warehouses.

#### `quantityToFulfill` ([`Int!`](/developer/api-reference/scalars/int.mdx))

#### `unitDiscountType` ([`DiscountValueTypeEnum`](/developer/api-reference/enums/discount-value-type-enum.mdx))

Type of the discount: fixed or percent
