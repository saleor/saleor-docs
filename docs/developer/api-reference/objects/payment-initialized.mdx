title: PaymentInitialized
---

Server-side data generated by a payment gateway. Optional step when the payment provider requires an additional action to initialize payment session.

```graphql
type PaymentInitialized {
  gateway: String!
  name: String!
  data: JSONString
}
```

### Fields

#### `gateway` ([`String`](../../../developer/api-reference/scalars/string))

ID of a payment gateway.

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Payment gateway name.

#### `data` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Initialized data by gateway.
