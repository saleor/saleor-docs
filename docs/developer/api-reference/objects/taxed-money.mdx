title: TaxedMoney
---

Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.

```graphql
type TaxedMoney {
  currency: String!
  gross: Money!
  net: Money!
  tax: Money!
}
```

### Fields

#### `currency` ([`String!`](../../../developer/api-reference/scalars/string))

Currency code.

#### `gross` ([`Money!`](../../../developer/api-reference/objects/money))

Amount of money including taxes.

#### `net` ([`Money!`](../../../developer/api-reference/objects/money))

Amount of money without taxes.

#### `tax` ([`Money!`](../../../developer/api-reference/objects/money))

Amount of taxes.
