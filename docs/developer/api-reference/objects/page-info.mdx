The Relay compliant `PageInfo` type, containing data necessary to paginate this connection.

```graphql
type PageInfo {
  hasNextPage: Boolean!
  hasPreviousPage: Boolean!
  startCursor: String
  endCursor: String
}
```

### Fields

#### `hasNextPage` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

When paginating forwards, are there more items?

#### `hasPreviousPage` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

When paginating backwards, are there more items?

#### `startCursor` ([`String`](/docs/developer/api-reference/scalars/string))

When paginating backwards, the cursor to continue.

#### `endCursor` ([`String`](/docs/developer/api-reference/scalars/string))

When paginating forwards, the cursor to continue.
