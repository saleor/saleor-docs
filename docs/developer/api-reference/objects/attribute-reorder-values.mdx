Reorder the values of an attribute. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
type AttributeReorderValues {
  attribute: Attribute
  attributeErrors: [AttributeError!]!
  errors: [AttributeError!]!
}
```
