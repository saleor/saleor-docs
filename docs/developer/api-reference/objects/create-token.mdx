Create JWT token.

```graphql
type CreateToken {
  token: String
  refreshToken: String
  csrfToken: String
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

### Fields

#### `token` ([`String`](../../../developer/api-reference/scalars/string))

JWT token, required to authenticate.

#### `refreshToken` ([`String`](../../../developer/api-reference/scalars/string))

JWT refresh token, required to re-generate access token.

#### `csrfToken` ([`String`](../../../developer/api-reference/scalars/string))

CSRF token required to re-generate access token.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

A user instance.

#### `accountErrors` ([`[AccountError!]!`](../../../developer/api-reference/objects/account-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
