---

Creates new fulfillments for an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderFulfill {
  fulfillments: [Fulfillment!]
  order: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `fulfillments` ([`[Fulfillment!]`](../../../developer/api-reference/objects/fulfillment))

List of created fulfillments.

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Fulfilled order.

#### `orderErrors` ([`[OrderError!]!`](../../../developer/api-reference/objects/order-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
