Represents an item in the checkout.

```graphql
type CheckoutLine implements Node {
  id: ID!
  variant: ProductVariant!
  quantity: Int!
  totalPrice: TaxedMoney
  requiresShipping: Boolean
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `variant` ([`ProductVariant!`](/docs/developer/api-reference/objects/product-variant))

#### `quantity` ([`Int!`](/docs/developer/api-reference/scalars/int))

#### `totalPrice` ([`TaxedMoney`](/docs/developer/api-reference/objects/taxed-money))

The sum of the checkout line price, taxes and discounts.

#### `requiresShipping` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Indicates whether the item need to be delivered.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
