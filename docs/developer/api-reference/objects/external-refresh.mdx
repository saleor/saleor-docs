---

Refresh user's access by custom plugin.

```graphql
type ExternalRefresh {
  token: String
  refreshToken: String
  csrfToken: String
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

### Fields

#### `token` ([`String`](/docs/developer/api-reference/scalars/string))

The token, required to authenticate.

#### `refreshToken` ([`String`](/docs/developer/api-reference/scalars/string))

The refresh token, required to re-generate external access token.

#### `csrfToken` ([`String`](/docs/developer/api-reference/scalars/string))

CSRF token required to re-generate external access token.

#### `user` ([`User`](/docs/developer/api-reference/objects/user))

A user instance.

#### `accountErrors` ([`[AccountError!]!`](/docs/developer/api-reference/objects/account-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
