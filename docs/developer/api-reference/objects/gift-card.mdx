A gift card is a prepaid electronic payment card accepted in stores. They can be used during checkout by providing a valid gift card codes.

```graphql
type GiftCard implements Node, ObjectWithMetadata {
  code: String!
  isActive: Boolean!
  expiryDate: Date
  tag: String
  created: DateTime!
  lastUsedOn: DateTime
  initialBalance: Money
  currentBalance: Money
  id: ID!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  displayCode: String!
  createdBy: User
  usedBy: User
  createdByEmail: String
  usedByEmail: String
  app: App
  product: Product
  events(filter: GiftCardEventFilterInput): [GiftCardEvent!]!
  boughtInChannel: String
  user: User
  endDate: DateTime
  startDate: DateTime
}
```

### Fields

#### `code` ([`String!`](../../../developer/api-reference/scalars/string))

Gift card code. Can be fetched by staff member with manage gift card permission when gift card wasn't used yet and by the gift card owner.

#### `privateMetadata` ([`[MetadataItem]!`](../../../developer/api-reference/objects/metadata-item))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](../../../developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.

#### `displayCode` ([`String!`](../../../developer/api-reference/scalars/string))

Code in format which allows displaying in a user interface.

#### `createdBy` ([`User`](../../../developer/api-reference/objects/user))

New in Saleor 3.1. The user who bought or issued a gift card.

#### `usedBy` ([`User`](../../../developer/api-reference/objects/user))

New in Saleor 3.1. The customer who used a gift card.

#### `createdByEmail` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. Email address of the user who bought or issued gift card.

#### `usedByEmail` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. Email address of the customer who used a gift card.

#### `app` ([`App`](../../../developer/api-reference/objects/app))

New in Saleor 3.1. App which created the gift card.

#### `product` ([`Product`](../../../developer/api-reference/objects/product))

New in Saleor 3.1. Related gift card product.

#### `events` ([`[GiftCardEvent!]!`](../../../developer/api-reference/objects/gift-card-event))

New in Saleor 3.1. List of events associated with the gift card.

- ##### `filter` ([`GiftCardEventFilterInput`](../../../developer/api-reference/inputs/gift-card-event-filter-input))

Filtering options for gift card events.

#### `boughtInChannel` ([`String`](../../../developer/api-reference/scalars/string))

#### `user` ([`User`](../../../developer/api-reference/objects/user))

The customer who bought a gift card.

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `expiryDate` field instead.</span>

End date of gift card.

#### `startDate` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0.</span>

Start date of gift card.
