History log of the order.

```graphql
type OrderEvent implements Node {
  id: ID!
  date: DateTime
  type: OrderEventsEnum
  user: User
  app: App
  message: String
  email: String
  emailType: OrderEventsEmailsEnum
  amount: Float
  paymentId: String
  paymentGateway: String
  quantity: Int
  composedId: String
  orderNumber: String
  invoiceNumber: String
  oversoldItems: [String!]
  lines: [OrderEventOrderLineObject!]
  fulfilledItems: [FulfillmentLine!]
  warehouse: Warehouse
  transactionReference: String
  shippingCostsIncluded: Boolean
  relatedOrder: Order
  discount: OrderEventDiscountObject
}
```

### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

#### `date` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Date when event happened at in ISO 8601 format.

#### `type` ([`OrderEventsEnum`](../../../developer/api-reference/enums/order-events-enum))

Order event type.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

User who performed the action.

#### `app` ([`App`](../../../developer/api-reference/objects/app))

App that performed the action. Requires of of the following permissions: AppPermission.MANAGE_APPS, OrderPermissions.MANAGE_ORDERS, AuthorizationFilters.OWNER.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

Content of the event.

#### `email` ([`String`](../../../developer/api-reference/scalars/string))

Email of the customer.

#### `emailType` ([`OrderEventsEmailsEnum`](../../../developer/api-reference/enums/order-events-emails-enum))

Type of an email sent to the customer.

#### `amount` ([`Float`](../../../developer/api-reference/scalars/float))

Amount of money.

#### `paymentId` ([`String`](../../../developer/api-reference/scalars/string))

The payment ID from the payment gateway.

#### `paymentGateway` ([`String`](../../../developer/api-reference/scalars/string))

The payment gateway of the payment.

#### `quantity` ([`Int`](../../../developer/api-reference/scalars/int))

Number of items.

#### `composedId` ([`String`](../../../developer/api-reference/scalars/string))

Composed ID of the Fulfillment.

#### `orderNumber` ([`String`](../../../developer/api-reference/scalars/string))

User-friendly number of an order.

#### `invoiceNumber` ([`String`](../../../developer/api-reference/scalars/string))

Number of an invoice related to the order.

#### `oversoldItems` ([`String`](../../../developer/api-reference/scalars/string))

List of oversold lines names.

#### `lines` ([`OrderEventOrderLineObject`](../../../developer/api-reference/objects/order-event-order-line-object))

The concerned lines.

#### `fulfilledItems` ([`FulfillmentLine`](../../../developer/api-reference/objects/fulfillment-line))

The lines fulfilled.

#### `warehouse` ([`Warehouse`](../../../developer/api-reference/objects/warehouse))

The warehouse were items were restocked.

#### `transactionReference` ([`String`](../../../developer/api-reference/scalars/string))

The transaction reference of captured payment.

#### `shippingCostsIncluded` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Define if shipping costs were included to the refund.

#### `relatedOrder` ([`Order`](../../../developer/api-reference/objects/order))

The order which is related to this order.

#### `discount` ([`OrderEventDiscountObject`](../../../developer/api-reference/objects/order-event-discount-object))

The discount applied to the order.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
