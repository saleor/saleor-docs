Updates an address of the logged-in user. Requires one of the following permissions: MANAGE_USERS, IS_OWNER.

```graphql
type AccountAddressUpdate {
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
  address: Address
}
```


### Fields
