New in Saleor 3.1. Deactivate gift cards.

```graphql
type GiftCardBulkDeactivate {
  count: Int!
  errors: [GiftCardError!]!
}
```

### Fields

#### `count` ([`Int!`](/docs/developer/api-reference/scalars/int))

Returns how many objects were affected.
