Checkout object.

```graphql
type Checkout implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  metadata: [MetadataItem!]!
  created: DateTime!
  lastChange: DateTime!
  user: User
  channel: Channel!
  billingAddress: Address
  shippingAddress: Address
  note: String!
  discount: Money
  discountName: String
  translatedDiscountName: String
  voucherCode: String
  availableShippingMethods: [ShippingMethod!]!
  shippingMethods: [ShippingMethod!]!
  availableCollectionPoints: [Warehouse!]!
  availablePaymentGateways: [PaymentGateway!]!
  email: String
  giftCards: [GiftCard!]
  isShippingRequired: Boolean!
  quantity: Int!
  stockReservationExpires: DateTime
  lines: [CheckoutLine!]
  shippingPrice: TaxedMoney
  shippingMethod: ShippingMethod
  deliveryMethod: DeliveryMethod
  subtotalPrice: TaxedMoney
  token: UUID!
  totalPrice: TaxedMoney
  languageCode: LanguageCodeEnum!
}
```

#### `availablePaymentGateways` ([`[PaymentGateway!]!`](../../../developer/api-reference/objects/payment-gateway))

#### `giftCards` ([`[GiftCard!]`](../../../developer/api-reference/objects/gift-card))

List of gift cards associated with this checkout.

#### `isShippingRequired` ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

Returns True, if checkout requires shipping.

#### `quantity` ([`Int!`](../../../developer/api-reference/scalars/int))

The number of items purchased.

#### `stockReservationExpires` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

New in Saleor 3.1. Date when oldest stock reservation for this checkout expires or null if no stock is reserved.

#### `lines` ([`[CheckoutLine!]`](../../../developer/api-reference/objects/checkout-line))

A list of checkout lines, each containing information about an item in the checkout.

#### `shippingPrice` ([`TaxedMoney`](../../../developer/api-reference/objects/taxed-money))

The price of the shipping, with all the taxes included.

#### `shippingMethod` ([`ShippingMethod`](../../../developer/api-reference/objects/shipping-method))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `deliveryMethod` instead.</span>

The shipping method related with checkout.

#### `deliveryMethod` ([`DeliveryMethod`](../../../developer/api-reference/unions/delivery-method))

New in Saleor 3.1. The delivery method selected for this checkout. Note: this feature is in a preview state and can be subject to changes at later point.

#### `subtotalPrice` ([`TaxedMoney`](../../../developer/api-reference/objects/taxed-money))

The price of the checkout before shipping, with taxes included.

#### `token` ([`UUID!`](../../../developer/api-reference/scalars/uuid))

The checkout's token.

#### `totalPrice` ([`TaxedMoney`](../../../developer/api-reference/objects/taxed-money))

The sum of the the checkout line prices, with all the taxes,shipping costs, and discounts included.

#### `languageCode` ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

Checkout language code.
