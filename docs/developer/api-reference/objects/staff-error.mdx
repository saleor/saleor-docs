```graphql
type StaffError {
  field: String
  message: String
  code: AccountErrorCode!
  addressType: AddressTypeEnum
  permissions: [PermissionEnum!]
  groups: [ID!]
  users: [ID!]
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`AccountErrorCode!`](/developer/api-reference/enums/account-error-code.mdx))

The error code.

#### `addressType` ([`AddressTypeEnum`](/developer/api-reference/enums/address-type-enum.mdx))

A type of address that causes the error.

#### `permissions` ([`[PermissionEnum!]`](/developer/api-reference/enums/permission-enum.mdx))

List of permissions which causes the error.

#### `groups` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of permission group IDs which cause the error.

#### `users` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of user IDs which causes the error.
