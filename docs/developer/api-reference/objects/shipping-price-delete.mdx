```graphql
type ShippingPriceDelete {
  shippingMethod: ShippingMethod

  shippingErrors: [ShippingError!]!

}
```

### Fields

#### `shippingMethod` ([`ShippingMethod`](/developer/api-reference/objects/shipping-method.mdx))

A shipping method to delete.

#### `shippingZone` ([`ShippingZone`](/developer/api-reference/objects/shipping-zone.mdx))

A shipping zone to which the shipping method belongs.
