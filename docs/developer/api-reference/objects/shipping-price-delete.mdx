Deletes a shipping price.

```graphql
type ShippingPriceDelete {
  shippingMethod: ShippingMethod
  shippingZone: ShippingZone
  shippingErrors: [ShippingError!]!
  errors: [ShippingError!]!
}
```

### Fields

#### `shippingMethod` ([`ShippingMethod`](/docs/developer/api-reference/objects/shipping-method))

A shipping method to delete.

#### `shippingZone` ([`ShippingZone`](/docs/developer/api-reference/objects/shipping-zone))

A shipping zone to which the shipping method belongs.

#### `shippingErrors` ([`[ShippingError!]!`](/docs/developer/api-reference/objects/shipping-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
