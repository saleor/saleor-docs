Deletes a shipping price. Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
type ShippingPriceDelete {
  shippingMethod: ShippingMethodType
  shippingZone: ShippingZone
  shippingErrors: [ShippingError!]!
  errors: [ShippingError!]!
}
```

### Fields

#### `shippingMethod` ([`ShippingMethodType`](../../../developer/api-reference/objects/shipping-method-type))

A shipping method to delete.

#### `shippingZone` ([`ShippingZone`](../../../developer/api-reference/objects/shipping-zone))

A shipping zone to which the shipping method belongs.

#### `shippingErrors` ([`[ShippingError!]!`](../../../developer/api-reference/objects/shipping-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
