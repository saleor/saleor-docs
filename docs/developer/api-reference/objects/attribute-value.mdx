Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  value: String
  translation(
  languageCode: LanguageCodeEnum!
): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  reference: ID
  file: File
  richText: JSONString
  boolean: Boolean
  date: Date
  dateTime: DateTime
}
```


### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))




#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of a value displayed in the interface.


#### `slug` ([`String`](../../../developer/api-reference/scalars/string))

Internal representation of a value (unique per attribute).


#### `value` ([`String`](../../../developer/api-reference/scalars/string))

Represent value of the attribute value (e.g. color values for swatch attributes).


#### `translation` ([`AttributeValueTranslation`](../../../developer/api-reference/objects/attribute-value-translation))

Returns translated attribute value fields for the given language code.
- ##### `languageCode` ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

A language code to return the translation for attribute value.


#### `inputType` ([`AttributeInputTypeEnum`](../../../developer/api-reference/enums/attribute-input-type-enum))

The input type to use for entering attribute values in the dashboard.


#### `reference` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the attribute reference.


#### `file` ([`File`](../../../developer/api-reference/objects/file))

Represents file URL and content type (if attribute value is a file).


#### `richText` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Represents the text (JSON) of the attribute value.


#### `boolean` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Represents the boolean value of the attribute value.


#### `date` ([`Date`](../../../developer/api-reference/scalars/date))

Represents the date value of the attribute value.


#### `dateTime` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Represents the date time value of the attribute value.


### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node) 

An object with an ID
