Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  value: String
  translation(languageCode: LanguageCodeEnum!): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  reference: ID
  file: File
  richText: JSONString
  boolean: Boolean
  date: Date
  dateTime: DateTime
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Name of a value displayed in the interface.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Internal representation of a value (unique per attribute).

#### `value` ([`String`](/docs/developer/api-reference/scalars/string))

Represents the value of the attribute value.

#### `translation` ([`AttributeValueTranslation`](/docs/developer/api-reference/objects/attribute-value-translation))

Returns translated attribute value fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

A language code to return the translation for attribute value.

#### `inputType` ([`AttributeInputTypeEnum`](/docs/developer/api-reference/enums/attribute-input-type-enum))

The input type to use for entering attribute values in the dashboard.

#### `reference` ([`ID`](/docs/developer/api-reference/scalars/id))

The ID of the attribute reference.

#### `file` ([`File`](/docs/developer/api-reference/objects/file))

Represents file URL and content type (if attribute value is a file).

#### `richText` ([`JSONString`](/docs/developer/api-reference/scalars/json-string))

Represents the text (JSON) of the attribute value.

#### `boolean` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Represents the boolean value of the attribute value.

#### `date` ([`Date`](/docs/developer/api-reference/scalars/date))

Represents the date value of the attribute value.

#### `dateTime` ([`DateTime`](/docs/developer/api-reference/scalars/date-time))

Represents the date time value of the attribute value.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
