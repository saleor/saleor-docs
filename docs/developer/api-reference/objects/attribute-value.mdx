Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  value: String
  translation(languageCode: LanguageCodeEnum!): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  reference: ID
  file: File
  richText: JSONString
  boolean: Boolean
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a value displayed in the interface.

#### `slug` ([`String`](/developer/api-reference/scalars/string.mdx))

Internal representation of a value (unique per attribute).

#### `value` ([`String`](/developer/api-reference/scalars/string.mdx))

Represents the value of the attribute value.

#### `translation` ([`AttributeValueTranslation`](/developer/api-reference/objects/attribute-value-translation.mdx))

Returns translated attribute value fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for attribute value.

#### `inputType` ([`AttributeInputTypeEnum`](/developer/api-reference/enums/attribute-input-type-enum.mdx))

The input type to use for entering attribute values in the dashboard.

#### `reference` ([`ID`](/developer/api-reference/scalars/id.mdx))

The ID of the attribute reference.

#### `file` ([`File`](/developer/api-reference/objects/file.mdx))

Represents file URL and content type (if attribute value is a file).

#### `richText` ([`JSONString`](/developer/api-reference/scalars/json-string.mdx))

Represents the text (JSON) of the attribute value.

#### `boolean` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Represents the boolean value of the attribute value.

### Interfaces

#### [`Node`](/developer/api-reference/interfaces/node.mdx)

An object with an ID
