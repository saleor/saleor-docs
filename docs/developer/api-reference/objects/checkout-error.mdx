```graphql
type CheckoutError {
  field: String
  message: String
  code: CheckoutErrorCode!
  variants: [ID!]
  addressType: AddressTypeEnum
}
```

### Fields

#### `field` ([`String`](/docs/developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

The error message.

#### `code` ([`CheckoutErrorCode!`](/docs/developer/api-reference/enums/checkout-error-code))

The error code.

#### `variants` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of varint IDs which causes the error.

#### `addressType` ([`AddressTypeEnum`](/docs/developer/api-reference/enums/address-type-enum))

A type of address that causes the error.
