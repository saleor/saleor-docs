title: CheckoutPaymentComplete
---

Complete an individual payment as part of the checkout.

```graphql
type CheckoutPaymentComplete {
  checkout: Checkout
  payment: Payment
  confirmationNeeded: Boolean!
  confirmationData: JSONString
  errors: [CheckoutError!]!
}
```

### Fields

#### `checkout` ([`Checkout`](../../../developer/api-reference/objects/checkout))

Related checkout object.

#### `payment` ([`Payment`](../../../developer/api-reference/objects/payment))

A newly created payment.

#### `confirmationNeeded` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Set to true if payment needs to be confirmed before checkout is complete.

#### `confirmationData` ([`JSONString`](../../../developer/api-reference/scalars/json-string))

The confirmation data used to process additional authorization steps.

#### `errors` ([`[CheckoutError!]!`](../../../developer/api-reference/objects/checkout-error))
