Update the shop's address. If the `null` value is passed, the currently selected address will be deleted.

Requires one of the following permissions: MANAGE_SETTINGS.

```graphql
type ShopAddressUpdate {
  shop: Shop
  shopErrors: [ShopError!]!
  errors: [ShopError!]!
}
```
