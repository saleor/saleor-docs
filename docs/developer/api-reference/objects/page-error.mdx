```graphql
type PageError {
  field: String
  message: String
  code: PageErrorCode!
  attributes: [ID!]

```

### Fields

#### [`field`](#) ([`String`](../../../developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### [`message`](#) ([`String`](../../../developer/api-reference/scalars/string))

The error message.

#### [`code`](#) ([`PageErrorCode!`](../../../developer/api-reference/enums/page-error-code))

The error code.

#### [`attributes`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of attributes IDs which causes the error.

#### [`values`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of attribute values IDs which causes the error.
