Represents stock.

```graphql
type Stock implements Node {
  id: ID!
  warehouse: Warehouse!
  productVariant: ProductVariant!
  quantity: Int!
  quantityAllocated: Int!
  quantityReserved: Int!
}
```

#### `warehouse` ([`Warehouse`](../../../developer/api-reference/objects/warehouse))

#### `quantity` ([`Int`](../../../developer/api-reference/scalars/int))

Quantity of a product in the warehouse's possession, including the allocated stock that is waiting for shipment.

Requires one of the following permissions: MANAGE_PRODUCTS, MANAGE_ORDERS.

#### `quantityAllocated` ([`Int`](../../../developer/api-reference/scalars/int))

Quantity allocated for orders.

Requires one of the following permissions: MANAGE_PRODUCTS, MANAGE_ORDERS.

#### `quantityReserved` ([`Int`](../../../developer/api-reference/scalars/int))

Quantity reserved for checkouts.

Requires one of the following permissions: MANAGE_PRODUCTS, MANAGE_ORDERS.
