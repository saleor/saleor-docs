Stores information about a configuration of plugin.

```graphql
type PluginConfiguration {
  active: Boolean!
  channel: Channel
  configuration: [ConfigurationItem]
}
```

### Fields

#### `active` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

Determines if plugin is active or not.

#### `channel` ([`Channel`](/developer/api-reference/objects/channel.mdx))

The channel to which the plugin configuration is assigned to.

#### `configuration` ([`[ConfigurationItem]`](/developer/api-reference/objects/configuration-item.mdx))

Configuration of the plugin.
