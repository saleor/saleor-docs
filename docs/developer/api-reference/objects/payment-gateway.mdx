title: PaymentGateway
---

Available payment gateway backend with configuration necessary to setup client.

```graphql
type PaymentGateway {
  name: String!
  id: ID!
  config: [GatewayConfigLine!]!
  currencies: [String]!
}
```

### Fields

#### `name` ([`String!`](/docs/developer/api-reference/scalars/string))

Payment gateway name.

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

Payment gateway ID.

#### `config` ([`[GatewayConfigLine!]!`](/docs/developer/api-reference/objects/gateway-config-line))

Payment gateway client configuration.

#### `currencies` ([`[String]!`](/docs/developer/api-reference/scalars/string))

Payment gateway supported currencies.
