title: PaymentGateway
---

Available payment gateway backend with configuration necessary to setup client.

```graphql
type PaymentGateway {
  name: String!
  id: ID!
  config: [GatewayConfigLine!]!
  currencies: [String!]!
}
```

### Fields

#### `name` ([`String!`](../../../developer/api-reference/scalars/string))

Payment gateway name.

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

Payment gateway ID.

#### `config` ([`[GatewayConfigLine!]!`](../../../developer/api-reference/objects/gateway-config-line))

Payment gateway client configuration.

#### `currencies` ([`[String!]!`](../../../developer/api-reference/scalars/string))

Payment gateway supported currencies.
