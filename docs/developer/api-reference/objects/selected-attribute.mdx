Represents a custom attribute.

```graphql
type SelectedAttribute {
  attribute: Attribute!
  values: [AttributeValue!]!
}
```

### Fields

#### `attribute` ([`Attribute!`](../../../developer/api-reference/objects/attribute))

Name of an attribute displayed in the interface.

#### `values` ([`[AttributeValue!]!`](../../../developer/api-reference/objects/attribute-value))

Values of an attribute.
