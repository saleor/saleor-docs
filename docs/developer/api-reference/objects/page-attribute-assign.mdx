Assign attributes to a given page type.

Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
type PageAttributeAssign {
  pageType: PageType
  pageErrors: [PageError!]!
  errors: [PageError!]!
}
```
