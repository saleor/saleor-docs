Represents a VAT rate for a country.

```graphql
type VAT {
  countryCode: String!
  standardRate: Float
  reducedRates: [ReducedRate!]!
}
```

### Fields

#### `countryCode` ([`String`](../../../developer/api-reference/scalars/string))

Country code.

#### `standardRate` ([`Float`](../../../developer/api-reference/scalars/float))

Standard VAT rate in percent.

#### `reducedRates` ([`ReducedRate`](../../../developer/api-reference/objects/reduced-rate))

Country's VAT rate exceptions for specific types of goods.
