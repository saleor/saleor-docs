Represents a VAT rate for a country.

```graphql
type VAT {
  countryCode: String!
  standardRate: Float
  reducedRates: [ReducedRate]!
}
```

### Fields

#### `countryCode` ([`String!`](/docs/developer/api-reference/scalars/string))

Country code.

#### `standardRate` ([`Float`](/docs/developer/api-reference/scalars/float))

Standard VAT rate in percent.

#### `reducedRates` ([`[ReducedRate]!`](/docs/developer/api-reference/objects/reduced-rate))

Country's VAT rate exceptions for specific types of goods.
