---

Represents a type of page. It defines what attributes are available to pages of this type.

```graphql
type PageType implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  metadata: [MetadataItem!]!
  name: String!
  slug: String!
  attributes: [Attribute!]
  availableAttributes(
    filter: AttributeFilterInput
    before: String
    after: String
    first: Int
    last: Int
  ): AttributeCountableConnection
  hasPages: Boolean
}
```

### Fields

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

#### `privateMetadata` ([`[MetadataItem!]!`](../../../developer/api-reference/objects/metadata-item))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem!]!`](../../../developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.

#### `name` ([`String!`](../../../developer/api-reference/scalars/string))

#### `slug` ([`String!`](../../../developer/api-reference/scalars/string))

#### `attributes` ([`[Attribute!]`](../../../developer/api-reference/objects/attribute))

Page attributes of that page type.

#### `availableAttributes` ([`AttributeCountableConnection`](../../../developer/api-reference/objects/attribute-countable-connection))

Attributes that can be assigned to the page type.

- ##### `filter` ([`AttributeFilterInput`](../../../developer/api-reference/inputs/attribute-filter-input))

- ##### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

- ##### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

- ##### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

- ##### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

#### `hasPages` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether page type has pages assigned.
