Obtain external access tokens for user by custom plugin.

```graphql
type ExternalObtainAccessTokens {
  token: String
  refreshToken: String
  csrfToken: String
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

### Fields

#### `token` ([`String`](/developer/api-reference/scalars/string.mdx))

The token, required to authenticate.

#### `refreshToken` ([`String`](/developer/api-reference/scalars/string.mdx))

The refresh token, required to re-generate external access token.

#### `csrfToken` ([`String`](/developer/api-reference/scalars/string.mdx))

CSRF token required to re-generate external access token.

#### `user` ([`User`](/developer/api-reference/objects/user.mdx))

A user instance.

#### `accountErrors` ([`[AccountError!]!`](/developer/api-reference/objects/account-error.mdx))
