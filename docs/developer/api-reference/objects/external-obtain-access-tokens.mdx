Obtain external access tokens for user by custom plugin.

```graphql
type ExternalObtainAccessTokens {
  token: String
  refreshToken: String
  csrfToken: String
  user: User
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```

### Fields

#### `token` ([`String`](../../../developer/api-reference/scalars/string))

The token, required to authenticate.

#### `refreshToken` ([`String`](../../../developer/api-reference/scalars/string))

The refresh token, required to re-generate external access token.

#### `csrfToken` ([`String`](../../../developer/api-reference/scalars/string))

CSRF token required to re-generate external access token.

#### `user` ([`User`](../../../developer/api-reference/objects/user))

A user instance.

#### `accountErrors` ([`AccountError`](../../../developer/api-reference/objects/account-error))

#### `errors` ([`AccountError`](../../../developer/api-reference/objects/account-error))
