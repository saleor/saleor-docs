Reorder the attributes of a page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
type PageTypeReorderAttributes {
  pageType: PageType
  pageErrors: [PageError!]!
  errors: [PageError!]!
}
```
