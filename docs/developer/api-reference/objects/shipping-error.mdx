```graphql
type ShippingError {
  field: String
  message: String
  code: ShippingErrorCode!
  warehouses: [ID!]
  channels: [ID!]
}
```


### Fields

#### `field` ([`String`](../../../developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.


#### `message` ([`String`](../../../developer/api-reference/scalars/string))

The error message.


#### `code` ([`ShippingErrorCode`](../../../developer/api-reference/enums/shipping-error-code))

The error code.


#### `warehouses` ([`ID`](../../../developer/api-reference/scalars/id))

List of warehouse IDs which causes the error.


#### `channels` ([`ID`](../../../developer/api-reference/scalars/id))

List of channels IDs which causes the error.
