Represents shop's domain.

```graphql
type Domain {
  host: String!
  sslEnabled: Boolean!
  url: String!
}
```

### Fields

#### `host` ([`String!`](/docs/developer/api-reference/scalars/string))

The host name of the domain.

#### `sslEnabled` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Inform if SSL is enabled.

#### `url` ([`String!`](/docs/developer/api-reference/scalars/string))

Shop's absolute URL.
