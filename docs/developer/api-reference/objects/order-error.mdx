```graphql
type OrderError {
  field: String
  message: String
  code: OrderErrorCode!
  warehouse: ID
  orderLines: [ID!]
  variants: [ID!]
  addressType: AddressTypeEnum
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`OrderErrorCode!`](/developer/api-reference/enums/order-error-code.mdx))

The error code.

#### `warehouse` ([`ID`](/developer/api-reference/scalars/id.mdx))

Warehouse ID which causes the error.

#### `orderLines` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of order line IDs that cause the error.

#### `variants` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of product variants that are associated with the error

#### `addressType` ([`AddressTypeEnum`](/developer/api-reference/enums/address-type-enum.mdx))

A type of address that causes the error.
