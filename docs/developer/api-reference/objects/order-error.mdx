No description

```graphql
type OrderError {
  field: String
  message: String
  code: OrderErrorCode!
  warehouse: ID
  orderLines: [ID!]
  variants: [ID!]
  addressType: AddressTypeEnum
}
```

### Fields

#### `field` ([`String`](../../../developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](../../../developer/api-reference/scalars/string))

The error message.

#### `code` ([`OrderErrorCode`](../../../developer/api-reference/enums/order-error-code))

The error code.

#### `warehouse` ([`ID`](../../../developer/api-reference/scalars/id))

Warehouse ID which causes the error.

#### `orderLines` ([`ID`](../../../developer/api-reference/scalars/id))

List of order line IDs that cause the error.

#### `variants` ([`ID`](../../../developer/api-reference/scalars/id))

List of product variants that are associated with the error

#### `addressType` ([`AddressTypeEnum`](../../../developer/api-reference/enums/address-type-enum))

A type of address that causes the error.
