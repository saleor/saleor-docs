Represents channel.

```graphql
type Channel implements Node {
  id: ID!
  name: String!
  isActive: Boolean!
  slug: String!
  currencyCode: String!
  hasOrders: Boolean!
  defaultCountry: CountryDisplay!
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `isActive` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

#### `slug` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `currencyCode` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `hasOrders` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

Whether a channel has associated orders.

#### `defaultCountry` ([`CountryDisplay!`](/developer/api-reference/objects/country-display.mdx))

New in Saleor 3.1. Default country for the channel. Default country can be used in checkout to determine the stock quantities or calculate taxes when the country was not explicitly provided.

### Interfaces

#### [`Node`](/developer/api-reference/interfaces/node.mdx)

An object with an ID
