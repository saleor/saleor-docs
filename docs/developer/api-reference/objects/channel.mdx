Represents channel.

```graphql
type Channel implements Node {
  id: ID!
  name: String!
  isActive: Boolean!
  currencyCode: String!
  slug: String!
  hasOrders: Boolean!
  defaultCountry: CountryDisplay!
}
```

#### `hasOrders` ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

Whether a channel has associated orders.

#### `defaultCountry` ([`CountryDisplay!`](../../../developer/api-reference/objects/country-display))

New in Saleor 3.1. Default country for the channel. Default country can be used in checkout to determine the stock quantities or calculate taxes when the country was not explicitly provided.
