title: FulfillmentUpdateTracking
---

Updates a fulfillment for an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type FulfillmentUpdateTracking {
  fulfillment: Fulfillment
  order: Order
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `fulfillment` ([`Fulfillment`](../../../developer/api-reference/objects/fulfillment))

A fulfillment with updated tracking.

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Order for which fulfillment was updated.

#### `orderErrors` ([`[OrderError!]!`](../../../developer/api-reference/objects/order-error))

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
