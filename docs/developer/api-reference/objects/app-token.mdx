Represents token data.

```graphql
type AppToken implements Node {
  name: String
  authToken: String
  id: ID!
}
```

### Fields

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Name of the authenticated token.

#### `authToken` ([`String`](/docs/developer/api-reference/scalars/string))

Last 4 characters of the token.

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

The ID of the object.

### Interfaces

#### [`Node`](/docs/developer/api-reference/interfaces/node)

An object with an ID
