Adds products, categories, collections to a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
type SaleAddCatalogues {
  sale: Sale
  discountErrors: [DiscountError!]!
  errors: [DiscountError!]!
}
```

### Fields

#### `sale` ([`Sale`](../../../developer/api-reference/objects/sale))

Sale of which catalogue IDs will be modified.

<span class="badge badge--warning">DEPRECATED: This field will be removed in Saleor 4.0. Use `errors` field instead.</span>
