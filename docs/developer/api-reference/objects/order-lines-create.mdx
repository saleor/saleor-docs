Create order lines for an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
type OrderLinesCreate {
  order: Order
  orderLines: [OrderLine!]
  orderErrors: [OrderError!]!
  errors: [OrderError!]!
}
```

### Fields

#### `order` ([`Order`](../../../developer/api-reference/objects/order))

Related order.

#### `orderLines` ([`[OrderLine!]`](../../../developer/api-reference/objects/order-line))

List of added order lines.

#### `orderErrors` ([`[OrderError!]!`](../../../developer/api-reference/objects/order-error))
