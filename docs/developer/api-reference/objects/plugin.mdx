Plugin.

```graphql
type Plugin {
  id: ID!
  name: String!
  description: String!
  globalConfiguration: PluginConfiguration
  channelConfigurations: [PluginConfiguration!]!
}
```


### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

Identifier of the plugin.


#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of the plugin.


#### `description` ([`String`](../../../developer/api-reference/scalars/string))

Description of the plugin.


#### `globalConfiguration` ([`PluginConfiguration`](../../../developer/api-reference/objects/plugin-configuration))

Global configuration of the plugin (not channel-specific).


#### `channelConfigurations` ([`PluginConfiguration`](../../../developer/api-reference/objects/plugin-configuration))

Channel-specific plugin configuration.
