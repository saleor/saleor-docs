Plugin.

```graphql
type Plugin {
  id: ID!
  name: String!
  description: String!
  globalConfiguration: PluginConfiguration
  channelConfigurations: [PluginConfiguration!]!
}
```

### Fields

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

Identifier of the plugin.

#### `name` ([`String!`](/docs/developer/api-reference/scalars/string))

Name of the plugin.

#### `description` ([`String!`](/docs/developer/api-reference/scalars/string))

Description of the plugin.

#### `globalConfiguration` ([`PluginConfiguration`](/docs/developer/api-reference/objects/plugin-configuration))

Global configuration of the plugin (not channel-specific).

#### `channelConfigurations` ([`[PluginConfiguration!]!`](/docs/developer/api-reference/objects/plugin-configuration))

Channel-specific plugin configuration.
