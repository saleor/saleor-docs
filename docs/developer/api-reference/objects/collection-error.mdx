No description

```graphql
type CollectionError {
  field: String
  message: String
  products: [ID!]
  code: CollectionErrorCode!
}
```

### Fields

#### `field` ([`String`](/docs/developer/api-reference/scalars/string))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/docs/developer/api-reference/scalars/string))

The error message.

#### `products` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of products IDs which causes the error.

#### `code` ([`CollectionErrorCode!`](/docs/developer/api-reference/enums/collection-error-code))

The error code.
