Represents an image.

```graphql
type Image {
  url: String!
  alt: String
}
```

### Fields

#### `url` ([`String!`](../../../developer/api-reference/scalars/string))

The URL of the image.

#### `alt` ([`String`](../../../developer/api-reference/scalars/string))

Alt text for an image.
