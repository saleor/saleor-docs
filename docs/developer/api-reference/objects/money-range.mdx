Represents a range of amounts of money.

```graphql
type MoneyRange {
  start: Money
  stop: Money
}
```


### Fields

#### `start` ([`Money`](../../../developer/api-reference/objects/money))

Lower bound of a price range.


#### `stop` ([`Money`](../../../developer/api-reference/objects/money))

Upper bound of a price range.
