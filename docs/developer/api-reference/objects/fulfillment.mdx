Represents order fulfillment.

```graphql
type Fulfillment implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  fulfillmentOrder: Int!
  status: FulfillmentStatus!
  trackingNumber: String!
  created: DateTime!
  lines: [FulfillmentLine]
  statusDisplay: String
  warehouse: Warehouse
}
```

### Fields

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

#### `privateMetadata` ([`[MetadataItem]!`](../../../developer/api-reference/objects/metadata-item))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](../../../developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.

#### `fulfillmentOrder` ([`Int!`](../../../developer/api-reference/scalars/int))

#### `status` ([`FulfillmentStatus!`](../../../developer/api-reference/enums/fulfillment-status))

#### `trackingNumber` ([`String!`](../../../developer/api-reference/scalars/string))

#### `created` ([`DateTime!`](../../../developer/api-reference/scalars/date-time))

#### `lines` ([`[FulfillmentLine]`](../../../developer/api-reference/objects/fulfillment-line))

List of lines for the fulfillment.

#### `statusDisplay` ([`String`](../../../developer/api-reference/scalars/string))

User-friendly fulfillment status.

#### `warehouse` ([`Warehouse`](../../../developer/api-reference/objects/warehouse))

Warehouse from fulfillment was fulfilled.
