Represents a product image.

```graphql
type ProductImage {
  id: ID!
  alt: String
  sortOrder: Int
  url(
  size: Int
): String!
}
```


### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the image.


#### `alt` ([`String`](../../../developer/api-reference/scalars/string))

The alt text of the image.


#### `sortOrder` ([`Int`](../../../developer/api-reference/scalars/int))

The new relative sorting position of the item (from -inf to +inf). 1 moves the item one position forward, -1 moves the item one position backward, 0 leaves the item unchanged.


#### `url` ([`String`](../../../developer/api-reference/scalars/string))

The URL of the image.
- ##### `size` ([`Int`](../../../developer/api-reference/scalars/int))

Size of the image.
