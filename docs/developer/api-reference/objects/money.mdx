Represents amount of money in specific currency.

```graphql
type Money {
  currency: String!
  amount: Float!
}
```

### Fields

#### `currency` ([`String!`](/docs/developer/api-reference/scalars/string))

Currency code.

#### `amount` ([`Float!`](/docs/developer/api-reference/scalars/float))

Amount of money.
