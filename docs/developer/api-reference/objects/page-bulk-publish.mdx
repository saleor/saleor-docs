Publish pages. Requires one of the following permissions: MANAGE_PAGES.

```graphql
type PageBulkPublish {
  count: Int!
  pageErrors: [PageError!]!
  errors: [PageError!]!
}
```

### Fields

#### `count` ([`Int!`](../../../developer/api-reference/scalars/int))

Returns how many objects were affected.
