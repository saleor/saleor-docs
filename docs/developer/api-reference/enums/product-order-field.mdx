### Values

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>NAME</b></code>](#) 
> Sort products by name.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>RANK</b></code>](#) 
> Sort products by rank. Note: This option is available only with the `search` filter.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>PRICE</b></code>](#) 
> Sort products by price.

This option requires a channel filter to work as the values can vary between channels.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>MINIMAL_PRICE</b></code>](#) 
> Sort products by a minimal price of a product's variant.

This option requires a channel filter to work as the values can vary between channels.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>LAST_MODIFIED</b></code>](#) 
> Sort products by update date.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>DATE</b></code>](#) 
> Sort products by update date.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>TYPE</b></code>](#) 
> Sort products by type.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>PUBLISHED</b></code>](#) 
> Sort products by publication status.

This option requires a channel filter to work as the values can vary between channels.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>PUBLICATION_DATE</b></code>](#) 
> Sort products by publication date.

This option requires a channel filter to work as the values can vary between channels.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>PUBLISHED_AT</b></code>](#) 
> Sort products by publication date.

This option requires a channel filter to work as the values can vary between channels.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>LAST_MODIFIED_AT</b></code>](#) 
> Sort products by update date.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductOrderField.<b>COLLECTION</b></code>](#) 
> Sort products by collection. Note: This option is available only for the `Collection.products` query.
