Updates a collection. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionUpdate(
  id: ID!
  input: CollectionInput!
): CollectionUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a collection to update.

#### `input` ([`CollectionInput`](../../../developer/api-reference/inputs/collection-input))

Fields required to update a collection.

### Type

#### [`CollectionUpdate`](../../../developer/api-reference/objects/collection-update)

Updates a collection. Requires one of the following permissions: MANAGE_PRODUCTS.
