Updates a collection.

```graphql
collectionUpdate(
  id: ID!
  input: CollectionInput!
): CollectionUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a collection to update.

#### `input` ([`CollectionInput!`](/docs/developer/api-reference/inputs/collection-input))

Fields required to update a collection.

### Type

#### [`CollectionUpdate`](/docs/developer/api-reference/objects/collection-update)

Updates a collection.
