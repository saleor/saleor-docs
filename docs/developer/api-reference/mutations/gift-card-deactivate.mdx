Deactivate a gift card.

```graphql
giftCardDeactivate(
  id: ID!
): GiftCardDeactivate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a gift card to deactivate.

### Type

#### [`GiftCardDeactivate`](/docs/developer/api-reference/objects/gift-card-deactivate)

Deactivate a gift card.
