Deletes an app. Requires one of the following permissions: MANAGE_APPS.

```graphql
appDelete(
  id: ID!
): AppDelete
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an app to delete.


### Type

#### [`AppDelete`](../../../developer/api-reference/objects/app-delete) 

Deletes an app. Requires one of the following permissions: MANAGE_APPS.
