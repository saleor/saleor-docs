Deletes vouchers. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
voucherBulkDelete(
  ids: [ID!]!
): VoucherBulkDelete
```


### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of voucher IDs to delete.


### Type

#### [`VoucherBulkDelete`](../../../developer/api-reference/objects/voucher-bulk-delete) 

Deletes vouchers. Requires one of the following permissions: MANAGE_DISCOUNTS.
