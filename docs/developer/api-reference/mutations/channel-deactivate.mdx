Deactivate a channel. Requires one of the following permissions: MANAGE_CHANNELS.

```graphql
channelDeactivate(
  id: ID!
): ChannelDeactivate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the channel to deactivate.


### Type

#### [`ChannelDeactivate`](../../../developer/api-reference/objects/channel-deactivate) 

Deactivate a channel. Requires one of the following permissions: MANAGE_CHANNELS.
