Confirm the email change of the logged-in user.

```graphql
confirmEmailChange(
  channel: String
  token: String!
): ConfirmEmailChange

```

### Arguments

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel which will be used to notify users. Optional when only one channel exists.

#### `token` ([`String!`](/developer/api-reference/scalars/string.mdx))

A one-time token required to change the email.

### Type

#### [`ConfirmEmailChange`](/developer/api-reference/objects/confirm-email-change.mdx)

Confirm the email change of the logged-in user.
