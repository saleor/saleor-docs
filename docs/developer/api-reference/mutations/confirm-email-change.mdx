Confirm the email change of the logged-in user.

```graphql
confirmEmailChange(
  channel: String
  token: String!
): ConfirmEmailChange

```

### Arguments

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel which will be used to notify users. Optional when only one channel exists.

#### `token` ([`String!`](/docs/developer/api-reference/scalars/string))

A one-time token required to change the email.

### Type

#### [`ConfirmEmailChange`](/docs/developer/api-reference/objects/confirm-email-change)

Confirm the email change of the logged-in user.
