Updates an existing staff user.

```graphql
staffUpdate(
  id: ID!
  input: StaffUpdateInput!
): StaffUpdate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a staff user to update.

#### `input` ([`StaffUpdateInput!`](../../../developer/api-reference/inputs/staff-update-input))

Fields required to update a staff user.

### Type

#### [`StaffUpdate`](../../../developer/api-reference/objects/staff-update)

Updates an existing staff user.
