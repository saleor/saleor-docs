Deletes an address.

Requires one of the following permissions: MANAGE_USERS.

```graphql
addressDelete(
  id: ID!
): AddressDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the address to delete.

### Type

#### [`AddressDelete`](../../../developer/api-reference/objects/address-delete)

Deletes an address.

Requires one of the following permissions: MANAGE_USERS.
