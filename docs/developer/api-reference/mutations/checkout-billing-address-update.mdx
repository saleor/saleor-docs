Update billing address in the existing checkout.

```graphql
checkoutBillingAddressUpdate(
  billingAddress: AddressInput!
  checkoutId: ID = 0
  token: UUID
): CheckoutBillingAddressUpdate

```

### Arguments

#### `billingAddress` ([`AddressInput!`](/developer/api-reference/inputs/address-input.mdx))

The billing address of the checkout.

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the checkout.DEPRECATED: Will be removed in Saleor 4.0. Use token instead.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutBillingAddressUpdate`](/developer/api-reference/objects/checkout-billing-address-update.mdx)

Update billing address in the existing checkout.
