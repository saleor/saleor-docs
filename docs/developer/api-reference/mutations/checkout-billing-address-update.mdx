Update billing address in the existing checkout.

```graphql
checkoutBillingAddressUpdate(
  billingAddress: AddressInput!
  checkoutId: ID
  token: UUID
): CheckoutBillingAddressUpdate

```

### Arguments

#### `billingAddress` ([`AddressInput!`](/docs/developer/api-reference/inputs/address-input))

The billing address of the checkout.

#### `checkoutId` ([`ID`](/docs/developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `token` ([`UUID`](/docs/developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutBillingAddressUpdate`](/docs/developer/api-reference/objects/checkout-billing-address-update)

Update billing address in the existing checkout.
