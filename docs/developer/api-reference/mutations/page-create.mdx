Creates a new page. Requires one of the following permissions: MANAGE_PAGES.

```graphql
pageCreate(
  input: PageCreateInput!
): PageCreate
```

### Arguments

#### `input` ([`PageCreateInput`](../../../developer/api-reference/inputs/page-create-input))

Fields required to create a page.

### Type

#### [`PageCreate`](../../../developer/api-reference/objects/page-create)

Creates a new page. Requires one of the following permissions: MANAGE_PAGES.
