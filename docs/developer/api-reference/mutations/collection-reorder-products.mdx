Reorder the products of a collection.

```graphql
collectionReorderProducts(
  collectionId: ID!
  moves: [MoveProductInput]!
): CollectionReorderProducts

```

### Arguments

#### `collectionId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a collection.

#### `moves` ([`[MoveProductInput]!`](/docs/developer/api-reference/inputs/move-product-input))

The collection products position operations.

### Type

#### [`CollectionReorderProducts`](/docs/developer/api-reference/objects/collection-reorder-products)

Reorder the products of a collection.
