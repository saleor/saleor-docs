Reorder the products of a collection. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionReorderProducts(
  collectionId: ID!
  moves: [MoveProductInput!]!
): CollectionReorderProducts
```


### Arguments

#### `collectionId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a collection.


#### `moves` ([`MoveProductInput`](../../../developer/api-reference/inputs/move-product-input))

The collection products position operations.


### Type

#### [`CollectionReorderProducts`](../../../developer/api-reference/objects/collection-reorder-products) 

Reorder the products of a collection. Requires one of the following permissions: MANAGE_PRODUCTS.
