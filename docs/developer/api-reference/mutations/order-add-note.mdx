Adds note to the order.

```graphql
orderAddNote(
  order: ID!
  input: OrderAddNoteInput!
): OrderAddNote

```

### Arguments

#### `order` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order to add a note for.

#### `input` ([`OrderAddNoteInput!`](/docs/developer/api-reference/inputs/order-add-note-input))

Fields required to create a note for the order.

### Type

#### [`OrderAddNote`](/docs/developer/api-reference/objects/order-add-note)

Adds note to the order.
