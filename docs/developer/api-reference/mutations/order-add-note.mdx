Adds note to the order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderAddNote(
  order: ID!
  input: OrderAddNoteInput!
): OrderAddNote
```


### Arguments

#### `order` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to add a note for.


#### `input` ([`OrderAddNoteInput`](../../../developer/api-reference/inputs/order-add-note-input))

Fields required to create a note for the order.


### Type

#### [`OrderAddNote`](../../../developer/api-reference/objects/order-add-note) 

Adds note to the order. Requires one of the following permissions: MANAGE_ORDERS.
