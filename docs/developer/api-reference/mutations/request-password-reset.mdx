Sends an email with the account password modification link.

```graphql
requestPasswordReset(
  channel: String
  email: String!
  redirectUrl: String!
): RequestPasswordReset

```

### Arguments

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel which will be used for notify user. Optional when only one channel exists.

#### `email` ([`String!`](/developer/api-reference/scalars/string.mdx))

Email of the user that will be used for password recovery.

#### `redirectUrl` ([`String!`](/developer/api-reference/scalars/string.mdx))

URL of a view where users should be redirected to reset the password. URL in RFC 1808 format.

### Type

#### [`RequestPasswordReset`](/developer/api-reference/objects/request-password-reset.mdx)

Sends an email with the account password modification link.
