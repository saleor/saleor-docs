Sends an email with the account password modification link.

```graphql
requestPasswordReset(
  channel: String
  email: String!
  redirectUrl: String!
): RequestPasswordReset

```

### Arguments

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel which will be used for notify user. Optional when only one channel exists.

#### `email` ([`String!`](/docs/developer/api-reference/scalars/string))

Email of the user that will be used for password recovery.

#### `redirectUrl` ([`String!`](/docs/developer/api-reference/scalars/string))

URL of a view where users should be redirected to reset the password. URL in RFC 1808 format.

### Type

#### [`RequestPasswordReset`](/docs/developer/api-reference/objects/request-password-reset)

Sends an email with the account password modification link.
