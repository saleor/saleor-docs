Changes ordering of the product media.

```graphql
productMediaReorder(
  mediaIds: [ID]!
  productId: ID!
): ProductMediaReorder

```

### Arguments

#### `mediaIds` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

IDs of a product media in the desired order.

#### `productId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of product that media order will be altered.

### Type

#### [`ProductMediaReorder`](/docs/developer/api-reference/objects/product-media-reorder)

Changes ordering of the product media.
