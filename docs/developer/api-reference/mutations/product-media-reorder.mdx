title: productMediaReorder
---

Changes ordering of the product media.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productMediaReorder(
  mediaIds: [ID!]!
  productId: ID!
): ProductMediaReorder
```

### Arguments

#### `mediaIds` ([`ID`](../../../developer/api-reference/scalars/id))

IDs of a product media in the desired order.

#### `productId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of product that media order will be altered.

### Type

#### [`ProductMediaReorder`](../../../developer/api-reference/objects/product-media-reorder)

Changes ordering of the product media.

Requires one of the following permissions: MANAGE_PRODUCTS.
