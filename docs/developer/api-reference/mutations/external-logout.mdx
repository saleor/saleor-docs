Logout user by custom plugin.

```graphql
externalLogout(
  input: JSONString!
  pluginId: String!
): ExternalLogout

```

### Arguments

#### `input` ([`JSONString!`](/docs/developer/api-reference/scalars/json-string))

The data required by plugin to proceed the logout process.

#### `pluginId` ([`String!`](/docs/developer/api-reference/scalars/string))

The ID of the authentication plugin.

### Type

#### [`ExternalLogout`](/docs/developer/api-reference/objects/external-logout)

Logout user by custom plugin.
