Logout user by custom plugin.

```graphql
externalLogout(
  input: JSONString!
  pluginId: String!
): ExternalLogout
```


### Arguments

#### `input` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

The data required by plugin to proceed the logout process.


#### `pluginId` ([`String`](../../../developer/api-reference/scalars/string))

The ID of the authentication plugin.


### Type

#### [`ExternalLogout`](../../../developer/api-reference/objects/external-logout) 

Logout user by custom plugin.
