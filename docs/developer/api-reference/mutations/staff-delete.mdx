Deletes a staff user. Requires one of the following permissions: MANAGE_STAFF.

```graphql
staffDelete(
  id: ID!
): StaffDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a staff user to delete.

### Type

#### [`StaffDelete`](../../../developer/api-reference/objects/staff-delete)

Deletes a staff user. Requires one of the following permissions: MANAGE_STAFF.
