Deletes a menu item. Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuItemDelete(
  id: ID!
): MenuItemDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a menu item to delete.

### Type

#### [`MenuItemDelete`](../../../developer/api-reference/objects/menu-item-delete)

Deletes a menu item. Requires one of the following permissions: MANAGE_MENUS.
