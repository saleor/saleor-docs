Manage product's availability in channels. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productChannelListingUpdate(
  id: ID!
  input: ProductChannelListingUpdateInput!
): ProductChannelListingUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product to update.


#### `input` ([`ProductChannelListingUpdateInput`](../../../developer/api-reference/inputs/product-channel-listing-update-input))

Fields required to create or update product channel listings.


### Type

#### [`ProductChannelListingUpdate`](../../../developer/api-reference/objects/product-channel-listing-update) 

Manage product's availability in channels. Requires one of the following permissions: MANAGE_PRODUCTS.
