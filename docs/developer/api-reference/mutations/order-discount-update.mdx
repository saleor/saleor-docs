Update discount for the order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderDiscountUpdate(
  discountId: ID!
  input: OrderDiscountCommonInput!
): OrderDiscountUpdate
```

### Arguments

#### `discountId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a discount to update.

#### `input` ([`OrderDiscountCommonInput`](../../../developer/api-reference/inputs/order-discount-common-input))

Fields required to update a discount for the order.

### Type

#### [`OrderDiscountUpdate`](../../../developer/api-reference/objects/order-discount-update)

Update discount for the order. Requires one of the following permissions: MANAGE_ORDERS.
