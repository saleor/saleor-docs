Update discount for the order.

```graphql
orderDiscountUpdate(
  discountId: ID!
  input: OrderDiscountCommonInput!
): OrderDiscountUpdate

```

### Arguments

#### `discountId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a discount to update.

#### `input` ([`OrderDiscountCommonInput!`](/docs/developer/api-reference/inputs/order-discount-common-input))

Fields required to update a discount for the order.

### Type

#### [`OrderDiscountUpdate`](/docs/developer/api-reference/objects/order-discount-update)

Update discount for the order.
