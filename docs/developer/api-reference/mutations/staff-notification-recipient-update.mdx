Updates a staff notification recipient. Requires one of the following permissions: MANAGE_SETTINGS.

```graphql
staffNotificationRecipientUpdate(
  id: ID!
  input: StaffNotificationRecipientInput!
): StaffNotificationRecipientUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a staff notification recipient to update.

#### `input` ([`StaffNotificationRecipientInput`](../../../developer/api-reference/inputs/staff-notification-recipient-input))

Fields required to update a staff notification recipient.

### Type

#### [`StaffNotificationRecipientUpdate`](../../../developer/api-reference/objects/staff-notification-recipient-update)

Updates a staff notification recipient. Requires one of the following permissions: MANAGE_SETTINGS.
