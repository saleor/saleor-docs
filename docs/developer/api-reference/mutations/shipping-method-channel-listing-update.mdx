Manage shipping method's availability in channels.

```graphql
shippingMethodChannelListingUpdate(
  id: ID!
  input: ShippingMethodChannelListingInput!
): ShippingMethodChannelListingUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a shipping method to update.

#### `input` ([`ShippingMethodChannelListingInput!`](/docs/developer/api-reference/inputs/shipping-method-channel-listing-input))

Fields required to update shipping method channel listings.

### Type

#### [`ShippingMethodChannelListingUpdate`](/docs/developer/api-reference/objects/shipping-method-channel-listing-update)

Manage shipping method's availability in channels.
