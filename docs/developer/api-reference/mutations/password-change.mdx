Change the password of the logged in user.

Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
passwordChange(
  newPassword: String!
  oldPassword: String!
): PasswordChange
```

### Arguments

#### [`newPassword`](#) ([`String!`](../../../developer/api-reference/scalars/string))

New user password.

#### [`oldPassword`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Current user password.

### Type

#### [`PasswordChange`](../../../developer/api-reference/objects/password-change)
