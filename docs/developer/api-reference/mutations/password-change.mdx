Change the password of the logged in user.

```graphql
passwordChange(
  newPassword: String!
  oldPassword: String!
): PasswordChange

```

### Arguments

#### `newPassword` ([`String!`](/docs/developer/api-reference/scalars/string))

New user password.

#### `oldPassword` ([`String!`](/docs/developer/api-reference/scalars/string))

Current user password.

### Type

#### [`PasswordChange`](/docs/developer/api-reference/objects/password-change)

Change the password of the logged in user.
