Create order lines for an order.

```graphql
orderLinesCreate(
  id: ID!
  input: [OrderLineCreateInput]!
): OrderLinesCreate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order to add the lines to.

#### `input` ([`[OrderLineCreateInput]!`](/developer/api-reference/inputs/order-line-create-input.mdx))

Fields required to add order lines.

### Type

#### [`OrderLinesCreate`](/developer/api-reference/objects/order-lines-create.mdx)

Create order lines for an order.
