title: productVariantReorder
---

Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantReorder(
  moves: [ReorderInput!]!
  productId: ID!
): ProductVariantReorder

```

### Arguments

#### `moves` ([`[ReorderInput!]!`](../../../developer/api-reference/inputs/reorder-input))

The list of variant reordering operations.

#### `productId` ([`ID!`](../../../developer/api-reference/scalars/id))

Id of product that variants order will be altered.

### Type

#### [`ProductVariantReorder`](../../../developer/api-reference/objects/product-variant-reorder)

Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook. Requires one of the following permissions: MANAGE_PRODUCTS.
