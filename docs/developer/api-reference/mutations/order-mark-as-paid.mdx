Mark order as manually paid.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderMarkAsPaid(
  id: ID!
  transactionReference: String
): OrderMarkAsPaid
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to mark paid.

#### `transactionReference` ([`String`](../../../developer/api-reference/scalars/string))

The external transaction reference.

### Type

#### [`OrderMarkAsPaid`](../../../developer/api-reference/objects/order-mark-as-paid)

Mark order as manually paid.

Requires one of the following permissions: MANAGE_ORDERS.
