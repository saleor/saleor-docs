Mark order as manually paid.

```graphql
orderMarkAsPaid(
  id: ID!
  transactionReference: String
): OrderMarkAsPaid

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order to mark paid.

#### `transactionReference` ([`String`](/docs/developer/api-reference/scalars/string))

The external transaction reference.

### Type

#### [`OrderMarkAsPaid`](/docs/developer/api-reference/objects/order-mark-as-paid)

Mark order as manually paid.
