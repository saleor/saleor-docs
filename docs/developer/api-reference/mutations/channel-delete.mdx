Delete a channel. Orders associated with the deleted channel will be moved to the target channel. Checkouts, product availability, and pricing will be removed.

```graphql
channelDelete(
  id: ID!
  input: ChannelDeleteInput
): ChannelDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a channel to delete.

#### `input` ([`ChannelDeleteInput`](/developer/api-reference/inputs/channel-delete-input.mdx))

Fields required to delete a channel.

### Type

#### [`ChannelDelete`](/developer/api-reference/objects/channel-delete.mdx)

Delete a channel. Orders associated with the deleted channel will be moved to the target channel. Checkouts, product availability, and pricing will be removed.
