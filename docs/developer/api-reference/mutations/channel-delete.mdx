Delete a channel. Orders associated with the deleted channel will be moved to the target channel. Checkouts, product availability, and pricing will be removed.

Requires one of the following permissions: MANAGE_CHANNELS.

```graphql
channelDelete(
  id: ID!
  input: ChannelDeleteInput
): ChannelDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a channel to delete.

#### `input` ([`ChannelDeleteInput`](../../../developer/api-reference/inputs/channel-delete-input))

Fields required to delete a channel.

### Type

#### [`ChannelDelete`](../../../developer/api-reference/objects/channel-delete)

Delete a channel. Orders associated with the deleted channel will be moved to the target channel. Checkouts, product availability, and pricing will be removed.
