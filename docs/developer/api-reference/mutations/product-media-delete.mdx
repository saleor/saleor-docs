Deletes a product media.

```graphql
productMediaDelete(
  id: ID!
): ProductMediaDelete

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product media to delete.

### Type

#### [`ProductMediaDelete`](/docs/developer/api-reference/objects/product-media-delete)

Deletes a product media.
