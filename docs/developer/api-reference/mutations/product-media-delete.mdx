title: productMediaDelete
---

Deletes a product media. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productMediaDelete(
  id: ID!
): ProductMediaDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product media to delete.

### Type

#### [`ProductMediaDelete`](../../../developer/api-reference/objects/product-media-delete)

Deletes a product media. Requires one of the following permissions: MANAGE_PRODUCTS.
