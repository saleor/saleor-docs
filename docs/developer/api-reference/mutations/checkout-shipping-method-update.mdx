Updates the shipping method of the checkout.

```graphql
checkoutShippingMethodUpdate(
  checkoutId: ID
  shippingMethodId: ID!
  token: UUID
): CheckoutShippingMethodUpdate

```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `shippingMethodId` ([`ID!`](../../../developer/api-reference/scalars/id))

Shipping method.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutShippingMethodUpdate`](../../../developer/api-reference/objects/checkout-shipping-method-update)

Updates the shipping method of the checkout.
