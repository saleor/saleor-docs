Deletes an authentication token assigned to app. Requires one of the following permissions: MANAGE_APPS.

```graphql
appTokenDelete(
  id: ID!
): AppTokenDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an auth token to delete.

### Type

#### [`AppTokenDelete`](../../../developer/api-reference/objects/app-token-delete)

Deletes an authentication token assigned to app. Requires one of the following permissions: MANAGE_APPS.
