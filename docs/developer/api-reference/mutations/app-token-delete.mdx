Deletes an authentication token assigned to app.

```graphql
appTokenDelete(
  id: ID!
): AppTokenDelete

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an auth token to delete.

### Type

#### [`AppTokenDelete`](/docs/developer/api-reference/objects/app-token-delete)

Deletes an authentication token assigned to app.
