Retries event delivery. Requires one of the following permissions: MANAGE_APPS.

```graphql
eventDeliveryRetry(
  id: ID!
): EventDeliveryRetry
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the event delivery to retry.


### Type

#### [`EventDeliveryRetry`](../../../developer/api-reference/objects/event-delivery-retry) 

Retries event delivery. Requires one of the following permissions: MANAGE_APPS.
