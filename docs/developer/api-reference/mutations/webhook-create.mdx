Creates a new webhook subscription. Requires one of the following permissions: MANAGE_APPS, AUTHENTICATED_APP.

```graphql
webhookCreate(
  input: WebhookCreateInput!
): WebhookCreate
```

### Arguments

#### `input` ([`WebhookCreateInput`](../../../developer/api-reference/inputs/webhook-create-input))

Fields required to create a webhook.

### Type

#### [`WebhookCreate`](../../../developer/api-reference/objects/webhook-create)

Creates a new webhook subscription. Requires one of the following permissions: MANAGE_APPS, AUTHENTICATED_APP.
