Updates site domain of the shop. Requires one of the following permissions: MANAGE_SETTINGS.

```graphql
shopDomainUpdate(
  input: SiteDomainInput
): ShopDomainUpdate
```


### Arguments

#### `input` ([`SiteDomainInput`](../../../developer/api-reference/inputs/site-domain-input))

Fields required to update site.


### Type

#### [`ShopDomainUpdate`](../../../developer/api-reference/objects/shop-domain-update) 

Updates site domain of the shop. Requires one of the following permissions: MANAGE_SETTINGS.
