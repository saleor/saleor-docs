New in Saleor 3.1. Delete gift cards. Note: this feature is in a preview state and can be subject to changes at later point.

```graphql
giftCardBulkDelete(
  ids: [ID]!
): GiftCardBulkDelete

```

### Arguments

#### `ids` ([`[ID]!`](../../../developer/api-reference/scalars/id))

List of gift card IDs to delete.

### Type

#### [`GiftCardBulkDelete`](../../../developer/api-reference/objects/gift-card-bulk-delete)

New in Saleor 3.1. Delete gift cards. Note: this feature is in a preview state and can be subject to changes at later point.
