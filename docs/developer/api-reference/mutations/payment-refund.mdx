---

Refunds the captured payment amount. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
paymentRefund(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentRefund

```

### Arguments

#### `amount` ([`PositiveDecimal`](../../../developer/api-reference/scalars/positive-decimal))

Transaction amount.

#### `paymentId` ([`ID!`](../../../developer/api-reference/scalars/id))

Payment ID.

### Type

#### [`PaymentRefund`](../../../developer/api-reference/objects/payment-refund)

Refunds the captured payment amount. Requires one of the following permissions: MANAGE_ORDERS.
