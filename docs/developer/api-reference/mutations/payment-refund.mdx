Refunds the captured payment amount.

```graphql
paymentRefund(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentRefund

```

### Arguments

#### `amount` ([`PositiveDecimal`](/docs/developer/api-reference/scalars/positive-decimal))

Transaction amount.

#### `paymentId` ([`ID!`](/docs/developer/api-reference/scalars/id))

Payment ID.

### Type

#### [`PaymentRefund`](/docs/developer/api-reference/objects/payment-refund)

Refunds the captured payment amount.
