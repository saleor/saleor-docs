title: voucherCataloguesAdd
---

Adds products, categories, collections to a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
voucherCataloguesAdd(
  id: ID!
  input: CatalogueInput!
): VoucherAddCatalogues
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a voucher.

#### `input` ([`CatalogueInput`](../../../developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of voucher.

### Type

#### [`VoucherAddCatalogues`](../../../developer/api-reference/objects/voucher-add-catalogues)

Adds products, categories, collections to a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.
