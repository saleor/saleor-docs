Retry failed installation of new app.

```graphql
appRetryInstall(
  activateAfterInstallation: Boolean = true
  id: ID!
): AppRetryInstall

```

### Arguments

#### `activateAfterInstallation` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determine if app will be set active or not.

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of failed installation.

### Type

#### [`AppRetryInstall`](/docs/developer/api-reference/objects/app-retry-install)

Retry failed installation of new app.
