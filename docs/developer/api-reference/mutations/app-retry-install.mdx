Retry failed installation of new app. Requires one of the following permissions: MANAGE_APPS.

```graphql
appRetryInstall(
  activateAfterInstallation: Boolean = true
  id: ID!
): AppRetryInstall
```


### Arguments

#### `activateAfterInstallation` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determine if app will be set active or not.


#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of failed installation.


### Type

#### [`AppRetryInstall`](../../../developer/api-reference/objects/app-retry-install) 

Retry failed installation of new app. Requires one of the following permissions: MANAGE_APPS.
