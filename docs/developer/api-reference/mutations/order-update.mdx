Updates an order.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderUpdate(
  id: ID!
  input: OrderUpdateInput!
): OrderUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an order to update.

#### `input` ([`OrderUpdateInput`](../../../developer/api-reference/inputs/order-update-input))

Fields required to update an order.

### Type

#### [`OrderUpdate`](../../../developer/api-reference/objects/order-update)

Updates an order.

Requires one of the following permissions: MANAGE_ORDERS.
