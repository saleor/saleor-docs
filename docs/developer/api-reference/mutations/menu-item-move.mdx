Moves items of menus. Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuItemMove(
  menu: ID!
  moves: [MenuItemMoveInput!]!
): MenuItemMove
```

### Arguments

#### `menu` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the menu.

#### `moves` ([`MenuItemMoveInput`](../../../developer/api-reference/inputs/menu-item-move-input))

The menu position data.

### Type

#### [`MenuItemMove`](../../../developer/api-reference/objects/menu-item-move)

Moves items of menus. Requires one of the following permissions: MANAGE_MENUS.
