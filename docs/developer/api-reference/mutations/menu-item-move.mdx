Moves items of menus.

```graphql
menuItemMove(
  menu: ID!
  moves: [MenuItemMoveInput]!
): MenuItemMove

```

### Arguments

#### `menu` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the menu.

#### `moves` ([`[MenuItemMoveInput]!`](/docs/developer/api-reference/inputs/menu-item-move-input))

The menu position data.

### Type

#### [`MenuItemMove`](/docs/developer/api-reference/objects/menu-item-move)

Moves items of menus.
