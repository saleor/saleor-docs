Request an invoice for the order using plugin.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
invoiceRequest(
  number: String
  orderId: ID!
): InvoiceRequest
```

### Arguments

#### `number` ([`String`](../../../developer/api-reference/scalars/string))

Invoice number, if not provided it will be generated.

#### `orderId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order related to invoice.

### Type

#### [`InvoiceRequest`](../../../developer/api-reference/objects/invoice-request)

Request an invoice for the order using plugin.

Requires one of the following permissions: MANAGE_ORDERS.
