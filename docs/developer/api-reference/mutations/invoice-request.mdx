Request an invoice for the order using plugin.

```graphql
invoiceRequest(
  number: String
  orderId: ID!
): InvoiceRequest

```

### Arguments

#### `number` ([`String`](/docs/developer/api-reference/scalars/string))

Invoice number, if not provided it will be generated.

#### `orderId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order related to invoice.

### Type

#### [`InvoiceRequest`](/docs/developer/api-reference/objects/invoice-request)

Request an invoice for the order using plugin.
