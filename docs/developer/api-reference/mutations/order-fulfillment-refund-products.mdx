Refund products.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderFulfillmentRefundProducts(
  input: OrderRefundProductsInput!
  order: ID!
): FulfillmentRefundProducts
```

### Arguments

#### `input` ([`OrderRefundProductsInput`](../../../developer/api-reference/inputs/order-refund-products-input))

Fields required to create an refund fulfillment.

#### `order` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to be refunded.

### Type

#### [`FulfillmentRefundProducts`](../../../developer/api-reference/objects/fulfillment-refund-products)

Refund products.

Requires one of the following permissions: MANAGE_ORDERS.
