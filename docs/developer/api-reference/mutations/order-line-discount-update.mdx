Fields required to update price for the order line.

#### `orderLineId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a order line to update price

### Type

#### [`OrderLineDiscountUpdate`](/developer/api-reference/objects/order-line-discount-update.mdx)

Update discount for the order line.
