```graphql
orderLineDiscountUpdate(
  input: OrderDiscountCommonInput!
  orderLineId: ID!

```

### Arguments

#### [`input`](#) ([`OrderDiscountCommonInput!`](../../../developer/api-reference/inputs/order-discount-common-input))

Fields required to update price for the order line.

#### [`orderLineId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a order line to update price
