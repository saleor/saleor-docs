Creates/updates translations for an attribute value. Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
attributeValueTranslate(
  id: ID!
  input: AttributeValueTranslationInput!
  languageCode: LanguageCodeEnum!
): AttributeValueTranslate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

AttributeValue ID or AttributeValueTranslatableContent ID.

#### `input` ([`AttributeValueTranslationInput`](../../../developer/api-reference/inputs/attribute-value-translation-input))

#### `languageCode` ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`AttributeValueTranslate`](../../../developer/api-reference/objects/attribute-value-translate)

Creates/updates translations for an attribute value. Requires one of the following permissions: MANAGE_TRANSLATIONS.
