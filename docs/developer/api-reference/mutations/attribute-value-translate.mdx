Creates/updates translations for an attribute value.

```graphql
attributeValueTranslate(
  id: ID!
  input: AttributeValueTranslationInput!
  languageCode: LanguageCodeEnum!
): AttributeValueTranslate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

AttributeValue ID or AttributeValueTranslatableContent ID.

#### `input` ([`AttributeValueTranslationInput!`](/docs/developer/api-reference/inputs/attribute-value-translation-input))

#### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`AttributeValueTranslate`](/docs/developer/api-reference/objects/attribute-value-translate)

Creates/updates translations for an attribute value.
