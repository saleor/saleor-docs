Updates an existing variant for product. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantUpdate(
  id: ID!
  input: ProductVariantInput!
): ProductVariantUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant to update.

#### `input` ([`ProductVariantInput`](../../../developer/api-reference/inputs/product-variant-input))

Fields required to update a product variant.

### Type

#### [`ProductVariantUpdate`](../../../developer/api-reference/objects/product-variant-update)

Updates an existing variant for product. Requires one of the following permissions: MANAGE_PRODUCTS.
