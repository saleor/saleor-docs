Delete object's private metadata.

```graphql
deletePrivateMetadata(
  id: ID!
  keys: [String!]!
): DeletePrivateMetadata

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an object to update.

#### `keys` ([`[String!]!`](/docs/developer/api-reference/scalars/string))

Metadata keys to delete.

### Type

#### [`DeletePrivateMetadata`](/docs/developer/api-reference/objects/delete-private-metadata)

Delete object's private metadata.
