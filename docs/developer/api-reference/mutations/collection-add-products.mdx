Adds products to a collection.

```graphql
collectionAddProducts(
  collectionId: ID!
  products: [ID]!
): CollectionAddProducts

```

### Arguments

#### `collectionId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a collection.

#### `products` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

List of product IDs.

### Type

#### [`CollectionAddProducts`](/docs/developer/api-reference/objects/collection-add-products)

Adds products to a collection.
