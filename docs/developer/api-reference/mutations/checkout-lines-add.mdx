Adds a checkout line to the existing checkout.If line was already in checkout, its quantity will be increased.

```graphql
checkoutLinesAdd(
  checkoutId: ID
  id: ID
  lines: [CheckoutLineInput!]!
  token: UUID
): CheckoutLinesAdd
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The checkout's ID.

Added in Saleor 3.4.

#### `lines` ([`CheckoutLineInput`](../../../developer/api-reference/inputs/checkout-line-input))

A list of checkout lines, each containing information about an item in the checkout.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

### Type

#### [`CheckoutLinesAdd`](../../../developer/api-reference/objects/checkout-lines-add)

Adds a checkout line to the existing checkout.If line was already in checkout, its quantity will be increased.
