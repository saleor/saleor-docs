Requests deletion of an invoice.

```graphql
invoiceRequestDelete(
  id: ID!
): InvoiceRequestDelete

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an invoice to request the deletion.

### Type

#### [`InvoiceRequestDelete`](/docs/developer/api-reference/objects/invoice-request-delete)

Requests deletion of an invoice.
