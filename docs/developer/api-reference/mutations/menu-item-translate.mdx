Creates/updates translations for a menu item. Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
menuItemTranslate(
  id: ID!
  input: NameTranslationInput!
  languageCode: LanguageCodeEnum!
): MenuItemTranslate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

MenuItem ID or MenuItemTranslatableContent ID.

#### `input` ([`NameTranslationInput`](../../../developer/api-reference/inputs/name-translation-input))

#### `languageCode` ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`MenuItemTranslate`](../../../developer/api-reference/objects/menu-item-translate)

Creates/updates translations for a menu item. Requires one of the following permissions: MANAGE_TRANSLATIONS.
