Creates/updates translations for a menu item.

```graphql
menuItemTranslate(
  id: ID!
  input: NameTranslationInput!
  languageCode: LanguageCodeEnum!
): MenuItemTranslate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

MenuItem ID or MenuItemTranslatableContent ID.

#### `input` ([`NameTranslationInput!`](/developer/api-reference/inputs/name-translation-input.mdx))

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`MenuItemTranslate`](/developer/api-reference/objects/menu-item-translate.mdx)

Creates/updates translations for a menu item.
