Updates an existing product.

```graphql
productUpdate(
  id: ID!
  input: ProductInput!
): ProductUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product to update.

#### `input` ([`ProductInput!`](/docs/developer/api-reference/inputs/product-input))

Fields required to update a product.

### Type

#### [`ProductUpdate`](/docs/developer/api-reference/objects/product-update)

Updates an existing product.
