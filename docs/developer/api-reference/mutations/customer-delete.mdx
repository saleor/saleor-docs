title: customerDelete
---

Deletes a customer. Requires one of the following permissions: MANAGE_USERS.

```graphql
customerDelete(
  id: ID!
): CustomerDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a customer to delete.

### Type

#### [`CustomerDelete`](../../../developer/api-reference/objects/customer-delete)

Deletes a customer. Requires one of the following permissions: MANAGE_USERS.
