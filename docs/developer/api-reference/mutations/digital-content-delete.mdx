Remove digital content assigned to given variant.

```graphql
digitalContentDelete(
  variantId: ID!
): DigitalContentDelete
```


### Arguments

#### `variantId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant with digital content to remove.


### Type

#### [`DigitalContentDelete`](../../../developer/api-reference/objects/digital-content-delete) 

Remove digital content assigned to given variant.
