---

Remove digital content assigned to given variant.

```graphql
digitalContentDelete(
  variantId: ID!
): DigitalContentDelete

```

### Arguments

#### `variantId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product variant with digital content to remove.

### Type

#### [`DigitalContentDelete`](/docs/developer/api-reference/objects/digital-content-delete)

Remove digital content assigned to given variant.
