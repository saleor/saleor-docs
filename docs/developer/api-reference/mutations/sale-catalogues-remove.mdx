title: saleCataloguesRemove
---


Removes products, categories, collections from a sale. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
saleCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): SaleRemoveCatalogues
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a sale.


#### `input` ([`CatalogueInput`](../../../developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of sale.


### Type

#### [`SaleRemoveCatalogues`](../../../developer/api-reference/objects/sale-remove-catalogues) 

Removes products, categories, collections from a sale. Requires one of the following permissions: MANAGE_DISCOUNTS.
