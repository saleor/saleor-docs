---

Removes products, categories, collections from a sale.

```graphql
saleCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): SaleRemoveCatalogues

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a sale.

#### `input` ([`CatalogueInput!`](/docs/developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of sale.

### Type

#### [`SaleRemoveCatalogues`](/docs/developer/api-reference/objects/sale-remove-catalogues)

Removes products, categories, collections from a sale.
