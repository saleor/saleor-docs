Updates a sale. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
saleUpdate(
  id: ID!
  input: SaleInput!
): SaleUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a sale to update.

#### `input` ([`SaleInput`](../../../developer/api-reference/inputs/sale-input))

Fields required to update a sale.

### Type

#### [`SaleUpdate`](../../../developer/api-reference/objects/sale-update)

Updates a sale. Requires one of the following permissions: MANAGE_DISCOUNTS.
