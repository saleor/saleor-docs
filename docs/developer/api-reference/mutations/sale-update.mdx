Updates a sale.

```graphql
saleUpdate(
  id: ID!
  input: SaleInput!
): SaleUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a sale to update.

#### `input` ([`SaleInput!`](/docs/developer/api-reference/inputs/sale-input))

Fields required to update a sale.

### Type

#### [`SaleUpdate`](/docs/developer/api-reference/objects/sale-update)

Updates a sale.
