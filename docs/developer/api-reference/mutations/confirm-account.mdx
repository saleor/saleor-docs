Confirm user account with token sent by email during registration.

```graphql
confirmAccount(
  email: String!
  token: String!
): ConfirmAccount
```


### Arguments

#### `email` ([`String`](../../../developer/api-reference/scalars/string))

E-mail of the user performing account confirmation.


#### `token` ([`String`](../../../developer/api-reference/scalars/string))

A one-time token required to confirm the account.


### Type

#### [`ConfirmAccount`](../../../developer/api-reference/objects/confirm-account) 

Confirm user account with token sent by email during registration.
