Updates a category.

```graphql
categoryUpdate(
  id: ID!
  input: CategoryInput!
): CategoryUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a category to update.

#### `input` ([`CategoryInput!`](/docs/developer/api-reference/inputs/category-input))

Fields required to update a category.

### Type

#### [`CategoryUpdate`](/docs/developer/api-reference/objects/category-update)

Updates a category.
