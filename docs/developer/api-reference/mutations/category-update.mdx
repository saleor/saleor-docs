Updates a category.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
categoryUpdate(
  id: ID!
  input: CategoryInput!
): CategoryUpdate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a category to update.

#### [`input`](#) ([`CategoryInput!`](../../../developer/api-reference/inputs/category-input))

Fields required to update a category.

### Type

#### [`CategoryUpdate`](../../../developer/api-reference/objects/category-update)

Updates a category.

Requires one of the following permissions: MANAGE_PRODUCTS.
