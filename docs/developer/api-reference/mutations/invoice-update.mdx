Updates an invoice. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
invoiceUpdate(
  id: ID!
  input: UpdateInvoiceInput!
): InvoiceUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an invoice to update.


#### `input` ([`UpdateInvoiceInput`](../../../developer/api-reference/inputs/update-invoice-input))

Fields to use when updating an invoice.


### Type

#### [`InvoiceUpdate`](../../../developer/api-reference/objects/invoice-update) 

Updates an invoice. Requires one of the following permissions: MANAGE_ORDERS.
