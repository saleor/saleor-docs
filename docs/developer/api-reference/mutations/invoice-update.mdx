Updates an invoice.

```graphql
invoiceUpdate(
  id: ID!
  input: UpdateInvoiceInput!
): InvoiceUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an invoice to update.

#### `input` ([`UpdateInvoiceInput!`](/docs/developer/api-reference/inputs/update-invoice-input))

Fields to use when updating an invoice.

### Type

#### [`InvoiceUpdate`](/docs/developer/api-reference/objects/invoice-update)

Updates an invoice.
