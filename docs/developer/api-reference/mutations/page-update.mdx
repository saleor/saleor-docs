Updates an existing page.

```graphql
pageUpdate(
  id: ID!
  input: PageInput!
): PageUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a page to update.

#### `input` ([`PageInput!`](/docs/developer/api-reference/inputs/page-input))

Fields required to update a page.

### Type

#### [`PageUpdate`](/docs/developer/api-reference/objects/page-update)

Updates an existing page.
