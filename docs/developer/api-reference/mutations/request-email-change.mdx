Request email change of the logged in user. Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
requestEmailChange(
  channel: String
  newEmail: String!
  password: String!
  redirectUrl: String!
): RequestEmailChange
```


### Arguments

#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel which will be used to notify users. Optional when only one channel exists.


#### `newEmail` ([`String`](../../../developer/api-reference/scalars/string))

New user email.


#### `password` ([`String`](../../../developer/api-reference/scalars/string))

User password.


#### `redirectUrl` ([`String`](../../../developer/api-reference/scalars/string))

URL of a view where users should be redirected to update the email address. URL in RFC 1808 format.


### Type

#### [`RequestEmailChange`](../../../developer/api-reference/objects/request-email-change) 

Request email change of the logged in user. Requires one of the following permissions: AUTHENTICATED_USER.
