Request email change of the logged in user. 

Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
requestEmailChange(
  channel: String
  newEmail: String!
  password: String!
  redirectUrl: String!
): RequestEmailChange
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>requestEmailChange.<b>channel</b></code>](#)<Bullet />[`String`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="scalar"/>
> Slug of a channel which will be used to notify users. Optional when only one channel exists.
> 

#### [<code style={{ fontWeight: 'normal' }}>requestEmailChange.<b>newEmail</b></code>](#)<Bullet />[`String!`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> New user email.
> 

#### [<code style={{ fontWeight: 'normal' }}>requestEmailChange.<b>password</b></code>](#)<Bullet />[`String!`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> User password.
> 

#### [<code style={{ fontWeight: 'normal' }}>requestEmailChange.<b>redirectUrl</b></code>](#)<Bullet />[`String!`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> URL of a view where users should be redirected to update the email address. URL in RFC 1808 format.
>
