Creates/updates translations for a category.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
categoryTranslate(
  id: ID!
  input: TranslationInput!
  languageCode: LanguageCodeEnum!
): CategoryTranslate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

Category ID or CategoryTranslatableContent ID.

#### [`input`](#) ([`TranslationInput!`](../../../developer/api-reference/inputs/translation-input))

#### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`CategoryTranslate`](../../../developer/api-reference/objects/category-translate)

Creates/updates translations for a category.

Requires one of the following permissions: MANAGE_TRANSLATIONS.
