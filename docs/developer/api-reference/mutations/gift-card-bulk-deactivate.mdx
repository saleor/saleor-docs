---

New in Saleor 3.1. Deactivate gift cards.

```graphql
giftCardBulkDeactivate(
  ids: [ID]!
): GiftCardBulkDeactivate

```

### Arguments

#### `ids` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

List of gift card IDs to deactivate.

### Type

#### [`GiftCardBulkDeactivate`](/docs/developer/api-reference/objects/gift-card-bulk-deactivate)

New in Saleor 3.1. Deactivate gift cards.
