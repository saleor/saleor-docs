Updates a new shipping price. Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
shippingPriceUpdate(
  id: ID!
  input: ShippingPriceInput!
): ShippingPriceUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a shipping price to update.


#### `input` ([`ShippingPriceInput`](../../../developer/api-reference/inputs/shipping-price-input))

Fields required to update a shipping price.


### Type

#### [`ShippingPriceUpdate`](../../../developer/api-reference/objects/shipping-price-update) 

Updates a new shipping price. Requires one of the following permissions: MANAGE_SHIPPING.
