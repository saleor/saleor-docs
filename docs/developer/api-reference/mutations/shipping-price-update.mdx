Updates a new shipping price.

```graphql
shippingPriceUpdate(
  id: ID!
  input: ShippingPriceInput!
): ShippingPriceUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a shipping price to update.

#### `input` ([`ShippingPriceInput!`](/docs/developer/api-reference/inputs/shipping-price-input))

Fields required to update a shipping price.

### Type

#### [`ShippingPriceUpdate`](/docs/developer/api-reference/objects/shipping-price-update)

Updates a new shipping price.
