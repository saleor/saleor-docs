---

Deletes customers. Requires one of the following permissions: MANAGE_USERS.

```graphql
customerBulkDelete(
  ids: [ID!]!
): CustomerBulkDelete

```

### Arguments

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of user IDs to delete.

### Type

#### [`CustomerBulkDelete`](../../../developer/api-reference/objects/customer-bulk-delete)

Deletes customers. Requires one of the following permissions: MANAGE_USERS.
