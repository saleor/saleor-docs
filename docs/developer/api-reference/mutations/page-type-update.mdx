Update page type.

```graphql
pageTypeUpdate(
  id: ID
  input: PageTypeUpdateInput!
): PageTypeUpdate

```

### Arguments

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the page type to update.

#### `input` ([`PageTypeUpdateInput!`](/docs/developer/api-reference/inputs/page-type-update-input))

Fields required to update page type.

### Type

#### [`PageTypeUpdate`](/docs/developer/api-reference/objects/page-type-update)

Update page type.
