Update page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
pageTypeUpdate(
  id: ID
  input: PageTypeUpdateInput!
): PageTypeUpdate

```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the page type to update.

#### `input` ([`PageTypeUpdateInput!`](../../../developer/api-reference/inputs/page-type-update-input))

Fields required to update page type.

### Type

#### [`PageTypeUpdate`](../../../developer/api-reference/objects/page-type-update)

Update page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.
