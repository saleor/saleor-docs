Manage sale's availability in channels. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
saleChannelListingUpdate(
  id: ID!
  input: SaleChannelListingInput!
): SaleChannelListingUpdate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a sale to update.

#### `input` ([`SaleChannelListingInput!`](../../../developer/api-reference/inputs/sale-channel-listing-input))

Fields required to update sale channel listings.

### Type

#### [`SaleChannelListingUpdate`](../../../developer/api-reference/objects/sale-channel-listing-update)

Manage sale's availability in channels. Requires one of the following permissions: MANAGE_DISCOUNTS.
