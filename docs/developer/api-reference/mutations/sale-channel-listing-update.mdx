Manage sale's availability in channels.

```graphql
saleChannelListingUpdate(
  id: ID!
  input: SaleChannelListingInput!

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

#### `input` ([`SaleChannelListingInput!`](/docs/developer/api-reference/inputs/sale-channel-listing-input))

Fields required to update sale channel listings.

#### [`SaleChannelListingUpdate`](/docs/developer/api-reference/objects/sale-channel-listing-update)

Manage sale's availability in channels.
