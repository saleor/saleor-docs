Sends an email with the account removal link for the logged-in user.

```graphql
accountRequestDeletion(
  channel: String
  redirectUrl: String!
): AccountRequestDeletion

```

### Arguments

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel which will be used to notify users. Optional when only one channel exists.

#### `redirectUrl` ([`String!`](/developer/api-reference/scalars/string.mdx))

URL of a view where users should be redirected to delete their account. URL in RFC 1808 format.

### Type

#### [`AccountRequestDeletion`](/developer/api-reference/objects/account-request-deletion.mdx)

Sends an email with the account removal link for the logged-in user.
