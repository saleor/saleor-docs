Creates a value for an attribute.

```graphql
attributeValueCreate(
  attribute: ID!
  input: AttributeValueCreateInput!
): AttributeValueCreate

```

### Arguments

#### `attribute` ([`ID!`](/docs/developer/api-reference/scalars/id))

Attribute to which value will be assigned.

#### `input` ([`AttributeValueCreateInput!`](/docs/developer/api-reference/inputs/attribute-value-create-input))

Fields required to create an AttributeValue.

### Type

#### [`AttributeValueCreate`](/docs/developer/api-reference/objects/attribute-value-create)

Creates a value for an attribute.
