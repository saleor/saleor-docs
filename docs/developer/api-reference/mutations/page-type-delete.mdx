Delete a page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
pageTypeDelete(
  id: ID!
): PageTypeDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the page type to delete.

### Type

#### [`PageTypeDelete`](../../../developer/api-reference/objects/page-type-delete)

Delete a page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.
