Updates a voucher.

```graphql
voucherUpdate(
  id: ID!
  input: VoucherInput!
): VoucherUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a voucher to update.

#### `input` ([`VoucherInput!`](/developer/api-reference/inputs/voucher-input.mdx))

Fields required to update a voucher.

### Type

#### [`VoucherUpdate`](/developer/api-reference/objects/voucher-update.mdx)

Updates a voucher.
