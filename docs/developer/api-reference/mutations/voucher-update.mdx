Updates a voucher.

```graphql
voucherUpdate(
  id: ID!
  input: VoucherInput!
): VoucherUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a voucher to update.

#### `input` ([`VoucherInput!`](/docs/developer/api-reference/inputs/voucher-input))

Fields required to update a voucher.

### Type

#### [`VoucherUpdate`](/docs/developer/api-reference/objects/voucher-update)

Updates a voucher.
