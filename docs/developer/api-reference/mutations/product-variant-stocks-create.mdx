---


Creates stocks for product variant. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantStocksCreate(
  stocks: [StockInput!]!
  variantId: ID!
): ProductVariantStocksCreate
```


### Arguments

#### `stocks` ([`StockInput`](../../../developer/api-reference/inputs/stock-input))

Input list of stocks to create.


#### `variantId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant for which stocks will be created.


### Type

#### [`ProductVariantStocksCreate`](../../../developer/api-reference/objects/product-variant-stocks-create) 

Creates stocks for product variant. Requires one of the following permissions: MANAGE_PRODUCTS.
