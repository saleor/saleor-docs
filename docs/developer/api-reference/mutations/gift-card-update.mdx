Update a gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.

```graphql
giftCardUpdate(
  id: ID!
  input: GiftCardUpdateInput!
): GiftCardUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a gift card to update.

#### `input` ([`GiftCardUpdateInput`](../../../developer/api-reference/inputs/gift-card-update-input))

Fields required to update a gift card.

### Type

#### [`GiftCardUpdate`](../../../developer/api-reference/objects/gift-card-update)

Update a gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.
