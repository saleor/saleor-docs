Update a gift card.

```graphql
giftCardUpdate(
  id: ID!
  input: GiftCardUpdateInput!
): GiftCardUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a gift card to update.

#### `input` ([`GiftCardUpdateInput!`](/docs/developer/api-reference/inputs/gift-card-update-input))

Fields required to update a gift card.

### Type

#### [`GiftCardUpdate`](/docs/developer/api-reference/objects/gift-card-update)

Update a gift card.
