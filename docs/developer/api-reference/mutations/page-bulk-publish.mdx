---

Publish pages.

```graphql
pageBulkPublish(
  ids: [ID]!
  isPublished: Boolean!
): PageBulkPublish

```

### Arguments

#### `ids` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

List of page IDs to (un)publish.

#### `isPublished` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Determine if pages will be published or not.

### Type

#### [`PageBulkPublish`](/docs/developer/api-reference/objects/page-bulk-publish)

Publish pages.
