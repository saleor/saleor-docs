title: pageBulkPublish
---


Publish pages. Requires one of the following permissions: MANAGE_PAGES.

```graphql
pageBulkPublish(
  ids: [ID!]!
  isPublished: Boolean!
): PageBulkPublish
```


### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of page IDs to (un)publish.


#### `isPublished` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determine if pages will be published or not.


### Type

#### [`PageBulkPublish`](../../../developer/api-reference/objects/page-bulk-publish) 

Publish pages. Requires one of the following permissions: MANAGE_PAGES.
