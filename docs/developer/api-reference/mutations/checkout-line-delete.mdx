Deletes a CheckoutLine.

```graphql
checkoutLineDelete(
  checkoutId: ID = 0
  lineId: ID = 0
  token: UUID
): CheckoutLineDelete

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

The ID of the checkout.DEPRECATED: Will be removed in Saleor 4.0. Use token instead.

#### `lineId` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the checkout line to delete.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutLineDelete`](/developer/api-reference/objects/checkout-line-delete.mdx)

Deletes a CheckoutLine.
