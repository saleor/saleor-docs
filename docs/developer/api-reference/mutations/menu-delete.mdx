Deletes a menu. Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuDelete(
  id: ID!
): MenuDelete
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a menu to delete.


### Type

#### [`MenuDelete`](../../../developer/api-reference/objects/menu-delete) 

Deletes a menu. Requires one of the following permissions: MANAGE_MENUS.
