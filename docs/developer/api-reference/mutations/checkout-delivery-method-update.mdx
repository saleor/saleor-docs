---

New in Saleor 3.1. Updates the delivery method (shipping method or pick up point) of the checkout.

```graphql
checkoutDeliveryMethodUpdate(
  deliveryMethodId: ID = 0
  token: UUID
): CheckoutDeliveryMethodUpdate

```

### Arguments

#### `deliveryMethodId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Delivery Method ID (`Warehouse` ID or `ShippingMethod` ID).

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutDeliveryMethodUpdate`](/developer/api-reference/objects/checkout-delivery-method-update.mdx)

New in Saleor 3.1. Updates the delivery method (shipping method or pick up point) of the checkout.
