Updates an existing app.

```graphql
appUpdate(
  id: ID!
  input: AppInput!
): AppUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an app to update.

#### `input` ([`AppInput!`](/docs/developer/api-reference/inputs/app-input))

Fields required to update an existing app.

### Type

#### [`AppUpdate`](/docs/developer/api-reference/objects/app-update)

Updates an existing app.
