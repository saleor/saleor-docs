Updates an existing app. Requires one of the following permissions: MANAGE_APPS.

```graphql
appUpdate(
  id: ID!
  input: AppInput!
): AppUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an app to update.


#### `input` ([`AppInput`](../../../developer/api-reference/inputs/app-input))

Fields required to update an existing app.


### Type

#### [`AppUpdate`](../../../developer/api-reference/objects/app-update) 

Updates an existing app. Requires one of the following permissions: MANAGE_APPS.
