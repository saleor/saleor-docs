Creates/Updates translations for collection.

```graphql
collectionTranslate(
  id: ID!
  input: TranslationInput!
  languageCode: LanguageCodeEnum!
): CollectionTranslate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Collection ID.

#### `input` ([`TranslationInput!`](/developer/api-reference/inputs/translation-input.mdx))

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`CollectionTranslate`](/developer/api-reference/objects/collection-translate.mdx)

Creates/Updates translations for collection.
