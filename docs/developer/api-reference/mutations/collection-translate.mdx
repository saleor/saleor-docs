Creates/updates translations for a collection.

```graphql
collectionTranslate(
  id: ID!
  input: TranslationInput!
  languageCode: LanguageCodeEnum!
): CollectionTranslate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

Collection ID or CollectionTranslatableContent ID.

#### `input` ([`TranslationInput!`](/docs/developer/api-reference/inputs/translation-input))

#### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`CollectionTranslate`](/docs/developer/api-reference/objects/collection-translate)

Creates/updates translations for a collection.
