Reorder page attribute values.

```graphql
pageReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  pageId: ID!
): PageReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/docs/developer/api-reference/inputs/reorder-input))

The list of reordering operations for given attribute values.

#### `pageId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a page.

### Type

#### [`PageReorderAttributeValues`](/docs/developer/api-reference/objects/page-reorder-attribute-values)

Reorder page attribute values.
