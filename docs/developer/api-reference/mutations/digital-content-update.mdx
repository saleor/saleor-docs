Update digital content.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
digitalContentUpdate(
  input: DigitalContentInput!
  variantId: ID!
): DigitalContentUpdate
```

### Arguments

#### `input` ([`DigitalContentInput`](../../../developer/api-reference/inputs/digital-content-input))

Fields required to update a digital content.

#### `variantId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant with digital content to update.

### Type

#### [`DigitalContentUpdate`](../../../developer/api-reference/objects/digital-content-update)

Update digital content.

Requires one of the following permissions: MANAGE_PRODUCTS.
