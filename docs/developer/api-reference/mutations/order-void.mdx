Void an order.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderVoid(
  id: ID!
): OrderVoid
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to void.

### Type

#### [`OrderVoid`](../../../developer/api-reference/objects/order-void)

Void an order.

Requires one of the following permissions: MANAGE_ORDERS.
