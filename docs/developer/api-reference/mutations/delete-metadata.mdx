Delete metadata of an object.

```graphql
deleteMetadata(
  id: ID!
  keys: [String!]!
): DeleteMetadata

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an object to update.

#### `keys` ([`[String!]!`](/docs/developer/api-reference/scalars/string))

Metadata keys to delete.

### Type

#### [`DeleteMetadata`](/docs/developer/api-reference/objects/delete-metadata)

Delete metadata of an object.
