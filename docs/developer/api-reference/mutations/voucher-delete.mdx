Deletes a voucher.

```graphql
voucherDelete(
  id: ID!
): VoucherDelete

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a voucher to delete.

### Type

#### [`VoucherDelete`](/docs/developer/api-reference/objects/voucher-delete)

Deletes a voucher.
