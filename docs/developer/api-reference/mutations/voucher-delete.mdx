Deletes a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
voucherDelete(
  id: ID!
): VoucherDelete
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a voucher to delete.


### Type

#### [`VoucherDelete`](../../../developer/api-reference/objects/voucher-delete) 

Deletes a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.
