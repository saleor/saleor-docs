Creates new fulfillments for an order.

```graphql
orderFulfill(
  input: OrderFulfillInput!
  order: ID = 0
): OrderFulfill

```

### Arguments

#### `input` ([`OrderFulfillInput!`](/developer/api-reference/inputs/order-fulfill-input.mdx))

Fields required to create a fulfillment.

#### `order` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the order to be fulfilled.

### Type

#### [`OrderFulfill`](/developer/api-reference/objects/order-fulfill.mdx)

Creates new fulfillments for an order.
