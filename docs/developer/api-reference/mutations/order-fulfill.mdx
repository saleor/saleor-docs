---

Creates new fulfillments for an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderFulfill(
  input: OrderFulfillInput!
  order: ID
): OrderFulfill

```

### Arguments

#### `input` ([`OrderFulfillInput!`](../../../developer/api-reference/inputs/order-fulfill-input))

Fields required to create a fulfillment.

#### `order` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to be fulfilled.

### Type

#### [`OrderFulfill`](../../../developer/api-reference/objects/order-fulfill)

Creates new fulfillments for an order. Requires one of the following permissions: MANAGE_ORDERS.
