Creates new fulfillments for an order.

```graphql
orderFulfill(
  input: OrderFulfillInput!
  order: ID
): OrderFulfill

```

### Arguments

#### `input` ([`OrderFulfillInput!`](/docs/developer/api-reference/inputs/order-fulfill-input))

Fields required to create a fulfillment.

#### `order` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the order to be fulfilled.

### Type

#### [`OrderFulfill`](/docs/developer/api-reference/objects/order-fulfill)

Creates new fulfillments for an order.
