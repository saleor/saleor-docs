Activate a gift card. Requires one of the following permissions: MANAGE_GIFT_CARD.

```graphql
giftCardActivate(
  id: ID!
): GiftCardActivate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a gift card to activate.

### Type

#### [`GiftCardActivate`](../../../developer/api-reference/objects/gift-card-activate)

Activate a gift card. Requires one of the following permissions: MANAGE_GIFT_CARD.
