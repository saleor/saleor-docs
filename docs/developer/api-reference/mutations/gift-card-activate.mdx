Activate a gift card.

```graphql
giftCardActivate(
  id: ID!
): GiftCardActivate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a gift card to activate.

### Type

#### [`GiftCardActivate`](/docs/developer/api-reference/objects/gift-card-activate)

Activate a gift card.
