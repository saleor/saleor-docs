title: unassignWarehouseShippingZone
---

Remove shipping zone from given warehouse.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
unassignWarehouseShippingZone(
  id: ID!
  shippingZoneIds: [ID!]!
): WarehouseShippingZoneUnassign
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a warehouse to update.

#### `shippingZoneIds` ([`ID`](../../../developer/api-reference/scalars/id))

List of shipping zone IDs.

### Type

#### [`WarehouseShippingZoneUnassign`](../../../developer/api-reference/objects/warehouse-shipping-zone-unassign)

Remove shipping zone from given warehouse.

Requires one of the following permissions: MANAGE_PRODUCTS.
