---

Reorder product variant attribute values. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput!]!
  variantId: ID!
): ProductVariantReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an attribute.

#### `moves` ([`[ReorderInput!]!`](../../../developer/api-reference/inputs/reorder-input))

The list of reordering operations for given attribute values.

#### `variantId` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product variant.

### Type

#### [`ProductVariantReorderAttributeValues`](../../../developer/api-reference/objects/product-variant-reorder-attribute-values)

Reorder product variant attribute values. Requires one of the following permissions: MANAGE_PRODUCTS.
