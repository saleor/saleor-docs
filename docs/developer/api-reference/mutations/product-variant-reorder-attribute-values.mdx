Reorder product variant attribute values.

```graphql
productVariantReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  variantId: ID!
): ProductVariantReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/docs/developer/api-reference/inputs/reorder-input))

The list of reordering operations for given attribute values.

#### `variantId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product variant.

### Type

#### [`ProductVariantReorderAttributeValues`](/docs/developer/api-reference/objects/product-variant-reorder-attribute-values)

Reorder product variant attribute values.
