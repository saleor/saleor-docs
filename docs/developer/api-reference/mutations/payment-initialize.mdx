Initializes payment process when it is required by gateway.

```graphql
paymentInitialize(
  channel: String
  gateway: String!
  paymentData: JSONString
): PaymentInitialize

```

### Arguments

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `gateway` ([`String!`](/developer/api-reference/scalars/string.mdx))

A gateway name used to initialize the payment.

#### `paymentData` ([`JSONString`](/developer/api-reference/scalars/json-string.mdx))

Client-side generated data required to initialize the payment.

### Type

#### [`PaymentInitialize`](/developer/api-reference/objects/payment-initialize.mdx)

Initializes payment process when it is required by gateway.
