Creates a ready to send invoice.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
invoiceCreate(
  input: InvoiceCreateInput!
  orderId: ID!
): InvoiceCreate
```

### Arguments

#### [`input`](#) ([`InvoiceCreateInput!`](../../../developer/api-reference/inputs/invoice-create-input))

Fields required when creating an invoice.

#### [`orderId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the order related to invoice.

### Type

#### [`InvoiceCreate`](../../../developer/api-reference/objects/invoice-create)

Creates a ready to send invoice.

Requires one of the following permissions: MANAGE_ORDERS.
