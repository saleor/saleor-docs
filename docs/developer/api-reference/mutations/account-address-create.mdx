Create a new address for the customer.

```graphql
accountAddressCreate(
  input: AddressInput!
  type: AddressTypeEnum
): AccountAddressCreate

```

### Arguments

#### `input` ([`AddressInput!`](../../../developer/api-reference/inputs/address-input))

Fields required to create address.

#### `type` ([`AddressTypeEnum`](../../../developer/api-reference/enums/address-type-enum))

A type of address. If provided, the new address will be automatically assigned as the customer's default address of that type.

### Type

#### [`AccountAddressCreate`](../../../developer/api-reference/objects/account-address-create)

Create a new address for the customer.
