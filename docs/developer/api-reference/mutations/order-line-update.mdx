Updates an order line of an order.

```graphql
orderLineUpdate(
  id: ID!
  input: OrderLineInput!
): OrderLineUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order line to update.

#### `input` ([`OrderLineInput!`](/docs/developer/api-reference/inputs/order-line-input))

Fields required to update an order line.

### Type

#### [`OrderLineUpdate`](/docs/developer/api-reference/objects/order-line-update)

Updates an order line of an order.
