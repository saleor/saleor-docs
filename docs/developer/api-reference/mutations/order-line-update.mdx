Updates an order line of an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderLineUpdate(
  id: ID!
  input: OrderLineInput!
): OrderLineUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order line to update.

#### `input` ([`OrderLineInput`](../../../developer/api-reference/inputs/order-line-input))

Fields required to update an order line.

### Type

#### [`OrderLineUpdate`](../../../developer/api-reference/objects/order-line-update)

Updates an order line of an order. Requires one of the following permissions: MANAGE_ORDERS.
