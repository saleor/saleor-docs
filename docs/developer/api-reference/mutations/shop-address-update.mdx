Update the shop's address. If the `null` value is passed, the currently selected address will be deleted.

```graphql
shopAddressUpdate(
  input: AddressInput
): ShopAddressUpdate

```

### Arguments

#### `input` ([`AddressInput`](../../../developer/api-reference/inputs/address-input))

Fields required to update shop address.

### Type

#### [`ShopAddressUpdate`](../../../developer/api-reference/objects/shop-address-update)

Update the shop's address. If the `null` value is passed, the currently selected address will be deleted.
