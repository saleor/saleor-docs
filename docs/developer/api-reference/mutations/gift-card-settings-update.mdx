```graphql
giftCardSettingsUpdate(

```

#### `input` ([`GiftCardSettingsUpdateInput`](../../../developer/api-reference/inputs/gift-card-settings-update-input))

Fields required to update gift card settings.

### Type

#### [`GiftCardSettingsUpdate`](../../../developer/api-reference/objects/gift-card-settings-update)

Update gift card settings. Requires one of the following permissions: MANAGE_GIFT_CARD.
