Deletes product variants.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantBulkDelete(
  ids: [ID!]!

```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of product variant IDs to delete.

### Type

#### [`ProductVariantBulkDelete`](../../../developer/api-reference/objects/product-variant-bulk-delete)

Deletes product variants.

Requires one of the following permissions: MANAGE_PRODUCTS.
