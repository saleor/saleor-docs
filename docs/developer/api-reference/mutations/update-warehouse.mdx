Updates given warehouse.

```graphql
updateWarehouse(
  id: ID!
  input: WarehouseUpdateInput!
): WarehouseUpdate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a warehouse to update.

#### `input` ([`WarehouseUpdateInput!`](../../../developer/api-reference/inputs/warehouse-update-input))

Fields required to update warehouse.

### Type

#### [`WarehouseUpdate`](../../../developer/api-reference/objects/warehouse-update)

Updates given warehouse.
