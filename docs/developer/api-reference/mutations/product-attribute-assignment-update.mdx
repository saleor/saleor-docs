---

New in Saleor 3.1. Update attributes assigned to product variant for given product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productAttributeAssignmentUpdate(
  operations: [ProductAttributeAssignmentUpdateInput!]!
  productTypeId: ID!
): ProductAttributeAssignmentUpdate
```

### Arguments

#### `operations` ([`ProductAttributeAssignmentUpdateInput`](../../../developer/api-reference/inputs/product-attribute-assignment-update-input))

The operations to perform.

#### `productTypeId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the product type to assign the attributes into.

### Type

#### [`ProductAttributeAssignmentUpdate`](../../../developer/api-reference/objects/product-attribute-assignment-update)

New in Saleor 3.1. Update attributes assigned to product variant for given product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
