Deletes a collection. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionDelete(
  id: ID!
): CollectionDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a collection to delete.

### Type

#### [`CollectionDelete`](../../../developer/api-reference/objects/collection-delete)

Deletes a collection. Requires one of the following permissions: MANAGE_PRODUCTS.
