Prepare external authentication url for user by custom plugin.

```graphql
externalAuthenticationUrl(
  input: JSONString!
  pluginId: String!
): ExternalAuthenticationUrl
```


### Arguments

#### `input` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

The data required by plugin to create external authentication url.


#### `pluginId` ([`String`](../../../developer/api-reference/scalars/string))

The ID of the authentication plugin.


### Type

#### [`ExternalAuthenticationUrl`](../../../developer/api-reference/objects/external-authentication-url) 

Prepare external authentication url for user by custom plugin.
