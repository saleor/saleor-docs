Update permission group.

Requires one of the following permissions: MANAGE_STAFF.

```graphql
permissionGroupUpdate(
  id: ID!
  input: PermissionGroupUpdateInput!
): PermissionGroupUpdate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the group to update.

#### [`input`](#) ([`PermissionGroupUpdateInput!`](../../../developer/api-reference/inputs/permission-group-update-input))

Input fields to create permission group.

#### [`PermissionGroupUpdate`](../../../developer/api-reference/objects/permission-group-update)

Update permission group.
