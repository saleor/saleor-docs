Assign attributes to a given page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
pageAttributeAssign(
  attributeIds: [ID!]!
  pageTypeId: ID!
): PageAttributeAssign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

The IDs of the attributes to assign.

#### `pageTypeId` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the page type to assign the attributes into.

### Type

#### [`PageAttributeAssign`](../../../developer/api-reference/objects/page-attribute-assign)

Assign attributes to a given page type. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.
