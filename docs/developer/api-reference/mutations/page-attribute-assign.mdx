Assign attributes to a given page type.

```graphql
pageAttributeAssign(
  attributeIds: [ID!]!
  pageTypeId: ID!
): PageAttributeAssign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](/docs/developer/api-reference/scalars/id))

The IDs of the attributes to assign.

#### `pageTypeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the page type to assign the attributes into.

### Type

#### [`PageAttributeAssign`](/docs/developer/api-reference/objects/page-attribute-assign)

Assign attributes to a given page type.
