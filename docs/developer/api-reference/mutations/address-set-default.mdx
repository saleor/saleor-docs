Sets a default address for the given user.

Requires one of the following permissions: MANAGE_USERS.

```graphql
addressSetDefault(
  addressId: ID!
  type: AddressTypeEnum!
  userId: ID!
): AddressSetDefault
```

### Arguments

#### `addressId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the address.

#### `type` ([`AddressTypeEnum`](../../../developer/api-reference/enums/address-type-enum))

The type of address.

#### `userId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the user to change the address for.

### Type

#### [`AddressSetDefault`](../../../developer/api-reference/objects/address-set-default)

Sets a default address for the given user.

Requires one of the following permissions: MANAGE_USERS.
