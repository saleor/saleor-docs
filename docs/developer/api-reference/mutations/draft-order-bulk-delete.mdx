Deletes draft orders. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
draftOrderBulkDelete(
  ids: [ID!]!
): DraftOrderBulkDelete
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of draft order IDs to delete.

### Type

#### [`DraftOrderBulkDelete`](../../../developer/api-reference/objects/draft-order-bulk-delete)

Deletes draft orders. Requires one of the following permissions: MANAGE_ORDERS.
