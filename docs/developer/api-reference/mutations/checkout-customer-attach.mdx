Sets the customer as the owner of the checkout.

```graphql
checkoutCustomerAttach(
  checkoutId: ID = 0
  customerId: ID = 0
  token: UUID
): CheckoutCustomerAttach

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `customerId` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of customer to attach to checkout. Can be used to attach customer to checkout by staff or app. Requires IMPERSONATE_USER permission.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutCustomerAttach`](/developer/api-reference/objects/checkout-customer-attach.mdx)

Sets the customer as the owner of the checkout.
