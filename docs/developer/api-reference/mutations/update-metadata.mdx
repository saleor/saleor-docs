Updates metadata of an object.

```graphql
updateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdateMetadata

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID or token (for Order and Checkout) of an object to update.

#### `input` ([`[MetadataInput!]!`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the object's metadata.

### Type

#### [`UpdateMetadata`](../../../developer/api-reference/objects/update-metadata)

Updates metadata of an object.
