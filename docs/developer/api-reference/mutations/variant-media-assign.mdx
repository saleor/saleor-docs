Assign an media to a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
variantMediaAssign(
  mediaId: ID!
  variantId: ID!
): VariantMediaAssign
```

### Arguments

#### [`mediaId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product media to assign to a variant.

#### [`variantId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product variant.

### Type

#### [`VariantMediaAssign`](../../../developer/api-reference/objects/variant-media-assign)

Assign an media to a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.
