Updates attribute. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeUpdate(
  id: ID!
  input: AttributeUpdateInput!
): AttributeUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an attribute to update.

#### `input` ([`AttributeUpdateInput`](../../../developer/api-reference/inputs/attribute-update-input))

Fields required to update an attribute.

### Type

#### [`AttributeUpdate`](../../../developer/api-reference/objects/attribute-update)

Updates attribute. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
