Unassign attributes from a given page type.

```graphql
pageAttributeUnassign(
  attributeIds: [ID!]!
  pageTypeId: ID!
): PageAttributeUnassign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](/docs/developer/api-reference/scalars/id))

The IDs of the attributes to unassign.

#### `pageTypeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the page type from which the attributes should be unassign.

### Type

#### [`PageAttributeUnassign`](/docs/developer/api-reference/objects/page-attribute-unassign)

Unassign attributes from a given page type.
