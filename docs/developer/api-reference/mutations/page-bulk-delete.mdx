---


Deletes pages. Requires one of the following permissions: MANAGE_PAGES.

```graphql
pageBulkDelete(
  ids: [ID!]!
): PageBulkDelete
```


### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of page IDs to delete.


### Type

#### [`PageBulkDelete`](../../../developer/api-reference/objects/page-bulk-delete) 

Deletes pages. Requires one of the following permissions: MANAGE_PAGES.
