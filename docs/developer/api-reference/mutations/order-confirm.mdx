Confirms an unconfirmed order by changing status to unfulfilled.

```graphql
orderConfirm(
  id: ID!
): OrderConfirm

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an order to confirm.

### Type

#### [`OrderConfirm`](/docs/developer/api-reference/objects/order-confirm)

Confirms an unconfirmed order by changing status to unfulfilled.
