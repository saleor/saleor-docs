Confirms an unconfirmed order by changing status to unfulfilled. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderConfirm(
  id: ID!
): OrderConfirm
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an order to confirm.

### Type

#### [`OrderConfirm`](../../../developer/api-reference/objects/order-confirm)

Confirms an unconfirmed order by changing status to unfulfilled. Requires one of the following permissions: MANAGE_ORDERS.
