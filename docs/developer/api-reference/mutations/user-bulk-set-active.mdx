---

Activate or deactivate users.

```graphql
userBulkSetActive(
  ids: [ID]!
  isActive: Boolean!
): UserBulkSetActive

```

### Arguments

#### `ids` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

List of user IDs to (de)activate).

#### `isActive` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Determine if users will be set active or not.

### Type

#### [`UserBulkSetActive`](/docs/developer/api-reference/objects/user-bulk-set-active)

Activate or deactivate users.
