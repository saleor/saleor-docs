title: userBulkSetActive
---

Activate or deactivate users. Requires one of the following permissions: MANAGE_USERS.

```graphql
userBulkSetActive(
  ids: [ID!]!
  isActive: Boolean!
): UserBulkSetActive

```

### Arguments

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of user IDs to (de)activate).

#### `isActive` ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

Determine if users will be set active or not.

### Type

#### [`UserBulkSetActive`](../../../developer/api-reference/objects/user-bulk-set-active)

Activate or deactivate users. Requires one of the following permissions: MANAGE_USERS.
