Remove user account.

```graphql
accountDelete(
  token: String!
): AccountDelete

```

### Arguments

#### `token` ([`String!`](/docs/developer/api-reference/scalars/string))

A one-time token required to remove account. Sent by email using AccountRequestDeletion mutation.

### Type

#### [`AccountDelete`](/docs/developer/api-reference/objects/account-delete)

Remove user account.
