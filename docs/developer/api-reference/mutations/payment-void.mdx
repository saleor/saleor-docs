Voids the authorized payment. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
paymentVoid(
  paymentId: ID!
): PaymentVoid
```


### Arguments

#### `paymentId` ([`ID`](../../../developer/api-reference/scalars/id))

Payment ID.


### Type

#### [`PaymentVoid`](../../../developer/api-reference/objects/payment-void) 

Voids the authorized payment. Requires one of the following permissions: MANAGE_ORDERS.
