Deletes a draft order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
draftOrderDelete(
  id: ID!
): DraftOrderDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a draft order to delete.

### Type

#### [`DraftOrderDelete`](../../../developer/api-reference/objects/draft-order-delete)

Deletes a draft order. Requires one of the following permissions: MANAGE_ORDERS.
