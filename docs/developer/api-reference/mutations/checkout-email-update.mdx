Updates email address in the existing checkout object.

```graphql
checkoutEmailUpdate(
  checkoutId: ID
  email: String!
  token: UUID
): CheckoutEmailUpdate

```

### Arguments

#### `checkoutId` ([`ID`](/docs/developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `email` ([`String!`](/docs/developer/api-reference/scalars/string))

email.

#### `token` ([`UUID`](/docs/developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutEmailUpdate`](/docs/developer/api-reference/objects/checkout-email-update)

Updates email address in the existing checkout object.
