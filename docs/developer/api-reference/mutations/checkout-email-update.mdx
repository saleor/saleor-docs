Updates email address in the existing checkout object.

```graphql
checkoutEmailUpdate(
  checkoutId: ID = 0
  email: String!
  token: UUID
): CheckoutEmailUpdate

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Checkout ID.DEPRECATED: Will be removed in Saleor 4.0. Use token instead.

#### `email` ([`String!`](/developer/api-reference/scalars/string.mdx))

email.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutEmailUpdate`](/developer/api-reference/objects/checkout-email-update.mdx)

Updates email address in the existing checkout object.
