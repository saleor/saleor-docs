Reorder the attributes of a product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productTypeReorderAttributes(
  moves: [ReorderInput!]!
  productTypeId: ID!
  type: ProductAttributeType!
): ProductTypeReorderAttributes
```


### Arguments

#### `moves` ([`ReorderInput`](../../../developer/api-reference/inputs/reorder-input))

The list of attribute reordering operations.


#### `productTypeId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product type.


#### `type` ([`ProductAttributeType`](../../../developer/api-reference/enums/product-attribute-type))

The attribute type to reorder.


### Type

#### [`ProductTypeReorderAttributes`](../../../developer/api-reference/objects/product-type-reorder-attributes) 

Reorder the attributes of a product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
