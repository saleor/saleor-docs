Reorder the attributes of a product type.

```graphql
productTypeReorderAttributes(
  moves: [ReorderInput]!
  productTypeId: ID!
  type: ProductAttributeType!
): ProductTypeReorderAttributes

```

### Arguments

#### `moves` ([`[ReorderInput]!`](/developer/api-reference/inputs/reorder-input.mdx))

The list of attribute reordering operations.

#### `productTypeId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product type.

#### `type` ([`ProductAttributeType!`](/developer/api-reference/enums/product-attribute-type.mdx))

The attribute type to reorder.

### Type

#### [`ProductTypeReorderAttributes`](/developer/api-reference/objects/product-type-reorder-attributes.mdx)

Reorder the attributes of a product type.
