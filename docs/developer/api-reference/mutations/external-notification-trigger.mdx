Trigger sending a notification with the notify plugin method. Serializes nodes provided as ids parameter and includes this data in the notification payload.

Added in Saleor 3.1.

```graphql
externalNotificationTrigger(
  channel: String!
  input: ExternalNotificationTriggerInput!
  pluginId: String
): ExternalNotificationTrigger
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>externalNotificationTrigger.<b>channel</b></code>](#)<Bullet />[`String!`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> Channel slug. Saleor will send a notification within a provided channel. Please, make sure that necessary plugins are active.

#### [<code style={{ fontWeight: 'normal' }}>externalNotificationTrigger.<b>input</b></code>](#)<Bullet />[`ExternalNotificationTriggerInput!`](../../../developer/api-reference/inputs/external-notification-trigger-input) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>

> Input for External Notification Trigger.

#### [<code style={{ fontWeight: 'normal' }}>externalNotificationTrigger.<b>pluginId</b></code>](#)<Bullet />[`String`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> The ID of notification plugin.

### Type

#### [`ExternalNotificationTrigger`](../../../developer/api-reference/objects/external-notification-trigger) <Badge class="secondary" text="object"/>

> Trigger sending a notification with the notify plugin method. Serializes nodes provided as ids parameter and includes this data in the notification payload.

Added in Saleor 3.1.
