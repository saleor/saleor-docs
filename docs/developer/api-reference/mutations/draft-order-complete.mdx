Completes creating an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
draftOrderComplete(
  id: ID!
): DraftOrderComplete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order that will be completed.

### Type

#### [`DraftOrderComplete`](../../../developer/api-reference/objects/draft-order-complete)

Completes creating an order. Requires one of the following permissions: MANAGE_ORDERS.
