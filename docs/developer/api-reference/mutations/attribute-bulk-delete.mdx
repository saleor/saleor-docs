Deletes attributes. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
attributeBulkDelete(
  ids: [ID!]!
): AttributeBulkDelete
```


### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of attribute IDs to delete.


### Type

#### [`AttributeBulkDelete`](../../../developer/api-reference/objects/attribute-bulk-delete) 

Deletes attributes. Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.
