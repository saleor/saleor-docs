Refresh user's access by custom plugin.

```graphql
externalRefresh(
  input: JSONString!
  pluginId: String!
): ExternalRefresh
```

### Arguments

#### `input` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

The data required by plugin to proceed the refresh process.

#### `pluginId` ([`String`](../../../developer/api-reference/scalars/string))

The ID of the authentication plugin.

### Type

#### [`ExternalRefresh`](../../../developer/api-reference/objects/external-refresh)

Refresh user's access by custom plugin.
