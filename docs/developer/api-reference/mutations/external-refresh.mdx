Refresh user's access by custom plugin.

```graphql
externalRefresh(
  input: JSONString!
  pluginId: String!
): ExternalRefresh

```

### Arguments

#### `input` ([`JSONString!`](/docs/developer/api-reference/scalars/json-string))

The data required by plugin to proceed the refresh process.

#### `pluginId` ([`String!`](/docs/developer/api-reference/scalars/string))

The ID of the authentication plugin.

### Type

#### [`ExternalRefresh`](/docs/developer/api-reference/objects/external-refresh)

Refresh user's access by custom plugin.
