title: menuBulkDelete
---

Deletes menus. Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuBulkDelete(
  ids: [ID!]!
): MenuBulkDelete
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of menu IDs to delete.

### Type

#### [`MenuBulkDelete`](../../../developer/api-reference/objects/menu-bulk-delete)

Deletes menus. Requires one of the following permissions: MANAGE_MENUS.
