Updates value of an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeValueUpdate(
  id: ID!
  input: AttributeValueUpdateInput!
): AttributeValueUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an AttributeValue to update.

#### `input` ([`AttributeValueUpdateInput`](../../../developer/api-reference/inputs/attribute-value-update-input))

Fields required to update an AttributeValue.

### Type

#### [`AttributeValueUpdate`](../../../developer/api-reference/objects/attribute-value-update)

Updates value of an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
