Updates value of an attribute.

```graphql
attributeValueUpdate(
  id: ID!
  input: AttributeValueCreateInput!
): AttributeValueUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an AttributeValue to update.

#### `input` ([`AttributeValueCreateInput!`](/docs/developer/api-reference/inputs/attribute-value-create-input))

Fields required to update an AttributeValue.

### Type

#### [`AttributeValueUpdate`](/docs/developer/api-reference/objects/attribute-value-update)

Updates value of an attribute.
