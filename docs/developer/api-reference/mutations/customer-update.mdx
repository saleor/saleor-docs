Updates an existing customer.

```graphql
customerUpdate(
  id: ID!
  input: CustomerInput!
): CustomerUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a customer to update.

#### `input` ([`CustomerInput!`](/docs/developer/api-reference/inputs/customer-input))

Fields required to update a customer.

### Type

#### [`CustomerUpdate`](/docs/developer/api-reference/objects/customer-update)

Updates an existing customer.
