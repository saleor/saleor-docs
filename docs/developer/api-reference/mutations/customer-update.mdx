Updates an existing customer.

Requires one of the following permissions: MANAGE_USERS.

```graphql
customerUpdate(
  id: ID!
  input: CustomerInput!
): CustomerUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a customer to update.

#### `input` ([`CustomerInput`](../../../developer/api-reference/inputs/customer-input))

Fields required to update a customer.

### Type

#### [`CustomerUpdate`](../../../developer/api-reference/objects/customer-update)

Updates an existing customer.

Requires one of the following permissions: MANAGE_USERS.
