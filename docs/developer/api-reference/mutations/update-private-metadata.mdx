Updates private metadata of an object.

```graphql
updatePrivateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdatePrivateMetadata

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an object to update.

#### `input` ([`[MetadataInput!]!`](/docs/developer/api-reference/inputs/metadata-input))

Fields required to update the object's metadata.

### Type

#### [`UpdatePrivateMetadata`](/docs/developer/api-reference/objects/update-private-metadata)

Updates private metadata of an object.
