Updates private metadata of an object. To use it, you need to be an authenticated staff user or an app and have access to the modified object.

```graphql
updatePrivateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdatePrivateMetadata
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID or token (for Order and Checkout) of an object to update.

#### [`input`](#) ([`[MetadataInput!]!`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the object's metadata.

### Type

#### [`UpdatePrivateMetadata`](../../../developer/api-reference/objects/update-private-metadata)

Updates private metadata of an object. To use it, you need to be an authenticated staff user or an app and have access to the modified object.
