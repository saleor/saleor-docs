Adds a gift card or a voucher to a checkout.

```graphql
checkoutAddPromoCode(
  checkoutId: ID = 0
  promoCode: String!
  token: UUID
): CheckoutAddPromoCode

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Checkout ID. DEPRECATED: Will be removed in Saleor 4.0. Use token instead.

#### `promoCode` ([`String!`](/developer/api-reference/scalars/string.mdx))

Gift card code or voucher code.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutAddPromoCode`](/developer/api-reference/objects/checkout-add-promo-code.mdx)

Adds a gift card or a voucher to a checkout.
