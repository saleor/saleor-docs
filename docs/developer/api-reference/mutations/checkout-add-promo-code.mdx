Adds a gift card or a voucher to a checkout.

```graphql
checkoutAddPromoCode(
  checkoutId: ID
  id: ID
  promoCode: String!
  token: UUID
): CheckoutAddPromoCode
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The checkout's ID.

Added in Saleor 3.4.

#### `promoCode` ([`String`](../../../developer/api-reference/scalars/string))

Gift card code or voucher code.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

### Type

#### [`CheckoutAddPromoCode`](../../../developer/api-reference/objects/checkout-add-promo-code)

Adds a gift card or a voucher to a checkout.
