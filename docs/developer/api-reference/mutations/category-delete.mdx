title: categoryDelete
---

Deletes a category. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
categoryDelete(
  id: ID!
): CategoryDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a category to delete.

### Type

#### [`CategoryDelete`](../../../developer/api-reference/objects/category-delete)

Deletes a category. Requires one of the following permissions: MANAGE_PRODUCTS.
