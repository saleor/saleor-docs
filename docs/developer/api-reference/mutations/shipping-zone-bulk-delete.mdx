Deletes shipping zones. Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
shippingZoneBulkDelete(
  ids: [ID!]!
): ShippingZoneBulkDelete

```

### Arguments

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of shipping zone IDs to delete.

### Type

#### [`ShippingZoneBulkDelete`](../../../developer/api-reference/objects/shipping-zone-bulk-delete)

Deletes shipping zones. Requires one of the following permissions: MANAGE_SHIPPING.
