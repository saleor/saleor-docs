Creates a new product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productTypeCreate(
  input: ProductTypeInput!
): ProductTypeCreate
```


### Arguments

#### `input` ([`ProductTypeInput`](../../../developer/api-reference/inputs/product-type-input))

Fields required to create a product type.


### Type

#### [`ProductTypeCreate`](../../../developer/api-reference/objects/product-type-create) 

Creates a new product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
