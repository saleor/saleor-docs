Creates a new shipping price.

Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
shippingPriceCreate(
  input: ShippingPriceInput!
): ShippingPriceCreate
```

### Arguments

#### `input` ([`ShippingPriceInput`](../../../developer/api-reference/inputs/shipping-price-input))

Fields required to create a shipping price.

### Type

#### [`ShippingPriceCreate`](../../../developer/api-reference/objects/shipping-price-create)

Creates a new shipping price.

Requires one of the following permissions: MANAGE_SHIPPING.
