Delete permission group. Requires one of the following permissions: MANAGE_STAFF.

```graphql
permissionGroupDelete(
  id: ID!
): PermissionGroupDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the group to delete.

### Type

#### [`PermissionGroupDelete`](../../../developer/api-reference/objects/permission-group-delete)

Delete permission group. Requires one of the following permissions: MANAGE_STAFF.
