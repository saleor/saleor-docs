Manage collection's availability in channels.

```graphql
collectionChannelListingUpdate(
  id: ID!
  input: CollectionChannelListingUpdateInput!

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a collection to update.

#### `input` ([`CollectionChannelListingUpdateInput!`](/docs/developer/api-reference/inputs/collection-channel-listing-update-input))

Fields required to create or update collection channel listings.

### Type

#### [`CollectionChannelListingUpdate`](/docs/developer/api-reference/objects/collection-channel-listing-update)

Manage collection's availability in channels.
