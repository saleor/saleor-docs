Sets a default address for the authenticated user. Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
accountSetDefaultAddress(
  id: ID!
  type: AddressTypeEnum!
): AccountSetDefaultAddress
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the address to set as default.


#### `type` ([`AddressTypeEnum`](../../../developer/api-reference/enums/address-type-enum))

The type of address.


### Type

#### [`AccountSetDefaultAddress`](../../../developer/api-reference/objects/account-set-default-address) 

Sets a default address for the authenticated user. Requires one of the following permissions: AUTHENTICATED_USER.
