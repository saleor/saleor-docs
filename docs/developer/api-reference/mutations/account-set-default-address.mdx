Sets a default address for the authenticated user.

```graphql
accountSetDefaultAddress(
  id: ID!
  type: AddressTypeEnum!
): AccountSetDefaultAddress

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the address to set as default.

#### `type` ([`AddressTypeEnum!`](/docs/developer/api-reference/enums/address-type-enum))

The type of address.

### Type

#### [`AccountSetDefaultAddress`](/docs/developer/api-reference/objects/account-set-default-address)

Sets a default address for the authenticated user.
