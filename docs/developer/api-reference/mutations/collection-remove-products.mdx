Remove products from a collection.

```graphql
collectionRemoveProducts(
  collectionId: ID!
  products: [ID]!
): CollectionRemoveProducts

```

### Arguments

#### `collectionId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a collection.

#### `products` ([`[ID]!`](/docs/developer/api-reference/scalars/id))

List of product IDs.

### Type

#### [`CollectionRemoveProducts`](/docs/developer/api-reference/objects/collection-remove-products)

Remove products from a collection.
