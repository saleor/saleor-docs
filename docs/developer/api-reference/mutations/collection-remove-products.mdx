Remove products from a collection. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionRemoveProducts(
  collectionId: ID!
  products: [ID!]!
): CollectionRemoveProducts
```

### Arguments

#### `collectionId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a collection.

#### `products` ([`ID`](../../../developer/api-reference/scalars/id))

List of product IDs.

### Type

#### [`CollectionRemoveProducts`](../../../developer/api-reference/objects/collection-remove-products)

Remove products from a collection. Requires one of the following permissions: MANAGE_PRODUCTS.
