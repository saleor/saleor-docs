Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.

```graphql
productVariantSetDefault(
  productId: ID!
  variantId: ID!
): ProductVariantSetDefault

```

### Arguments

#### `productId` ([`ID!`](/docs/developer/api-reference/scalars/id))

Id of a product that will have the default variant set.

#### `variantId` ([`ID!`](/docs/developer/api-reference/scalars/id))

Id of a variant that will be set as default.

### Type

#### [`ProductVariantSetDefault`](/docs/developer/api-reference/objects/product-variant-set-default)

Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.
