---

Obtain external access tokens for user by custom plugin.

```graphql
externalObtainAccessTokens(
  input: JSONString!
  pluginId: String!
): ExternalObtainAccessTokens

```

### Arguments

#### `input` ([`JSONString!`](/docs/developer/api-reference/scalars/json-string))

The data required by plugin to create authentication data.

#### `pluginId` ([`String!`](/docs/developer/api-reference/scalars/string))

The ID of the authentication plugin.

### Type

#### [`ExternalObtainAccessTokens`](/docs/developer/api-reference/objects/external-obtain-access-tokens)

Obtain external access tokens for user by custom plugin.
