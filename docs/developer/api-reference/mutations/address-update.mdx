Updates an address.

```graphql
addressUpdate(
  id: ID!
  input: AddressInput!
): AddressUpdate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the address to update.

#### `input` ([`AddressInput!`](../../../developer/api-reference/inputs/address-input))

Fields required to update the address.

### Type

#### [`AddressUpdate`](../../../developer/api-reference/objects/address-update)

Updates an address.
