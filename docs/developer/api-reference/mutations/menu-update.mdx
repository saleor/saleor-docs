Updates a menu. Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuUpdate(
  id: ID!
  input: MenuInput!
): MenuUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a menu to update.

#### `input` ([`MenuInput`](../../../developer/api-reference/inputs/menu-input))

Fields required to update a menu.

### Type

#### [`MenuUpdate`](../../../developer/api-reference/objects/menu-update)

Updates a menu. Requires one of the following permissions: MANAGE_MENUS.
