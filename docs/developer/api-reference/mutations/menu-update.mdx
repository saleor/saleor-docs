Updates a menu.

```graphql
menuUpdate(
  id: ID!
  input: MenuInput!
): MenuUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a menu to update.

#### `input` ([`MenuInput!`](/docs/developer/api-reference/inputs/menu-input))

Fields required to update a menu.

### Type

#### [`MenuUpdate`](/docs/developer/api-reference/objects/menu-update)

Updates a menu.
