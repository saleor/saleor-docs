Updates the account of the logged-in user. Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
accountUpdate(
  input: AccountInput!
): AccountUpdate

```

### Arguments

#### `input` ([`AccountInput!`](../../../developer/api-reference/inputs/account-input))

Fields required to update the account of the logged-in user.

### Type

#### [`AccountUpdate`](../../../developer/api-reference/objects/account-update)

Updates the account of the logged-in user. Requires one of the following permissions: AUTHENTICATED_USER.
