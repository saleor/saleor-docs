Adds products, categories, collections to a voucher.

```graphql
saleCataloguesAdd(
  id: ID!
  input: CatalogueInput!
): SaleAddCatalogues

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a sale.

#### `input` ([`CatalogueInput!`](../../../developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of sale.

### Type

#### [`SaleAddCatalogues`](../../../developer/api-reference/objects/sale-add-catalogues)

Adds products, categories, collections to a voucher.
