title: productBulkDelete
---

Deletes products. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productBulkDelete(
  ids: [ID!]!
): ProductBulkDelete
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of product IDs to delete.

### Type

#### [`ProductBulkDelete`](../../../developer/api-reference/objects/product-bulk-delete)

Deletes products. Requires one of the following permissions: MANAGE_PRODUCTS.
