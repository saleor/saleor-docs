New in Saleor 3.1. Delete gift card.

```graphql
giftCardDelete(
  id: ID!
): GiftCardDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the gift card to delete.

### Type

#### [`GiftCardDelete`](/developer/api-reference/objects/gift-card-delete.mdx)

New in Saleor 3.1. Delete gift card.
