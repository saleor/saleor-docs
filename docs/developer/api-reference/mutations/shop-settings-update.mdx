Updates shop settings. Requires one of the following permissions: MANAGE_SETTINGS.

```graphql
shopSettingsUpdate(
  input: ShopSettingsInput!
): ShopSettingsUpdate

```

### Arguments

#### `input` ([`ShopSettingsInput!`](../../../developer/api-reference/inputs/shop-settings-input))

Fields required to update shop settings.

### Type

#### [`ShopSettingsUpdate`](../../../developer/api-reference/objects/shop-settings-update)

Updates shop settings. Requires one of the following permissions: MANAGE_SETTINGS.
