Verify external authentication data by plugin.

```graphql
externalVerify(
  input: JSONString!
  pluginId: String!
): ExternalVerify
```


### Arguments

#### `input` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

The data required by plugin to proceed the verification.


#### `pluginId` ([`String`](../../../developer/api-reference/scalars/string))

The ID of the authentication plugin.


### Type

#### [`ExternalVerify`](../../../developer/api-reference/objects/external-verify) 

Verify external authentication data by plugin.
