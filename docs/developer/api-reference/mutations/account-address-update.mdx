Updates an address of the logged-in user. Requires one of the following permissions: MANAGE_USERS, IS_OWNER.

```graphql
accountAddressUpdate(
  id: ID!
  input: AddressInput!
): AccountAddressUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the address to update.


#### `input` ([`AddressInput`](../../../developer/api-reference/inputs/address-input))

Fields required to update the address.


### Type

#### [`AccountAddressUpdate`](../../../developer/api-reference/objects/account-address-update) 

Updates an address of the logged-in user. Requires one of the following permissions: MANAGE_USERS, IS_OWNER.
