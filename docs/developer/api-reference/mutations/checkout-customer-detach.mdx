Removes the user assigned as the owner of the checkout. Requires one of the following permissions: AUTHENTICATED_APP, AUTHENTICATED_USER.

```graphql
checkoutCustomerDetach(
  checkoutId: ID
  token: UUID
): CheckoutCustomerDetach
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutCustomerDetach`](../../../developer/api-reference/objects/checkout-customer-detach)

Removes the user assigned as the owner of the checkout. Requires one of the following permissions: AUTHENTICATED_APP, AUTHENTICATED_USER.
