Removes the user assigned as the owner of the checkout.

```graphql
checkoutCustomerDetach(
  checkoutId: ID = 0
  token: UUID
): CheckoutCustomerDetach

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Checkout ID.DEPRECATED: Will be removed in Saleor 4.0. Use token instead.

#### `token` ([`UUID`](/developer/api-reference/scalars/uuid.mdx))

Checkout token.

### Type

#### [`CheckoutCustomerDetach`](/developer/api-reference/objects/checkout-customer-detach.mdx)

Removes the user assigned as the owner of the checkout.
