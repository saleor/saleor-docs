Updates a menu item.

```graphql
menuItemUpdate(
  id: ID!
  input: MenuItemInput!
): MenuItemUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a menu item to update.

#### `input` ([`MenuItemInput!`](/docs/developer/api-reference/inputs/menu-item-input))

Fields required to update a menu item. Only one of `url`, `category`, `page`, `collection` is allowed per item.

### Type

#### [`MenuItemUpdate`](/docs/developer/api-reference/objects/menu-item-update)

Updates a menu item.
