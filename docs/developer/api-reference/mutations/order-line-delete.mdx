Deletes an order line from an order.

orderLineDelete(

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the order line to delete.

### Type

#### [`OrderLineDelete`](../../../developer/api-reference/objects/order-line-delete)

Deletes an order line from an order.
