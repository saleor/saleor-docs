---

Return products. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderFulfillmentReturnProducts(
  input: OrderReturnProductsInput!
  order: ID!
): FulfillmentReturnProducts

```

### Arguments

#### `input` ([`OrderReturnProductsInput!`](../../../developer/api-reference/inputs/order-return-products-input))

Fields required to return products.

#### `order` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the order to be returned.

### Type

#### [`FulfillmentReturnProducts`](../../../developer/api-reference/objects/fulfillment-return-products)

Return products. Requires one of the following permissions: MANAGE_ORDERS.
