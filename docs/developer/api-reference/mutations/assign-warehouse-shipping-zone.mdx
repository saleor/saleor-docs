---

Add shipping zone to given warehouse.

```graphql
assignWarehouseShippingZone(
  id: ID!
  shippingZoneIds: [ID!]!
): WarehouseShippingZoneAssign

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a warehouse to update.

#### `shippingZoneIds` ([`[ID!]!`](/docs/developer/api-reference/scalars/id))

List of shipping zone IDs.

### Type

#### [`WarehouseShippingZoneAssign`](/docs/developer/api-reference/objects/warehouse-shipping-zone-assign)

Add shipping zone to given warehouse.
