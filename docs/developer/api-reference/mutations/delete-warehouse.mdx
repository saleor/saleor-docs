---

Deletes selected warehouse. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
deleteWarehouse(
  id: ID!
): WarehouseDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a warehouse to delete.

### Type

#### [`WarehouseDelete`](../../../developer/api-reference/objects/warehouse-delete)

Deletes selected warehouse. Requires one of the following permissions: MANAGE_PRODUCTS.
