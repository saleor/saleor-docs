Create JWT token.

```graphql
tokenCreate(
  email: String!

```

### Arguments

#### `email` ([`String!`](/docs/developer/api-reference/scalars/string))

Email of a user.

#### `password` ([`String!`](/docs/developer/api-reference/scalars/string))

Password of a user.

### Type

#### [`CreateToken`](/docs/developer/api-reference/objects/create-token)

Create JWT token.
