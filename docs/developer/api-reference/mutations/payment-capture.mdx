Captures the authorized payment amount.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
paymentCapture(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentCapture
```

### Arguments

#### `amount` ([`PositiveDecimal`](../../../developer/api-reference/scalars/positive-decimal))

Transaction amount.

#### `paymentId` ([`ID`](../../../developer/api-reference/scalars/id))

Payment ID.

### Type

#### [`PaymentCapture`](../../../developer/api-reference/objects/payment-capture)

Captures the authorized payment amount.

Requires one of the following permissions: MANAGE_ORDERS.
