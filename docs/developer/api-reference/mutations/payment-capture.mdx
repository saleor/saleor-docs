Captures the authorized payment amount.

```graphql
paymentCapture(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentCapture

```

### Arguments

#### `amount` ([`PositiveDecimal`](/docs/developer/api-reference/scalars/positive-decimal))

Transaction amount.

#### `paymentId` ([`ID!`](/docs/developer/api-reference/scalars/id))

Payment ID.

### Type

#### [`PaymentCapture`](/docs/developer/api-reference/objects/payment-capture)

Captures the authorized payment amount.
