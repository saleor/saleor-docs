Updates a draft order.

```graphql
draftOrderUpdate(
  id: ID!
  input: DraftOrderInput!
): DraftOrderUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a draft order to update.

#### `input` ([`DraftOrderInput!`](/developer/api-reference/inputs/draft-order-input.mdx))

Fields required to update an order.

### Type

#### [`DraftOrderUpdate`](/developer/api-reference/objects/draft-order-update.mdx)

Updates a draft order.
