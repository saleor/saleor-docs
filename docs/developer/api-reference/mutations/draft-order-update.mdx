Updates a draft order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
draftOrderUpdate(
  id: ID!
  input: DraftOrderInput!
): DraftOrderUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a draft order to update.


#### `input` ([`DraftOrderInput`](../../../developer/api-reference/inputs/draft-order-input))

Fields required to update an order.


### Type

#### [`DraftOrderUpdate`](../../../developer/api-reference/objects/draft-order-update) 

Updates a draft order. Requires one of the following permissions: MANAGE_ORDERS.
