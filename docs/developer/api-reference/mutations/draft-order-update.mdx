Updates a draft order.

```graphql
draftOrderUpdate(
  id: ID!
  input: DraftOrderInput!
): DraftOrderUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a draft order to update.

#### `input` ([`DraftOrderInput!`](/docs/developer/api-reference/inputs/draft-order-input))

Fields required to update an order.

### Type

#### [`DraftOrderUpdate`](/docs/developer/api-reference/objects/draft-order-update)

Updates a draft order.
