Reorder the attributes of a page type.

```graphql
pageTypeReorderAttributes(
  moves: [ReorderInput!]!
  pageTypeId: ID!
): PageTypeReorderAttributes

```

### Arguments

#### `moves` ([`[ReorderInput!]!`](/docs/developer/api-reference/inputs/reorder-input))

The list of attribute reordering operations.

#### `pageTypeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a page type.

### Type

#### [`PageTypeReorderAttributes`](/docs/developer/api-reference/objects/page-type-reorder-attributes)

Reorder the attributes of a page type.
