Updates checkout line in the existing checkout.

```graphql
checkoutLinesUpdate(
  checkoutId: ID
  lines: [CheckoutLineInput]!
  token: UUID
): CheckoutLinesUpdate

```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `lines` ([`[CheckoutLineInput]!`](../../../developer/api-reference/inputs/checkout-line-input))

A list of checkout lines, each containing information about an item in the checkout.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutLinesUpdate`](../../../developer/api-reference/objects/checkout-lines-update)

Updates checkout line in the existing checkout.
