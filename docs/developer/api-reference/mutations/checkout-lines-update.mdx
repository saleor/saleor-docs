Updates checkout line in the existing checkout.

```graphql
checkoutLinesUpdate(
  checkoutId: ID
  id: ID
  lines: [CheckoutLineUpdateInput!]!
  token: UUID
): CheckoutLinesUpdate
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The checkout's ID.

Added in Saleor 3.4.

#### `lines` ([`CheckoutLineUpdateInput`](../../../developer/api-reference/inputs/checkout-line-update-input))

A list of checkout lines, each containing information about an item in the checkout.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

### Type

#### [`CheckoutLinesUpdate`](../../../developer/api-reference/objects/checkout-lines-update)

Updates checkout line in the existing checkout.
