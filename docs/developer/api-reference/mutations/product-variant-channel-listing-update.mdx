Manage product variant prices in channels.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantChannelListingUpdate(
  id: ID!
  input: [ProductVariantChannelListingAddInput!]!
): ProductVariantChannelListingUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant to update.

#### `input` ([`ProductVariantChannelListingAddInput`](../../../developer/api-reference/inputs/product-variant-channel-listing-add-input))

List of fields required to create or upgrade product variant channel listings.

### Type

#### [`ProductVariantChannelListingUpdate`](../../../developer/api-reference/objects/product-variant-channel-listing-update)

Manage product variant prices in channels.

Requires one of the following permissions: MANAGE_PRODUCTS.
