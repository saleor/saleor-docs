Deletes staff users. Requires one of the following permissions: MANAGE_STAFF.

```graphql
staffBulkDelete(
  ids: [ID!]!
): StaffBulkDelete

```

### Arguments

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of user IDs to delete.

### Type

#### [`StaffBulkDelete`](../../../developer/api-reference/objects/staff-bulk-delete)

Deletes staff users. Requires one of the following permissions: MANAGE_STAFF.
