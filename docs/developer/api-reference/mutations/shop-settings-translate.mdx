Creates/updates translations for shop settings.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
shopSettingsTranslate(
  input: ShopSettingsTranslationInput!
  languageCode: LanguageCodeEnum!
): ShopSettingsTranslate
```

### Arguments

#### [`input`](#) ([`ShopSettingsTranslationInput!`](../../../developer/api-reference/inputs/shop-settings-translation-input))

Fields required to update shop settings translations.

#### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`ShopSettingsTranslate`](../../../developer/api-reference/objects/shop-settings-translate)

Creates/updates translations for shop settings.

Requires one of the following permissions: MANAGE_TRANSLATIONS.
