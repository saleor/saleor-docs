Updates a fulfillment for an order.

```graphql
orderFulfillmentUpdateTracking(
  id: ID!
  input: FulfillmentUpdateTrackingInput!
): FulfillmentUpdateTracking

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a fulfillment to update.

#### `input` ([`FulfillmentUpdateTrackingInput!`](/docs/developer/api-reference/inputs/fulfillment-update-tracking-input))

Fields required to update a fulfillment.

### Type

#### [`FulfillmentUpdateTracking`](/docs/developer/api-reference/objects/fulfillment-update-tracking)

Updates a fulfillment for an order.
