Updates a fulfillment for an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderFulfillmentUpdateTracking(
  id: ID!
  input: FulfillmentUpdateTrackingInput!
): FulfillmentUpdateTracking

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a fulfillment to update.

#### `input` ([`FulfillmentUpdateTrackingInput!`](../../../developer/api-reference/inputs/fulfillment-update-tracking-input))

Fields required to update a fulfillment.

### Type

#### [`FulfillmentUpdateTracking`](../../../developer/api-reference/objects/fulfillment-update-tracking)

Updates a fulfillment for an order. Requires one of the following permissions: MANAGE_ORDERS.
