---


Deletes categories. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
categoryBulkDelete(
  ids: [ID!]!
): CategoryBulkDelete
```


### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of category IDs to delete.


### Type

#### [`CategoryBulkDelete`](../../../developer/api-reference/objects/category-bulk-delete) 

Deletes categories. Requires one of the following permissions: MANAGE_PRODUCTS.
