---

Cancels existing fulfillment and optionally restocks items.

```graphql
orderFulfillmentCancel(
  id: ID!
  input: FulfillmentCancelInput!
): FulfillmentCancel

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an fulfillment to cancel.

#### `input` ([`FulfillmentCancelInput!`](../../../developer/api-reference/inputs/fulfillment-cancel-input))

Fields required to cancel an fulfillment.

### Type

#### [`FulfillmentCancel`](../../../developer/api-reference/objects/fulfillment-cancel)

Cancels existing fulfillment and optionally restocks items.
