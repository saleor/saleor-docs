---

Cancels existing fulfillment and optionally restocks items.

```graphql
orderFulfillmentCancel(
  id: ID!
  input: FulfillmentCancelInput
): FulfillmentCancel

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a fulfillment to cancel.

#### `input` ([`FulfillmentCancelInput`](/developer/api-reference/inputs/fulfillment-cancel-input.mdx))

Fields required to cancel a fulfillment.

### Type

#### [`FulfillmentCancel`](/developer/api-reference/objects/fulfillment-cancel.mdx)

Cancels existing fulfillment and optionally restocks items.
