---

Cancels existing fulfillment and optionally restocks items.

```graphql
orderFulfillmentCancel(
  id: ID!
  input: FulfillmentCancelInput
): FulfillmentCancel

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a fulfillment to cancel.

#### `input` ([`FulfillmentCancelInput`](/docs/developer/api-reference/inputs/fulfillment-cancel-input))

Fields required to cancel a fulfillment.

### Type

#### [`FulfillmentCancel`](/docs/developer/api-reference/objects/fulfillment-cancel)

Cancels existing fulfillment and optionally restocks items.
