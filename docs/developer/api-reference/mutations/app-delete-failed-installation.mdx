Delete failed installation. Requires one of the following permissions: MANAGE_APPS.

```graphql
appDeleteFailedInstallation(
  id: ID!
): AppDeleteFailedInstallation
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of failed installation to delete.


### Type

#### [`AppDeleteFailedInstallation`](../../../developer/api-reference/objects/app-delete-failed-installation) 

Delete failed installation. Requires one of the following permissions: MANAGE_APPS.
