Creates a new category.

```graphql
categoryCreate(
  input: CategoryInput!
  parent: ID
): CategoryCreate

```

### Arguments

#### `input` ([`CategoryInput!`](../../../developer/api-reference/inputs/category-input))

Fields required to create a category.

#### `parent` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the parent category. If empty, category will be top level category.

### Type

#### [`CategoryCreate`](../../../developer/api-reference/objects/category-create)

Creates a new category.
