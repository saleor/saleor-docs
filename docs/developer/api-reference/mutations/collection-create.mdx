Creates a new collection.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionCreate(
  input: CollectionCreateInput!
): CollectionCreate
```

### Arguments

#### `input` ([`CollectionCreateInput`](../../../developer/api-reference/inputs/collection-create-input))

Fields required to create a collection.

### Type

#### [`CollectionCreate`](../../../developer/api-reference/objects/collection-create)

Creates a new collection.

Requires one of the following permissions: MANAGE_PRODUCTS.
