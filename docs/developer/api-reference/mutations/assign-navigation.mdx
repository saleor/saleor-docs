Assigns storefront's navigation menus. Requires one of the following permissions: MANAGE_MENUS, MANAGE_SETTINGS.

```graphql
assignNavigation(
  menu: ID
  navigationType: NavigationType!
): AssignNavigation

```

### Arguments

#### `menu` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the menu.

#### `navigationType` ([`NavigationType!`](../../../developer/api-reference/enums/navigation-type))

Type of the navigation bar to assign the menu to.

### Type

#### [`AssignNavigation`](../../../developer/api-reference/objects/assign-navigation)

Assigns storefront's navigation menus. Requires one of the following permissions: MANAGE_MENUS, MANAGE_SETTINGS.
