Assigns storefront's navigation menus.

```graphql
assignNavigation(
  menu: ID
  navigationType: NavigationType!
): AssignNavigation

```

### Arguments

#### `menu` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the menu.

#### `navigationType` ([`NavigationType!`](/docs/developer/api-reference/enums/navigation-type))

Type of the navigation bar to assign the menu to.

### Type

#### [`AssignNavigation`](/docs/developer/api-reference/objects/assign-navigation)

Assigns storefront's navigation menus.
