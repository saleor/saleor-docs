Remove discount from the order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderDiscountDelete(
  discountId: ID!
): OrderDiscountDelete
```

### Arguments

#### `discountId` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a discount to remove.

### Type

#### [`OrderDiscountDelete`](../../../developer/api-reference/objects/order-discount-delete)

Remove discount from the order. Requires one of the following permissions: MANAGE_ORDERS.
