Capture an order.

```graphql
orderCapture(
  amount: PositiveDecimal!
  id: ID!
): OrderCapture

```

### Arguments

#### `amount` ([`PositiveDecimal!`](/docs/developer/api-reference/scalars/positive-decimal))

Amount of money to capture.

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order to capture.

### Type

#### [`OrderCapture`](/docs/developer/api-reference/objects/order-capture)

Capture an order.
