Capture an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderCapture(
  amount: PositiveDecimal!
  id: ID!
): OrderCapture
```


### Arguments

#### `amount` ([`PositiveDecimal`](../../../developer/api-reference/scalars/positive-decimal))

Amount of money to capture.


#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the order to capture.


### Type

#### [`OrderCapture`](../../../developer/api-reference/objects/order-capture) 

Capture an order. Requires one of the following permissions: MANAGE_ORDERS.
