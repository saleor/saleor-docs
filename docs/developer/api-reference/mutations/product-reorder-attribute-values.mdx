Reorder product attribute values.

```graphql
productReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  productId: ID!
): ProductReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/docs/developer/api-reference/inputs/reorder-input))

The list of reordering operations for given attribute values.

#### `productId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product.

### Type

#### [`ProductReorderAttributeValues`](/docs/developer/api-reference/objects/product-reorder-attribute-values)

Reorder product attribute values.
