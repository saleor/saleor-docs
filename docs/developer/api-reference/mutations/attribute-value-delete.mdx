Deletes a value of an attribute. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeValueDelete(
  id: ID!
): AttributeValueDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a value to delete.

### Type

#### [`AttributeValueDelete`](../../../developer/api-reference/objects/attribute-value-delete)

Deletes a value of an attribute. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
