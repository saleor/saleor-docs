Creates/updates translations for a product.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
productTranslate(
  id: ID!
  input: TranslationInput!
  languageCode: LanguageCodeEnum!
): ProductTranslate
```
