Creates/updates translations for a product.

```graphql
productTranslate(
  id: ID!
  input: TranslationInput!
  languageCode: LanguageCodeEnum!
): ProductTranslate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

Product ID or ProductTranslatableContent ID.

#### `input` ([`TranslationInput!`](../../../developer/api-reference/inputs/translation-input))

#### `languageCode` ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`ProductTranslate`](../../../developer/api-reference/objects/product-translate)

Creates/updates translations for a product.
