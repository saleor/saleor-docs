Activate gift cards.

Added in Saleor 3.1.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: MANAGE_GIFT_CARD.

```graphql
giftCardBulkActivate(
  ids: [ID!]!
): GiftCardBulkActivate
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of gift card IDs to activate.

### Type

#### [`GiftCardBulkActivate`](../../../developer/api-reference/objects/gift-card-bulk-activate)

Activate gift cards.

Added in Saleor 3.1.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: MANAGE_GIFT_CARD.
