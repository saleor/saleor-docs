Deletes a shipping price.

```graphql
shippingPriceDelete(
  id: ID!
): ShippingPriceDelete

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a shipping price to delete.

### Type

#### [`ShippingPriceDelete`](/docs/developer/api-reference/objects/shipping-price-delete)

Deletes a shipping price.
