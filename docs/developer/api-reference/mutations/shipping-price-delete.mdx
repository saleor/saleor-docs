Deletes a shipping price. Requires one of the following permissions: MANAGE_SHIPPING.

```graphql
shippingPriceDelete(
  id: ID!
): ShippingPriceDelete
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a shipping price to delete.


### Type

#### [`ShippingPriceDelete`](../../../developer/api-reference/objects/shipping-price-delete) 

Deletes a shipping price. Requires one of the following permissions: MANAGE_SHIPPING.
