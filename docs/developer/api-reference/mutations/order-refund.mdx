Refund an order. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderRefund(
  amount: PositiveDecimal!
  id: ID!
): OrderRefund

```

### Arguments

#### `amount` ([`PositiveDecimal!`](../../../developer/api-reference/scalars/positive-decimal))

Amount of money to refund.

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the order to refund.

### Type

#### [`OrderRefund`](../../../developer/api-reference/objects/order-refund)

Refund an order. Requires one of the following permissions: MANAGE_ORDERS.
