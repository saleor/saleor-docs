Refund an order.

```graphql
orderRefund(
  amount: PositiveDecimal!
  id: ID!
): OrderRefund

```

### Arguments

#### `amount` ([`PositiveDecimal!`](/docs/developer/api-reference/scalars/positive-decimal))

Amount of money to refund.

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the order to refund.

### Type

#### [`OrderRefund`](/docs/developer/api-reference/objects/order-refund)

Refund an order.
