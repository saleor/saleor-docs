Refund an order.

```graphql
orderRefund(
  amount: PositiveDecimal
  paymentsToRefund: [OrderPaymentToRefundInput!]
  id: ID!
): OrderRefund

```

### Arguments

#### `amount` ([`PositiveDecimal!`](../../../developer/api-reference/scalars/positive-decimal))

Amount of money to refund.

DEPRECATED: this field will be removed in Saleor 4.0. Use paymentsToRefund instead.

#### `paymentsToRefund` ([[`OrderPaymentToRefundInput`](../../../developer/api-reference/inputs/order-payment-to-refund-input)])

New in Saleor 3.0. A list of payments to be refunded.

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the order to refund.

### Type

#### [`OrderRefund`](../../../developer/api-reference/objects/order-refund)

Refund an order.
