Creates/updates translations for a product variant.

```graphql
productVariantTranslate(
  id: ID!
  input: NameTranslationInput!
  languageCode: LanguageCodeEnum!
): ProductVariantTranslate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ProductVariant ID or ProductVariantTranslatableContent ID.

#### `input` ([`NameTranslationInput!`](/docs/developer/api-reference/inputs/name-translation-input))

#### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`ProductVariantTranslate`](/docs/developer/api-reference/objects/product-variant-translate)

Creates/updates translations for a product variant.
