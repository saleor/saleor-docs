Cancels orders. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderBulkCancel(
  ids: [ID!]!
): OrderBulkCancel
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of orders IDs to cancel.

### Type

#### [`OrderBulkCancel`](../../../developer/api-reference/objects/order-bulk-cancel)

Cancels orders. Requires one of the following permissions: MANAGE_ORDERS.
