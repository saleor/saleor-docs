Remove a gift card or a voucher from a checkout.

```graphql
checkoutRemovePromoCode(
  checkoutId: ID
  promoCode: String
  promoCodeId: ID
  token: UUID
): CheckoutRemovePromoCode
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `promoCode` ([`String`](../../../developer/api-reference/scalars/string))

Gift card code or voucher code.

#### `promoCodeId` ([`ID`](../../../developer/api-reference/scalars/id))

Gift card or voucher ID.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutRemovePromoCode`](../../../developer/api-reference/objects/checkout-remove-promo-code)

Remove a gift card or a voucher from a checkout.
