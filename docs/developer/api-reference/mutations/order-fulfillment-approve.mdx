Approve existing fulfillment.

Added in Saleor 3.1.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderFulfillmentApprove(
  allowStockToBeExceeded: Boolean
  id: ID!
  notifyCustomer: Boolean!
): FulfillmentApprove
```

### Arguments

#### `allowStockToBeExceeded` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

True if stock could be exceeded.

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a fulfillment to approve.

#### `notifyCustomer` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

True if confirmation email should be send.

### Type

#### [`FulfillmentApprove`](../../../developer/api-reference/objects/fulfillment-approve)

Approve existing fulfillment.

Added in Saleor 3.1.

Requires one of the following permissions: MANAGE_ORDERS.
