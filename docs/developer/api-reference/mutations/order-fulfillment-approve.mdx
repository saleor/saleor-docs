---

New in Saleor 3.1. Approve existing fulfillment.

```graphql
orderFulfillmentApprove(
  id: ID!
  notifyCustomer: Boolean!
): FulfillmentApprove

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a fulfillment to approve.

#### `notifyCustomer` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

True if confirmation email should be send.

### Type

#### [`FulfillmentApprove`](/developer/api-reference/objects/fulfillment-approve.mdx)

New in Saleor 3.1. Approve existing fulfillment.
