---

New in Saleor 3.1. Approve existing fulfillment.

```graphql
orderFulfillmentApprove(
  id: ID!
  notifyCustomer: Boolean!
): FulfillmentApprove

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a fulfillment to approve.

#### `notifyCustomer` ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

True if confirmation email should be send.

### Type

#### [`FulfillmentApprove`](../../../developer/api-reference/objects/fulfillment-approve)

New in Saleor 3.1. Approve existing fulfillment.
