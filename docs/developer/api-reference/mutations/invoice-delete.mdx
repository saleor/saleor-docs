Deletes an invoice. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
invoiceDelete(
  id: ID!
): InvoiceDelete

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an invoice to delete.

### Type

#### [`InvoiceDelete`](../../../developer/api-reference/objects/invoice-delete)

Deletes an invoice. Requires one of the following permissions: MANAGE_ORDERS.
