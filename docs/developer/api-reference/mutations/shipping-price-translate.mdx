Creates/updates translations for a shipping method. Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
shippingPriceTranslate(
  id: ID!
  input: ShippingPriceTranslationInput!
  languageCode: LanguageCodeEnum!
): ShippingPriceTranslate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ShippingMethodType ID or ShippingMethodTranslatableContent ID.


#### `input` ([`ShippingPriceTranslationInput`](../../../developer/api-reference/inputs/shipping-price-translation-input))




#### `languageCode` ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.


### Type

#### [`ShippingPriceTranslate`](../../../developer/api-reference/objects/shipping-price-translate) 

Creates/updates translations for a shipping method. Requires one of the following permissions: MANAGE_TRANSLATIONS.
