Creates/updates translations for a shipping method.

```graphql
shippingPriceTranslate(
  id: ID!
  input: ShippingPriceTranslationInput!
  languageCode: LanguageCodeEnum!
): ShippingPriceTranslate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ShippingMethod ID or ShippingMethodTranslatableContent ID.

#### `input` ([`ShippingPriceTranslationInput!`](/docs/developer/api-reference/inputs/shipping-price-translation-input))

#### `languageCode` ([`LanguageCodeEnum!`](/docs/developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`ShippingPriceTranslate`](/docs/developer/api-reference/objects/shipping-price-translate)

Creates/updates translations for a shipping method.
