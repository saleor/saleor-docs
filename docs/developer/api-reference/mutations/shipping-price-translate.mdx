Creates/updates translations for a shipping method.

```graphql
shippingPriceTranslate(
  id: ID!
  input: ShippingPriceTranslationInput!
  languageCode: LanguageCodeEnum!
): ShippingPriceTranslate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ShippingMethod ID or ShippingMethodTranslatableContent ID.

#### `input` ([`ShippingPriceTranslationInput!`](/developer/api-reference/inputs/shipping-price-translation-input.mdx))

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`ShippingPriceTranslate`](/developer/api-reference/objects/shipping-price-translate.mdx)

Creates/updates translations for a shipping method.
