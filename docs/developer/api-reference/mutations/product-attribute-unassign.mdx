title: productAttributeUnassign
---

Un-assign attributes from a given product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productAttributeUnassign(
  attributeIds: [ID!]!
  productTypeId: ID!
): ProductAttributeUnassign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

The IDs of the attributes to unassign.

#### `productTypeId` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the product type from which the attributes should be unassigned.

### Type

#### [`ProductAttributeUnassign`](../../../developer/api-reference/objects/product-attribute-unassign)

Un-assign attributes from a given product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
