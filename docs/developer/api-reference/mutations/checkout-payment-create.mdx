Create a new payment for given checkout.

```graphql
checkoutPaymentCreate(
  checkoutId: ID
  id: ID
  input: PaymentInput!
  token: UUID
): CheckoutPaymentCreate
```

### Arguments

#### `checkoutId` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The checkout's ID.

Added in Saleor 3.4.

#### `input` ([`PaymentInput`](../../../developer/api-reference/inputs/payment-input))

Data required to create a new payment.

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

### Type

#### [`CheckoutPaymentCreate`](../../../developer/api-reference/objects/checkout-payment-create)

Create a new payment for given checkout.
