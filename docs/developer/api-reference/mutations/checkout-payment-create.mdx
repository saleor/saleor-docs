Create a new payment for given checkout.

```graphql
checkoutPaymentCreate(
  checkoutId: ID
  input: PaymentInput!
  token: UUID
): CheckoutPaymentCreate

```

### Arguments

#### `checkoutId` ([`ID`](/docs/developer/api-reference/scalars/id))

The ID of the checkout.

DEPRECATED: this field will be removed in Saleor 4.0. Use token instead.

#### `input` ([`PaymentInput!`](/docs/developer/api-reference/inputs/payment-input))

Data required to create a new payment.

#### `token` ([`UUID`](/docs/developer/api-reference/scalars/uuid))

Checkout token.

### Type

#### [`CheckoutPaymentCreate`](/docs/developer/api-reference/objects/checkout-payment-create)

Create a new payment for given checkout.
