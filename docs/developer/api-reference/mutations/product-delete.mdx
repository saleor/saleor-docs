title: productDelete
---

Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productDelete(
  id: ID!
): ProductDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product to delete.

### Type

#### [`ProductDelete`](../../../developer/api-reference/objects/product-delete)

Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.
