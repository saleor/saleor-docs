Updates a webhook subscription. Requires one of the following permissions: MANAGE_APPS.

```graphql
webhookUpdate(
  id: ID!
  input: WebhookUpdateInput!
): WebhookUpdate

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a webhook to update.

#### `input` ([`WebhookUpdateInput!`](../../../developer/api-reference/inputs/webhook-update-input))

Fields required to update a webhook.

### Type

#### [`WebhookUpdate`](../../../developer/api-reference/objects/webhook-update)

Updates a webhook subscription. Requires one of the following permissions: MANAGE_APPS.
