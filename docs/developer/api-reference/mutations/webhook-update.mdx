Updates a webhook subscription.

```graphql
webhookUpdate(
  id: ID!
  input: WebhookUpdateInput!
): WebhookUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a webhook to update.

#### `input` ([`WebhookUpdateInput!`](/docs/developer/api-reference/inputs/webhook-update-input))

Fields required to update a webhook.

### Type

#### [`WebhookUpdate`](/docs/developer/api-reference/objects/webhook-update)

Updates a webhook subscription.
