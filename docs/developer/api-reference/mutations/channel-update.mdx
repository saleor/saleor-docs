Update a channel.

```graphql
channelUpdate(
  id: ID!
  input: ChannelUpdateInput!
): ChannelUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a channel to update.

#### `input` ([`ChannelUpdateInput!`](/docs/developer/api-reference/inputs/channel-update-input))

Fields required to update a channel.

### Type

#### [`ChannelUpdate`](/docs/developer/api-reference/objects/channel-update)

Update a channel.
