Update a channel.

Requires one of the following permissions: MANAGE_CHANNELS.

```graphql
channelUpdate(
  id: ID!
  input: ChannelUpdateInput!
): ChannelUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a channel to update.

#### `input` ([`ChannelUpdateInput`](../../../developer/api-reference/inputs/channel-update-input))

Fields required to update a channel.

### Type

#### [`ChannelUpdate`](../../../developer/api-reference/objects/channel-update)

Update a channel.

Requires one of the following permissions: MANAGE_CHANNELS.
