Deletes a sale.

Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
saleDelete(
  id: ID!
): SaleDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a sale to delete.

### Type

#### [`SaleDelete`](../../../developer/api-reference/objects/sale-delete)

Deletes a sale.

Requires one of the following permissions: MANAGE_DISCOUNTS.
