Deletes order lines. Requires one of the following permissions: MANAGE_ORDERS.

```graphql
draftOrderLinesBulkDelete(
  ids: [ID!]!
): DraftOrderLinesBulkDelete
```

### Arguments

#### `ids` ([`ID`](../../../developer/api-reference/scalars/id))

List of order lines IDs to delete.

### Type

#### [`DraftOrderLinesBulkDelete`](../../../developer/api-reference/objects/draft-order-lines-bulk-delete)

Deletes order lines. Requires one of the following permissions: MANAGE_ORDERS.
