Removes products, categories, collections from a voucher.

```graphql
voucherCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): VoucherRemoveCatalogues

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a voucher.

#### `input` ([`CatalogueInput!`](/docs/developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of voucher.

### Type

#### [`VoucherRemoveCatalogues`](/docs/developer/api-reference/objects/voucher-remove-catalogues)

Removes products, categories, collections from a voucher.
