title: voucherCataloguesRemove

Removes products, categories, collections from a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
voucherCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): VoucherRemoveCatalogues
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a voucher.

#### `input` ([`CatalogueInput`](../../../developer/api-reference/inputs/catalogue-input))

Fields required to modify catalogue IDs of voucher.

### Type

#### [`VoucherRemoveCatalogues`](../../../developer/api-reference/objects/voucher-remove-catalogues)

Removes products, categories, collections from a voucher. Requires one of the following permissions: MANAGE_DISCOUNTS.
