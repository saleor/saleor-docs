Deletes checkout lines.

```graphql
checkoutLinesDelete(
  linesIds: [ID!]!
  token: UUID!
): CheckoutLinesDelete
```


### Arguments

#### `linesIds` ([`ID`](../../../developer/api-reference/scalars/id))

A list of checkout lines.


#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.


### Type

#### [`CheckoutLinesDelete`](../../../developer/api-reference/objects/checkout-lines-delete) 

Deletes checkout lines.
