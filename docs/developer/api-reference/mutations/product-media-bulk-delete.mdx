title: productMediaBulkDelete

Deletes product media. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productMediaBulkDelete(
  ids: [ID!]!
): ProductMediaBulkDelete

```

### Arguments

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of product media IDs to delete.

### Type

#### [`ProductMediaBulkDelete`](../../../developer/api-reference/objects/product-media-bulk-delete)

Deletes product media. Requires one of the following permissions: MANAGE_PRODUCTS.
