Install new app by using app manifest.

```graphql
appInstall(
  input: AppInstallInput!
): AppInstall

```

### Arguments

#### `input` ([`AppInstallInput!`](/docs/developer/api-reference/inputs/app-install-input))

Fields required to install a new app.

### Type

#### [`AppInstall`](/docs/developer/api-reference/objects/app-install)

Install new app by using app manifest.
