Exclude products from shipping price.

```graphql
shippingPriceExcludeProducts(
  id: ID!
  input: ShippingPriceExcludeProductsInput!
): ShippingPriceExcludeProducts

```

### Arguments

#### `id` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a shipping price.

#### `input` ([`ShippingPriceExcludeProductsInput!`](../../../developer/api-reference/inputs/shipping-price-exclude-products-input))

Exclude products input.

### Type

#### [`ShippingPriceExcludeProducts`](../../../developer/api-reference/objects/shipping-price-exclude-products)

Exclude products from shipping price.
