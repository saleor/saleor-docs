Sets the user's password from the token sent by email using the RequestPasswordReset mutation.

```graphql
setPassword(
  email: String!
  password: String!
  token: String!
): SetPassword
```


### Arguments

#### `email` ([`String`](../../../developer/api-reference/scalars/string))

Email of a user.


#### `password` ([`String`](../../../developer/api-reference/scalars/string))

Password of a user.


#### `token` ([`String`](../../../developer/api-reference/scalars/string))

A one-time token required to set the password.


### Type

#### [`SetPassword`](../../../developer/api-reference/objects/set-password) 

Sets the user's password from the token sent by email using the RequestPasswordReset mutation.
