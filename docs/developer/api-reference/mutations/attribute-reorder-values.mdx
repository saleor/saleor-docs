Reorder the values of an attribute.

```graphql
attributeReorderValues(
  attributeId: ID!
  moves: [ReorderInput]!
): AttributeReorderValues

```

### Arguments

#### `attributeId` ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](../../../developer/api-reference/inputs/reorder-input))

The list of reordering operations for given attribute values.

### Type

#### [`AttributeReorderValues`](../../../developer/api-reference/objects/attribute-reorder-values)

Reorder the values of an attribute.
