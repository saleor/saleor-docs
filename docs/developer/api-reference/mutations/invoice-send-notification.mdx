Send an invoice notification to the customer.

```graphql
invoiceSendNotification(
  id: ID!
): InvoiceSendNotification

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an invoice to be sent.

### Type

#### [`InvoiceSendNotification`](/developer/api-reference/objects/invoice-send-notification.mdx)

Send an invoice notification to the customer.
