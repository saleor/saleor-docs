Send an invoice notification to the customer.

```graphql
invoiceSendNotification(
  id: ID!
): InvoiceSendNotification

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an invoice to be sent.

### Type

#### [`InvoiceSendNotification`](/docs/developer/api-reference/objects/invoice-send-notification)

Send an invoice notification to the customer.
