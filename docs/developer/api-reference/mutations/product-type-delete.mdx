Deletes a product type.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productTypeDelete(
  id: ID!
): ProductTypeDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product type to delete.

### Type

#### [`ProductTypeDelete`](../../../developer/api-reference/objects/product-type-delete)

Deletes a product type.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
