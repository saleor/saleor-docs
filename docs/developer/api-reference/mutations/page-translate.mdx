Creates/updates translations for a page. Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
pageTranslate(
  id: ID!
  input: PageTranslationInput!
  languageCode: LanguageCodeEnum!
): PageTranslate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

Page ID or PageTranslatableContent ID.

#### `input` ([`PageTranslationInput`](../../../developer/api-reference/inputs/page-translation-input))

#### `languageCode` ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`PageTranslate`](../../../developer/api-reference/objects/page-translate)

Creates/updates translations for a page. Requires one of the following permissions: MANAGE_TRANSLATIONS.
