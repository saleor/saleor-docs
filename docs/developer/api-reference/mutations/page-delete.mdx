Deletes a page. Requires one of the following permissions: MANAGE_PAGES.

```graphql
pageDelete(
  id: ID!
): PageDelete
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a page to delete.

### Type

#### [`PageDelete`](../../../developer/api-reference/objects/page-delete)

Deletes a page. Requires one of the following permissions: MANAGE_PAGES.
