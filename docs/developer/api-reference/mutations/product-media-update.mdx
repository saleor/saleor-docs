Updates a product media. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productMediaUpdate(
  id: ID!
  input: ProductMediaUpdateInput!
): ProductMediaUpdate
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product media to update.

#### `input` ([`ProductMediaUpdateInput`](../../../developer/api-reference/inputs/product-media-update-input))

Fields required to update a product media.

### Type

#### [`ProductMediaUpdate`](../../../developer/api-reference/objects/product-media-update)

Updates a product media. Requires one of the following permissions: MANAGE_PRODUCTS.
