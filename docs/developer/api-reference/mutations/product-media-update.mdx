Updates a product media.

```graphql
productMediaUpdate(
  id: ID!
  input: ProductMediaUpdateInput!
): ProductMediaUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product media to update.

#### `input` ([`ProductMediaUpdateInput!`](/docs/developer/api-reference/inputs/product-media-update-input))

Fields required to update a product media.

### Type

#### [`ProductMediaUpdate`](/docs/developer/api-reference/objects/product-media-update)

Updates a product media.
