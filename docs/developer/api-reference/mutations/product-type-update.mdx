Updates an existing product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
productTypeUpdate(
  id: ID!
  input: ProductTypeInput!
): ProductTypeUpdate
```


### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product type to update.


#### `input` ([`ProductTypeInput`](../../../developer/api-reference/inputs/product-type-input))

Fields required to update a product type.


### Type

#### [`ProductTypeUpdate`](../../../developer/api-reference/objects/product-type-update) 

Updates an existing product type. Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
