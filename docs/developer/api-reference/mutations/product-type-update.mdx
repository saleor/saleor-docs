Updates an existing product type.

```graphql
productTypeUpdate(
  id: ID!
  input: ProductTypeInput!
): ProductTypeUpdate

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a product type to update.

#### `input` ([`ProductTypeInput!`](/docs/developer/api-reference/inputs/product-type-input))

Fields required to update a product type.

### Type

#### [`ProductTypeUpdate`](/docs/developer/api-reference/objects/product-type-update)

Updates an existing product type.
