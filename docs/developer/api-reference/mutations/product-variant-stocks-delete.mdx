Delete stocks from product variant.

```graphql
productVariantStocksDelete(

  warehouseIds: [ID!]

```

### Arguments

#### `variantId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of product variant for which stocks will be deleted.

#### `warehouseIds` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

### Type

#### [`ProductVariantStocksDelete`](/docs/developer/api-reference/objects/product-variant-stocks-delete)

Delete stocks from product variant.
