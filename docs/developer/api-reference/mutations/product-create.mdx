Creates a new product. Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productCreate(
  input: ProductCreateInput!
): ProductCreate
```

### Arguments

#### `input` ([`ProductCreateInput`](../../../developer/api-reference/inputs/product-create-input))

Fields required to create a product.

### Type

#### [`ProductCreate`](../../../developer/api-reference/objects/product-create)

Creates a new product. Requires one of the following permissions: MANAGE_PRODUCTS.
