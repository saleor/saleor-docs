### Possible types

#### [`App`](/developer/api-reference/objects/app.mdx)

Represents app data.

#### [`Address`](/developer/api-reference/objects/address.mdx)

Represents user address data.

#### [`User`](/developer/api-reference/objects/user.mdx)

Represents user data.

#### [`Group`](/developer/api-reference/objects/group.mdx)

Represents permission group data.

#### [`ProductVariant`](/developer/api-reference/objects/product-variant.mdx)

Represents a version of a product such as different size or color.

#### [`Product`](/developer/api-reference/objects/product.mdx)

Represents an individual item for sale in the storefront.

#### [`ProductType`](/developer/api-reference/objects/product-type.mdx)

Represents a type of product. It defines what attributes are available to products of this type.

#### [`Collection`](/developer/api-reference/objects/collection.mdx)

Represents a collection of products.

#### [`Category`](/developer/api-reference/objects/category.mdx)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.

#### [`ProductMedia`](/developer/api-reference/objects/product-media.mdx)

Represents a product media.

#### [`ProductImage`](/developer/api-reference/objects/product-image.mdx)

Represents a product image.

#### [`PageType`](/developer/api-reference/objects/page-type.mdx)

Represents a type of page. It defines what attributes are available to pages of this type.
