### Fields

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

The new name of the webhook.

#### `targetUrl` ([`String`](/docs/developer/api-reference/scalars/string))

The url to receive the payload.

#### `events` ([`[WebhookEventTypeEnum]`](/docs/developer/api-reference/enums/webhook-event-type-enum))

The events that webhook wants to subscribe.

#### `app` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the app to which webhook belongs.

#### `isActive` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determine if webhook will be set active or not.

#### `secretKey` ([`String`](/docs/developer/api-reference/scalars/string))

Use to create a hash signature with each payload.
