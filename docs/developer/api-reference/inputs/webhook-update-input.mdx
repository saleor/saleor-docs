```graphql
type WebhookUpdateInput {
  name: String
  targetUrl: String
  events: [WebhookEventTypeEnum]
  asyncEvents: [WebhookEventTypeAsyncEnum!]
  syncEvents: [WebhookEventTypeSyncEnum!]
  app: ID
  isActive: Boolean
  secretKey: String
}
```

### Fields

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

The new name of the webhook.

#### `targetUrl` ([`String`](../../../developer/api-reference/scalars/string))

The url to receive the payload.

#### `events` ([`[WebhookEventTypeEnum]`](../../../developer/api-reference/enums/webhook-event-type-enum))

The events that webhook wants to subscribe.

DEPRECATED: this field will be removed in Saleor 4.0. Use `asyncEvents` or `syncEvents` instead.

#### `asyncEvents` ([`[WebhookEventTypeAsyncEnum!]`](../../../developer/api-reference/enums/webhook-event-type-async-enum))

The asynchronous events that webhook wants to subscribe.

#### `syncEvents` ([`[WebhookEventTypeSyncEnum!]`](../../../developer/api-reference/enums/webhook-event-type-sync-enum))

The synchronous events that webhook wants to subscribe.

#### `app` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the app to which webhook belongs.

#### `isActive` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determine if webhook will be set active or not.

#### `secretKey` ([`String`](../../../developer/api-reference/scalars/string))

Use to create a hash signature with each payload.
