```graphql
input ProductChannelListingAddInput {
  channelId: ID!
  isPublished: Boolean
  publicationDate: Date
  publishedAt: DateTime
  visibleInListings: Boolean
  isAvailableForPurchase: Boolean
  availableForPurchaseDate: Date
  availableForPurchaseAt: DateTime
  addVariants: [ID!]
  removeVariants: [ID!]
}
```

### Fields

#### [`channelId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a channel.

#### [`isPublished`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determines if object is visible to customers.

#### [`publicationDate`](#) ([`Date`](../../../developer/api-reference/scalars/date))

Publication date. ISO 8601 standard.

DEPRECATED: this field will be removed in Saleor 4.0. Use `publishedAt` field instead.

#### [`publishedAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Publication date time. ISO 8601 standard.

Added in Saleor 3.3.

#### [`visibleInListings`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determines if product is visible in product listings (doesn't apply to product collections).

#### [`isAvailableForPurchase`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determine if product should be available for purchase.

#### [`availableForPurchaseDate`](#) ([`Date`](../../../developer/api-reference/scalars/date))

A start date from which a product will be available for purchase. When not set and isAvailable is set to True, the current day is assumed.

DEPRECATED: this field will be removed in Saleor 4.0. Use `availableForPurchaseAt` field instead.

#### [`availableForPurchaseAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

A start date time from which a product will be available for purchase. When not set and `isAvailable` is set to True, the current day is assumed.

Added in Saleor 3.3.

#### [`addVariants`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of variants to which the channel should be assigned.

#### [`removeVariants`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of variants from which the channel should be unassigned.
