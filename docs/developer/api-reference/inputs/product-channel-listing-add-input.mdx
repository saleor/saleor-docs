```graphql
type ProductChannelListingAddInput {
  channelId: ID!
  isPublished: Boolean
  publicationDate: Date
  visibleInListings: Boolean
  isAvailableForPurchase: Boolean
  availableForPurchaseDate: Date
  addVariants: [ID!]
  removeVariants: [ID!]
}
```

### Fields

#### `channelId` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of a channel.

#### `isPublished` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determines if object is visible to customers.

#### `publicationDate` ([`Date`](/docs/developer/api-reference/scalars/date))

Publication date. ISO 8601 standard.

#### `visibleInListings` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determines if product is visible in product listings (doesn't apply to product collections).

#### `isAvailableForPurchase` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determine if product should be available for purchase.

#### `availableForPurchaseDate` ([`Date`](/docs/developer/api-reference/scalars/date))

A start date from which a product will be available for purchase. When not set and isAvailable is set to True, the current day is assumed.

#### `addVariants` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of variants to which the channel should be assigned.

#### `removeVariants` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of variants from which the channel should be unassigned.
