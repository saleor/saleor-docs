Category to which item points.

#### `collection` ([`ID`](/docs/developer/api-reference/scalars/id))

Collection to which item points.

#### `page` ([`ID`](/docs/developer/api-reference/scalars/id))

Page to which item points.

#### `menu` ([`ID!`](/docs/developer/api-reference/scalars/id))

Menu to which item belongs.

#### `parent` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the parent menu. If empty, menu will be top level menu.
