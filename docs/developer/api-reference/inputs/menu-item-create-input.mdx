### Fields

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of the menu item.


#### `url` ([`String`](../../../developer/api-reference/scalars/string))

URL of the pointed item.


#### `category` ([`ID`](../../../developer/api-reference/scalars/id))

Category to which item points.


#### `collection` ([`ID`](../../../developer/api-reference/scalars/id))

Collection to which item points.


#### `page` ([`ID`](../../../developer/api-reference/scalars/id))

Page to which item points.


#### `menu` ([`ID`](../../../developer/api-reference/scalars/id))

Menu to which item belongs.


#### `parent` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the parent menu. If empty, menu will be top level menu.
