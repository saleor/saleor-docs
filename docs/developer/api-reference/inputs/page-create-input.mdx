```graphql
input PageCreateInput {
  slug: String
  title: String
  content: JSONString
  attributes: [AttributeValueInput!]
  isPublished: Boolean
  publicationDate: String
  publishedAt: DateTime
  seo: SeoInput
  pageType: ID!
}
```

### Fields

#### `slug` ([`String`](../../../developer/api-reference/scalars/string))

Page internal name.

#### `title` ([`String`](../../../developer/api-reference/scalars/string))

Page title.

#### `content` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Page content.

Rich text format. For reference see https://editorjs.io/

#### `attributes` ([`AttributeValueInput`](../../../developer/api-reference/inputs/attribute-value-input))

List of attributes.

#### `isPublished` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determines if page is visible in the storefront.

#### `publicationDate` ([`String`](../../../developer/api-reference/scalars/string))

Publication date. ISO 8601 standard.

DEPRECATED: this field will be removed in Saleor 4.0. Use `publishedAt` field instead.

#### `publishedAt` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Publication date time. ISO 8601 standard.

Added in Saleor 3.3.

#### `seo` ([`SeoInput`](../../../developer/api-reference/inputs/seo-input))

Search engine optimization fields.

#### `pageType` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the page type that page belongs to.
