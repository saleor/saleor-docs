### Fields

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel in which to create a checkout.

#### `lines` ([`[CheckoutLineInput]!`](/docs/developer/api-reference/inputs/checkout-line-input))

A list of checkout lines, each containing information about an item in the checkout.

#### `email` ([`String`](/docs/developer/api-reference/scalars/string))

The customer's email address.

#### `shippingAddress` ([`AddressInput`](/docs/developer/api-reference/inputs/address-input))

The mailing address to where the checkout will be shipped. Note: the address will be ignored if the checkout doesn't contain shippable items.

#### `billingAddress` ([`AddressInput`](/docs/developer/api-reference/inputs/address-input))

Billing address of the customer.

#### `languageCode` ([`LanguageCodeEnum`](/docs/developer/api-reference/enums/language-code-enum))

Checkout language code.
