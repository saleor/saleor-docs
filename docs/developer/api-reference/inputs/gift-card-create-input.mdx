```graphql
type GiftCardCreateInput {
  addTags: [String!]
  expiryDate: Date
  startDate: Date
  endDate: Date
  balance: PriceInput!
  userEmail: String
  channel: String
  isActive: Boolean!
  code: String
  note: String
}
```

### Fields

#### `addTags` ([`[String!]`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. The gift card tags to add. Note: this feature is in a preview state and can be subject to changes at later point.

#### `expiryDate` ([`Date`](../../../developer/api-reference/scalars/date))

New in Saleor 3.1. The gift card expiry date. Note: this feature is in a preview state and can be subject to changes at later point.

#### `startDate` ([`Date`](../../../developer/api-reference/scalars/date))

Start date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0.

#### `endDate` ([`Date`](../../../developer/api-reference/scalars/date))

End date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0. Use `expiryDate` from `expirySettings` instead.

#### `balance` ([`PriceInput!`](../../../developer/api-reference/inputs/price-input))

Balance of the gift card.

#### `userEmail` ([`String`](../../../developer/api-reference/scalars/string))

Email of the customer to whom gift card will be sent.

#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. Slug of a channel from which the email should be sent. Note: this feature is in a preview state and can be subject to changes at later point.

#### `isActive` ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

New in Saleor 3.1. Determine if gift card is active. Note: this feature is in a preview state and can be subject to changes at later point.

#### `code` ([`String`](../../../developer/api-reference/scalars/string))

Code to use the gift card.

DEPRECATED: this field will be removed in Saleor 4.0. The code is now auto generated.

#### `note` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. The gift card note from the staff member. Note: this feature is in a preview state and can be subject to changes at later point.
