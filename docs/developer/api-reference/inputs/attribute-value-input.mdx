No description

```graphql
type AttributeValueInput {
  id: ID
  values: [String!]
  file: String
  contentType: String
  references: [ID!]
  richText: JSONString
  boolean: Boolean
}
```

### Fields

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the selected attribute.

#### `values` ([`[String!]`](/developer/api-reference/scalars/string.mdx))

The value or slug of an attribute to resolve. If the passed value is non-existent, it will be created.

#### `file` ([`String`](/developer/api-reference/scalars/string.mdx))

URL of the file attribute. Every time, a new value is created.

#### `contentType` ([`String`](/developer/api-reference/scalars/string.mdx))

File content type.

#### `references` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of entity IDs that will be used as references.

#### `richText` ([`JSONString`](/developer/api-reference/scalars/json-string.mdx))

Text content in JSON format.

#### `boolean` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Represents the boolean value of the attribute value.
