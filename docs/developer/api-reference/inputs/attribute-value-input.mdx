```graphql
input AttributeValueInput {
  id: ID
  values: [String!]
  file: String
  contentType: String
  references: [ID!]
  richText: JSONString
  plainText: String
  boolean: Boolean
  date: Date
  dateTime: DateTime
}
```

### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the selected attribute.

#### `values` ([`String`](../../../developer/api-reference/scalars/string))

The value or slug of an attribute to resolve. If the passed value is non-existent, it will be created.

#### `file` ([`String`](../../../developer/api-reference/scalars/string))

URL of the file attribute. Every time, a new value is created.

#### `contentType` ([`String`](../../../developer/api-reference/scalars/string))

File content type.

#### `references` ([`ID`](../../../developer/api-reference/scalars/id))

List of entity IDs that will be used as references.

#### `richText` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Text content in JSON format.

#### `plainText` ([`String`](../../../developer/api-reference/scalars/string))

Plain text content.

#### `boolean` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Represents the boolean value of the attribute value.

#### `date` ([`Date`](../../../developer/api-reference/scalars/date))

Represents the date value of the attribute value.

#### `dateTime` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Represents the date/time value of the attribute value.
