```graphql
type AttributeValueInput {
  id: ID
  values: [String!]
  file: String
  contentType: String
  references: [ID!]
  richText: JSONString
  boolean: Boolean
  date: Date
  dateTime: DateTime
}
```

### Fields

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the selected attribute.

#### `values` ([`[String!]`](/docs/developer/api-reference/scalars/string))

The value or slug of an attribute to resolve. If the passed value is non-existent, it will be created.

#### `file` ([`String`](/docs/developer/api-reference/scalars/string))

URL of the file attribute. Every time, a new value is created.

#### `contentType` ([`String`](/docs/developer/api-reference/scalars/string))

File content type.

#### `references` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of entity IDs that will be used as references.

#### `richText` ([`JSONString`](/docs/developer/api-reference/scalars/json-string))

Text content in JSON format.

#### `boolean` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Represents the boolean value of the attribute value.

#### `date` ([`Date`](/docs/developer/api-reference/scalars/date))

Represents the date value of the attribute value.

#### `dateTime` ([`DateTime`](/docs/developer/api-reference/scalars/date-time))

Represents the date time value of the attribute value.
