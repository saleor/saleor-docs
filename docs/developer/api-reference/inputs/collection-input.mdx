No description

```graphql
input CollectionInput {
  isPublished: Boolean
  name: String
  slug: String
  description: JSONString
  backgroundImage: Upload
  backgroundImageAlt: String
  seo: SeoInput
  publicationDate: Date
}
```

### Fields

#### `isPublished` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Informs whether a collection is published.

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of the collection.

#### `slug` ([`String`](../../../developer/api-reference/scalars/string))

Slug of the collection.

#### `description` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Description of the collection.

Rich text format. For reference see https://editorjs.io/

#### `backgroundImage` ([`Upload`](../../../developer/api-reference/scalars/upload))

Background image file.

#### `backgroundImageAlt` ([`String`](../../../developer/api-reference/scalars/string))

Alt text for an image.

#### `seo` ([`SeoInput`](../../../developer/api-reference/inputs/seo-input))

Search engine optimization fields.

#### `publicationDate` ([`Date`](../../../developer/api-reference/scalars/date))

Publication date. ISO 8601 standard.

DEPRECATED: this field will be removed in Saleor 4.0.
