```graphql
type GiftCardUpdateInput {
  tag: String
  startDate: Date
  endDate: Date
  balanceAmount: PositiveDecimal
  expirySettings: GiftCardExpirySettingsInput
}
```

### Fields

#### `tag` ([`String`](/developer/api-reference/scalars/string.mdx))

New in Saleor 3.1. The gift card tag.

#### `startDate` ([`Date`](/developer/api-reference/scalars/date.mdx))

Start date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0.

#### `endDate` ([`Date`](/developer/api-reference/scalars/date.mdx))

End date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0. Use `expiryDate` from `expirySettings` instead.

#### `balanceAmount` ([`PositiveDecimal`](/developer/api-reference/scalars/positive-decimal.mdx))

New in Saleor 3.1. The gift card balance amount.

#### `expirySettings` ([`GiftCardExpirySettingsInput`](/developer/api-reference/inputs/gift-card-expiry-settings-input.mdx))

New in Saleor 3.1. The gift card expiry settings.
