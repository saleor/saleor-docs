No description

```graphql
type GiftCardUpdateInput {
  tag: String
  expiryDate: Date
  startDate: Date
  endDate: Date
  balanceAmount: PositiveDecimal
}
```

### Fields

#### `tag` ([`String`](../../../developer/api-reference/scalars/string))

New in Saleor 3.1. The gift card tag.

#### `expiryDate` ([`Date`](../../../developer/api-reference/scalars/date))

New in Saleor 3.1. The gift card expiry date.

#### `startDate` ([`Date`](../../../developer/api-reference/scalars/date))

Start date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0.

#### `endDate` ([`Date`](../../../developer/api-reference/scalars/date))

End date of the gift card in ISO 8601 format.

DEPRECATED: this field will be removed in Saleor 4.0. Use `expiryDate` from `expirySettings` instead.

#### `balanceAmount` ([`PositiveDecimal`](../../../developer/api-reference/scalars/positive-decimal))

New in Saleor 3.1. The gift card balance amount.
