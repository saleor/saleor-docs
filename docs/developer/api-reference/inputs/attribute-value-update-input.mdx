### Fields

#### `value` ([`String`](../../../developer/api-reference/scalars/string))

Represent value of the attribute value (e.g. color values for swatch attributes).

#### `richText` ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Represents the text of the attribute value, includes formatting.

Rich text format. For reference see https://editorjs.io/

#### `plainText` ([`String`](../../../developer/api-reference/scalars/string))

Represents the text of the attribute value, plain text without formating.

#### `fileUrl` ([`String`](../../../developer/api-reference/scalars/string))

URL of the file attribute. Every time, a new value is created.

#### `contentType` ([`String`](../../../developer/api-reference/scalars/string))

File content type.

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of a value displayed in the interface.
