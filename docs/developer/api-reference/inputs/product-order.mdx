
### Fields

#### `direction` ([`OrderDirection!`](/developer/api-reference/enums/order-direction.mdx))

Specifies the direction in which to sort products.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Specifies the channel in which to sort the data.

#### `attributeId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Sort product by the selected attribute's values.
Note: this doesn't take translations into account yet.

#### `field` ([`ProductOrderField`](/developer/api-reference/enums/product-order-field.mdx))

Sort products by the selected field.
