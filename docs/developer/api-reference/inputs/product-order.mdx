### Fields

#### `direction` ([`OrderDirection`](../../../developer/api-reference/enums/order-direction))

Specifies the direction in which to sort products.


#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Specifies the channel in which to sort the data.

DEPRECATED: this field will be removed in Saleor 4.0. Use root-level channel argument instead.


#### `attributeId` ([`ID`](../../../developer/api-reference/scalars/id))

Sort product by the selected attribute's values.
Note: this doesn't take translations into account yet.


#### `field` ([`ProductOrderField`](../../../developer/api-reference/enums/product-order-field))

Sort products by the selected field.
