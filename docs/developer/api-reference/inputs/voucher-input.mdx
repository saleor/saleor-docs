No description

```graphql
input VoucherInput {
  type: VoucherTypeEnum
  name: String
  code: String
  startDate: DateTime
  endDate: DateTime
  discountValueType: DiscountValueTypeEnum
  products: [ID!]
  variants: [ID!]
  collections: [ID!]
  categories: [ID!]
  minCheckoutItemsQuantity: Int
  countries: [String!]
  applyOncePerOrder: Boolean
  applyOncePerCustomer: Boolean
  onlyForStaff: Boolean
  usageLimit: Int
}
```

### Fields

#### `type` ([`VoucherTypeEnum`](../../../developer/api-reference/enums/voucher-type-enum))

Voucher type: PRODUCT, CATEGORY SHIPPING or ENTIRE_ORDER.

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Voucher name.

#### `code` ([`String`](../../../developer/api-reference/scalars/string))

Code to use the voucher.

#### `startDate` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Start date of the voucher in ISO 8601 format.

#### `endDate` ([`DateTime`](../../../developer/api-reference/scalars/date-time))

End date of the voucher in ISO 8601 format.

#### `discountValueType` ([`DiscountValueTypeEnum`](../../../developer/api-reference/enums/discount-value-type-enum))

Choices: fixed or percentage.

#### `products` ([`ID`](../../../developer/api-reference/scalars/id))

Products discounted by the voucher.

#### `variants` ([`ID`](../../../developer/api-reference/scalars/id))

New in Saleor 3.1. Variants discounted by the voucher.

#### `collections` ([`ID`](../../../developer/api-reference/scalars/id))

Collections discounted by the voucher.

#### `categories` ([`ID`](../../../developer/api-reference/scalars/id))

Categories discounted by the voucher.

#### `minCheckoutItemsQuantity` ([`Int`](../../../developer/api-reference/scalars/int))

Minimal quantity of checkout items required to apply the voucher.

#### `countries` ([`String`](../../../developer/api-reference/scalars/string))

Country codes that can be used with the shipping voucher.

#### `applyOncePerOrder` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Voucher should be applied to the cheapest item or entire order.

#### `applyOncePerCustomer` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Voucher should be applied once per customer.

#### `onlyForStaff` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Voucher can be used only by staff user.

#### `usageLimit` ([`Int`](../../../developer/api-reference/scalars/int))

Limit number of times this voucher can be used in total.
