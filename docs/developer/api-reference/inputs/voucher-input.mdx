```graphql
type VoucherInput {
  type: VoucherTypeEnum
  name: String
  code: String
  startDate: DateTime
  endDate: DateTime
  discountValueType: DiscountValueTypeEnum
  products: [ID]
  collections: [ID]
  categories: [ID]
  minCheckoutItemsQuantity: Int
  countries: [String]
  applyOncePerOrder: Boolean
  applyOncePerCustomer: Boolean
  onlyForStaff: Boolean
  usageLimit: Int
}
```

### Fields

#### `type` ([`VoucherTypeEnum`](/docs/developer/api-reference/enums/voucher-type-enum))

Voucher type: PRODUCT, CATEGORY SHIPPING or ENTIRE_ORDER.

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Voucher name.

#### `code` ([`String`](/docs/developer/api-reference/scalars/string))

Code to use the voucher.

#### `startDate` ([`DateTime`](/docs/developer/api-reference/scalars/date-time))

Start date of the voucher in ISO 8601 format.

#### `endDate` ([`DateTime`](/docs/developer/api-reference/scalars/date-time))

End date of the voucher in ISO 8601 format.

#### `discountValueType` ([`DiscountValueTypeEnum`](/docs/developer/api-reference/enums/discount-value-type-enum))

Choices: fixed or percentage.

#### `products` ([`[ID]`](/docs/developer/api-reference/scalars/id))

Products discounted by the voucher.

#### `collections` ([`[ID]`](/docs/developer/api-reference/scalars/id))

Collections discounted by the voucher.

#### `categories` ([`[ID]`](/docs/developer/api-reference/scalars/id))

Categories discounted by the voucher.

#### `minCheckoutItemsQuantity` ([`Int`](/docs/developer/api-reference/scalars/int))

Minimal quantity of checkout items required to apply the voucher.

#### `countries` ([`[String]`](/docs/developer/api-reference/scalars/string))

Country codes that can be used with the shipping voucher.

#### `applyOncePerOrder` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Voucher should be applied to the cheapest item or entire order.

#### `applyOncePerCustomer` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Voucher should be applied once per customer.

#### `onlyForStaff` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Voucher can be used only by staff user.

#### `usageLimit` ([`Int`](/docs/developer/api-reference/scalars/int))

Limit number of times this voucher can be used in total.
