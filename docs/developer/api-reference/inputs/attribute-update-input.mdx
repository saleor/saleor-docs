### Fields

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Name of an attribute displayed in the interface.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Internal representation of an attribute name.

#### `unit` ([`MeasurementUnitsEnum`](/docs/developer/api-reference/enums/measurement-units-enum))

The unit of attribute values.

#### `removeValues` ([`[ID]`](/docs/developer/api-reference/scalars/id))

IDs of values to be removed from this attribute.

#### `addValues` ([`[AttributeValueCreateInput]`](/docs/developer/api-reference/inputs/attribute-value-create-input))

New values to be created for this attribute.

#### `valueRequired` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute requires values to be passed or not.

#### `isVariantOnly` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute is for variants only.

#### `visibleInStorefront` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute should be visible or not in storefront.

#### `filterableInStorefront` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute can be filtered in storefront.

#### `filterableInDashboard` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute can be filtered in dashboard.

#### `storefrontSearchPosition` ([`Int`](/docs/developer/api-reference/scalars/int))

The position of the attribute in the storefront navigation (0 by default).

#### `availableInGrid` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Whether the attribute can be displayed in the admin product list.
