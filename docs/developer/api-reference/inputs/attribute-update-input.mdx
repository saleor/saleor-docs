### Fields

#### `name` ([`String`](../../../developer/api-reference/scalars/string))

Name of an attribute displayed in the interface.

#### `slug` ([`String`](../../../developer/api-reference/scalars/string))

Internal representation of an attribute name.

#### `unit` ([`MeasurementUnitsEnum`](../../../developer/api-reference/enums/measurement-units-enum))

The unit of attribute values.

#### `removeValues` ([`[ID!]`](../../../developer/api-reference/scalars/id))

IDs of values to be removed from this attribute.

#### `addValues` ([`[AttributeValueUpdateInput!]`](../../../developer/api-reference/inputs/attribute-value-update-input))

New values to be created for this attribute.

#### `valueRequired` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute requires values to be passed or not.

#### `isVariantOnly` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute is for variants only.

#### `visibleInStorefront` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute should be visible or not in storefront.

#### `filterableInStorefront` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute can be filtered in storefront.

#### `filterableInDashboard` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute can be filtered in dashboard.

#### `storefrontSearchPosition` ([`Int`](../../../developer/api-reference/scalars/int))

The position of the attribute in the storefront navigation (0 by default).

#### `availableInGrid` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Whether the attribute can be displayed in the admin product list.
