```graphql
input DigitalContentUploadInput {
```

### Fields

#### `useDefaultSettings` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Use default digital content settings for this product.


#### `maxDownloads` ([`Int`](../../../developer/api-reference/scalars/int))

Determines how many times a download link can be accessed by a customer.


#### `urlValidDays` ([`Int`](../../../developer/api-reference/scalars/int))

Determines for how many days a download link is active since it was generated.


#### `automaticFulfillment` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Overwrite default automatic_fulfillment setting for variant.


#### `contentFile` ([`Upload`](../../../developer/api-reference/scalars/upload))

Represents an file in a multipart request.
