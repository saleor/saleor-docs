```graphql
type DigitalContentUploadInput {
```

### Fields

#### `useDefaultSettings` ([`Boolean!`](/docs/developer/api-reference/scalars/boolean))

Use default digital content settings for this product.

#### `maxDownloads` ([`Int`](/docs/developer/api-reference/scalars/int))

Determines how many times a download link can be accessed by a customer.

#### `urlValidDays` ([`Int`](/docs/developer/api-reference/scalars/int))

Determines for how many days a download link is active since it was generated.

#### `automaticFulfillment` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Overwrite default automatic_fulfillment setting for variant.

#### `contentFile` ([`Upload!`](/docs/developer/api-reference/scalars/upload))

Represents an file in a multipart request.
