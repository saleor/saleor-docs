### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the attribute to assign.


#### `type` ([`ProductAttributeType`](../../../developer/api-reference/enums/product-attribute-type))

The attribute type to be assigned as.


#### `variantSelection` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

New in Saleor 3.1. Whether attribute is allowed in variant selection. Allowed types are: ['dropdown', 'boolean', 'swatch', 'numeric'].
