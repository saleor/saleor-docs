### Fields

#### `isActive` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

isActive flag.

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Name of the channel.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of the channel.

#### `defaultCountry` ([`CountryCode`](/docs/developer/api-reference/enums/country-code))

New in Saleor 3.1. Default country for the channel. Default country can be used in checkout to determine the stock quantities or calculate taxes when the country was not explicitly provided.

#### `addShippingZones` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of shipping zones to assign to the channel.

#### `removeShippingZones` ([`[ID!]`](/docs/developer/api-reference/scalars/id))

List of shipping zones to unassign from the channel.
