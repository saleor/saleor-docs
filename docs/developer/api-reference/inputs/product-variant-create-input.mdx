```graphql
input ProductVariantCreateInput {
  attributes: [AttributeValueInput!]!
  sku: String
  trackInventory: Boolean
  weight: WeightScalar
  preorder: PreorderSettingsInput
  quantityLimitPerCustomer: Int
  product: ID!
  stocks: [StockInput!]
}
```


### Fields

#### `attributes` ([`AttributeValueInput`](../../../developer/api-reference/inputs/attribute-value-input))

List of attributes specific to this variant.


#### `sku` ([`String`](../../../developer/api-reference/scalars/string))

Stock keeping unit.


#### `trackInventory` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determines if the inventory of this variant should be tracked. If false, the quantity won't change when customers buy this item.


#### `weight` ([`WeightScalar`](../../../developer/api-reference/scalars/weight-scalar))

Weight of the Product Variant.


#### `preorder` ([`PreorderSettingsInput`](../../../developer/api-reference/inputs/preorder-settings-input))

New in Saleor 3.1. Determines if variant is in preorder. Note: this feature is in a preview state and can be subject to changes at later point.


#### `quantityLimitPerCustomer` ([`Int`](../../../developer/api-reference/scalars/int))

New in Saleor 3.1. Determines maximum quantity of `ProductVariant`,that can be bought in a single checkout. Note: this feature is in a preview state and can be subject to changes at later point.


#### `product` ([`ID`](../../../developer/api-reference/scalars/id))

Product ID of which type is the variant.


#### `stocks` ([`StockInput`](../../../developer/api-reference/inputs/stock-input))

Stocks of a product available for sale.
