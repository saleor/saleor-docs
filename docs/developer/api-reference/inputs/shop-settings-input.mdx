### Fields

#### `headerText` ([`String`](../../../developer/api-reference/scalars/string))

Header text.


#### `description` ([`String`](../../../developer/api-reference/scalars/string))

SEO description.


#### `includeTaxesInPrices` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Include taxes in prices.


#### `displayGrossPrices` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Display prices with tax in store.


#### `chargeTaxesOnShipping` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Charge taxes on shipping.


#### `trackInventoryByDefault` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Enable inventory tracking.


#### `defaultWeightUnit` ([`WeightUnitsEnum`](../../../developer/api-reference/enums/weight-units-enum))

Default weight unit.


#### `automaticFulfillmentDigitalProducts` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Enable automatic fulfillment for all digital products.


#### `fulfillmentAutoApprove` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

New in Saleor 3.1. Enable automatic approval of all new fulfillments.


#### `fulfillmentAllowUnpaid` ([`Boolean`](../../../developer/api-reference/scalars/boolean))

New in Saleor 3.1. Enable ability to approve fulfillments which are unpaid.


#### `defaultDigitalMaxDownloads` ([`Int`](../../../developer/api-reference/scalars/int))

Default number of max downloads per digital content URL.


#### `defaultDigitalUrlValidDays` ([`Int`](../../../developer/api-reference/scalars/int))

Default number of days which digital content URL will be valid.


#### `defaultMailSenderName` ([`String`](../../../developer/api-reference/scalars/string))

Default email sender's name.


#### `defaultMailSenderAddress` ([`String`](../../../developer/api-reference/scalars/string))

Default email sender's address.


#### `customerSetPasswordUrl` ([`String`](../../../developer/api-reference/scalars/string))

URL of a view where customers can set their password.


#### `reserveStockDurationAnonymousUser` ([`Int`](../../../developer/api-reference/scalars/int))

New in Saleor 3.1. Default number of minutes stock will be reserved for anonymous checkout. Enter 0 or null to disable.


#### `reserveStockDurationAuthenticatedUser` ([`Int`](../../../developer/api-reference/scalars/int))

New in Saleor 3.1. Default number of minutes stock will be reserved for authenticated checkout. Enter 0 or null to disable.


#### `limitQuantityPerCheckout` ([`Int`](../../../developer/api-reference/scalars/int))

New in Saleor 3.1. Default number of maximum line quantity in single checkout. Minimum possible value is 1, default value is 50. Note: this feature is in a preview state and can be subject to changes at later point.
