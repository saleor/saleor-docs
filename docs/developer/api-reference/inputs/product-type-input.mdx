```graphql
type ProductTypeInput {
  name: String
  slug: String
  kind: ProductTypeKindEnum
  hasVariants: Boolean
  productAttributes: [ID]
  variantAttributes: [ID]
  isShippingRequired: Boolean
  isDigital: Boolean
  weight: WeightScalar
  taxCode: String
}
```

### Fields

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

Name of the product type.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Product type slug.

#### `kind` ([`ProductTypeKindEnum`](/docs/developer/api-reference/enums/product-type-kind-enum))

The product type kind.

#### `hasVariants` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determines if product of this type has multiple variants. This option mainly simplifies product management in the dashboard. There is always at least one variant created under the hood.

#### `productAttributes` ([`[ID]`](/docs/developer/api-reference/scalars/id))

List of attributes shared among all product variants.

#### `variantAttributes` ([`[ID]`](/docs/developer/api-reference/scalars/id))

List of attributes used to distinguish between different variants of a product.

#### `isShippingRequired` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determines if shipping is required for products of this variant.

#### `isDigital` ([`Boolean`](/docs/developer/api-reference/scalars/boolean))

Determines if products are digital.

#### `weight` ([`WeightScalar`](/docs/developer/api-reference/scalars/weight-scalar))

Weight of the ProductType items.

#### `taxCode` ([`String`](/docs/developer/api-reference/scalars/string))

Tax rate for enabled tax gateway.
