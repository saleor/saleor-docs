### Fields

#### `attributes` ([`[AttributeValueInput!]`](/developer/api-reference/inputs/attribute-value-input.mdx))

List of attributes specific to this variant.

#### `sku` ([`String`](/developer/api-reference/scalars/string.mdx))

Stock keeping unit.

#### `trackInventory` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Determines if the inventory of this variant should be tracked. If false, the quantity won't change when customers buy this item.

#### `weight` ([`WeightScalar`](/developer/api-reference/scalars/weight-scalar.mdx))

Weight of the Product Variant.
