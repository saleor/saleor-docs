```graphql
type ExternalNotificationTriggerInput {
  ids: [ID!]!
  extraPayload: JSONString
  externalEventType: String!
}
```

### Fields

#### `ids` ([`[ID!]!`](../../../developer/api-reference/scalars/id))

The list of customers or orders node IDs that will be serialized and included in the notification payload.

#### `extraPayload` ([`JSONString`](../../../developer/api-reference/scalars/json-string))

Additional payload that will be merged with the one based on the bussines object ID.

#### `externalEventType` ([`String!`](../../../developer/api-reference/scalars/string))

External event type. This field is passed to a plugin as an event type.
