Look up an app by ID. If ID is not provided, return the currently authenticated app.

```graphql
app(
  id: ID
): App

```

### Arguments

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the app.

### Type

#### [`App`](/docs/developer/api-reference/objects/app)

Represents app data.
