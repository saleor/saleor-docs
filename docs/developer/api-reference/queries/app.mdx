Look up an app by ID. If ID is not provided, return the currently authenticated app.

```graphql
app(
  id: ID = 0
): App

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the app.

### Type

#### [`App`](/developer/api-reference/objects/app.mdx)

Represents app data.
