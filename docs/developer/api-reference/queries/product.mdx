Look up a product by ID. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
product(
  id: ID
  slug: String
  channel: String
): Product
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the product.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of the product.

#### [`channel`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

### Type

#### [`Product`](../../../developer/api-reference/objects/product)

Represents an individual item for sale in the storefront.
