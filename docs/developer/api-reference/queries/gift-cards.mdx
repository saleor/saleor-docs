List of gift cards.

```graphql
giftCards(
  sortBy: GiftCardSortingInput
  filter: GiftCardFilterInput
  before: String
  after: String
  first: Int
  last: Int
): GiftCardCountableConnection

```

### Arguments

#### `sortBy` ([`GiftCardSortingInput`](/docs/developer/api-reference/inputs/gift-card-sorting-input))

New in Saleor 3.1. Sort gift cards.

#### `filter` ([`GiftCardFilterInput`](/docs/developer/api-reference/inputs/gift-card-filter-input))

New in Saleor 3.1. Filtering options for gift cards.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`GiftCardCountableConnection`](/docs/developer/api-reference/objects/gift-card-countable-connection)
