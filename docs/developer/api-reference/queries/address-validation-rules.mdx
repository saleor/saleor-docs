Returns address validation rules.

```graphql
addressValidationRules(
  countryCode: CountryCode!
  countryArea: String
  city: String
  cityArea: String
): AddressValidationData

```

### Arguments

#### `countryCode` ([`CountryCode!`](/docs/developer/api-reference/enums/country-code))

Two-letter ISO 3166-1 country code.

#### `countryArea` ([`String`](/docs/developer/api-reference/scalars/string))

Designation of a region, province or state.

#### `city` ([`String`](/docs/developer/api-reference/scalars/string))

City or a town name.

#### `cityArea` ([`String`](/docs/developer/api-reference/scalars/string))

Sublocality like a district.

### Type

#### [`AddressValidationData`](/docs/developer/api-reference/objects/address-validation-data)
