List of the shop's categories.

```graphql
categories(
  filter: CategoryFilterInput
  sortBy: CategorySortingInput
  level: Int
  before: String
  after: String
  first: Int
  last: Int
): CategoryCountableConnection

```

### Arguments

#### `filter` ([`CategoryFilterInput`](/docs/developer/api-reference/inputs/category-filter-input))

Filtering options for categories.

#### `sortBy` ([`CategorySortingInput`](/docs/developer/api-reference/inputs/category-sorting-input))

Sort categories.

#### `level` ([`Int`](/docs/developer/api-reference/scalars/int))

Filter categories by the nesting level in the category tree.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`CategoryCountableConnection`](/docs/developer/api-reference/objects/category-countable-connection)
