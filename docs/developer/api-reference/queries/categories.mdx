List of the shop's categories.

```graphql
categories(
  filter: CategoryFilterInput
  sortBy: CategorySortingInput
  level: Int
  before: String
  after: String
  first: Int
  last: Int
): CategoryCountableConnection
```


### Arguments

#### `filter` ([`CategoryFilterInput`](../../../developer/api-reference/inputs/category-filter-input))

Filtering options for categories.


#### `sortBy` ([`CategorySortingInput`](../../../developer/api-reference/inputs/category-sorting-input))

Sort categories.


#### `level` ([`Int`](../../../developer/api-reference/scalars/int))

Filter categories by the nesting level in the category tree.


#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.


#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.


#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.


#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.


### Type

#### [`CategoryCountableConnection`](../../../developer/api-reference/objects/category-countable-connection)
