Look up a payment by ID.

```graphql
payment(
  id: ID!
): Payment

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the payment.

### Type

#### [`Payment`](/docs/developer/api-reference/objects/payment)

Represents a payment of a given type.
