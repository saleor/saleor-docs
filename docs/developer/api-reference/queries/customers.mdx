List of the shop's customers.

```graphql
customers(
  filter: CustomerFilterInput
  sortBy: UserSortingInput
  before: String
  after: String
  first: Int
  last: Int
): UserCountableConnection

```

### Arguments

#### `filter` ([`CustomerFilterInput`](/docs/developer/api-reference/inputs/customer-filter-input))

Filtering options for customers.

#### `sortBy` ([`UserSortingInput`](/docs/developer/api-reference/inputs/user-sorting-input))

Sort customers.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`UserCountableConnection`](/docs/developer/api-reference/objects/user-countable-connection)
