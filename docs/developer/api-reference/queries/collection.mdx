Look up a collection by ID. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
collection(
  id: ID
  slug: String
  channel: String
): Collection
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the collection.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of the category

#### [`channel`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

### Type

#### [`Collection`](../../../developer/api-reference/objects/collection)

Represents a collection of products.
