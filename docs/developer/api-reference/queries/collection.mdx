Look up a collection by ID.

```graphql
collection(
  id: ID
  slug: String
  channel: String
): Collection

```

### Arguments

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the collection.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of the category

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

### Type

#### [`Collection`](/docs/developer/api-reference/objects/collection)

Represents a collection of products.
