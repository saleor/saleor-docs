Look up a webhook by ID. Requires one of the following permissions: MANAGE_APPS, OWNER.

```graphql
webhook(
  id: ID!
): Webhook
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the webhook.

### Type

#### [`Webhook`](../../../developer/api-reference/objects/webhook)

Webhook.
