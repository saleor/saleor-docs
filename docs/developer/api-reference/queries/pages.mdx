List of the shop's pages.

```graphql
pages(
  sortBy: PageSortingInput
  filter: PageFilterInput
  before: String
  after: String
  first: Int
  last: Int
): PageCountableConnection

```

### Arguments

#### `sortBy` ([`PageSortingInput`](/docs/developer/api-reference/inputs/page-sorting-input))

Sort pages.

#### `filter` ([`PageFilterInput`](/docs/developer/api-reference/inputs/page-filter-input))

Filtering options for pages.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`PageCountableConnection`](/docs/developer/api-reference/objects/page-countable-connection)
