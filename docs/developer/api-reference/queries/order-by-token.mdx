&#x007B;DEPRECATED_IN_3X_FIELD&#x007D; Look up an order by token.

```graphql
orderByToken(
  token: UUID!
): Order
```

### Arguments

#### `token` ([`UUID`](../../../developer/api-reference/scalars/uuid))

The order's token.

### Type

#### [`Order`](../../../developer/api-reference/objects/order)

Represents an order in the shop.
