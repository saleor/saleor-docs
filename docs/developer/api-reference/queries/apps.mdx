List of the apps.

Requires one of the following permissions: AUTHENTICATED_STAFF_USER, MANAGE_APPS.

```graphql
apps(
  filter: AppFilterInput
  sortBy: AppSortingInput
  before: String
  after: String
  first: Int
  last: Int
): AppCountableConnection
```

### Arguments

#### [`filter`](#) ([`AppFilterInput`](../../../developer/api-reference/inputs/app-filter-input))

Filtering options for apps.

#### [`sortBy`](#) ([`AppSortingInput`](../../../developer/api-reference/inputs/app-sorting-input))

Sort apps.

#### [`before`](#) ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### [`after`](#) ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### [`first`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### [`last`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`AppCountableConnection`](../../../developer/api-reference/objects/app-countable-connection)
