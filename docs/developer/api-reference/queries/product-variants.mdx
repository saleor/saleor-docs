List of product variants.

```graphql
productVariants(
  ids: [ID]
  channel: String
  filter: ProductVariantFilterInput
  sortBy: ProductVariantSortingInput
  before: String
  after: String
  first: Int
  last: Int
): ProductVariantCountableConnection

```

### Arguments

#### `ids` ([`[ID]`](../../../developer/api-reference/scalars/id))

Filter product variants by given IDs.

#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `filter` ([`ProductVariantFilterInput`](../../../developer/api-reference/inputs/product-variant-filter-input))

Filtering options for product variant.

#### `sortBy` ([`ProductVariantSortingInput`](../../../developer/api-reference/inputs/product-variant-sorting-input))

Sort products variants.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`ProductVariantCountableConnection`](../../../developer/api-reference/objects/product-variant-countable-connection)
