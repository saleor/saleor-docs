List of orders.

```graphql
orders(
  sortBy: OrderSortingInput
  filter: OrderFilterInput
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): OrderCountableConnection

```

### Arguments

#### `sortBy` ([`OrderSortingInput`](/docs/developer/api-reference/inputs/order-sorting-input))

Sort orders.

#### `filter` ([`OrderFilterInput`](/docs/developer/api-reference/inputs/order-filter-input))

Filtering options for orders.

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`OrderCountableConnection`](/docs/developer/api-reference/objects/order-countable-connection)
