Look up a product variant by ID or SKU.

```graphql
productVariant(
  id: ID
  sku: String
  channel: String
): ProductVariant

```

### Arguments

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the product variant.

#### `sku` ([`String`](/docs/developer/api-reference/scalars/string))

Sku of the product variant.

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

### Type

#### [`ProductVariant`](/docs/developer/api-reference/objects/product-variant)

Represents a version of a product such as different size or color.
