---

New in Saleor 3.1. Look up an app extension by ID. Note: this feature is in a preview state and can be subject to changes at later point.

```graphql
appExtension(
  id: ID!
): AppExtension
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the app extension.

### Type

#### [`AppExtension`](../../../developer/api-reference/objects/app-extension)

Represents app data.
