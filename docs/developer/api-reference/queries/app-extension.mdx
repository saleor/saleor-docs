---

New in Saleor 3.1. Look up an app extension by ID.

```graphql
appExtension(
  id: ID!
): AppExtension

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the app extension.

### Type

#### [`AppExtension`](/developer/api-reference/objects/app-extension.mdx)

Represents app data.
