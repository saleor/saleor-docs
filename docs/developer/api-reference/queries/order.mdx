Look up an order by ID.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
order(
  id: ID!
): Order
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of an order.

### Type

#### [`Order`](../../../developer/api-reference/objects/order)

Represents an order in the shop.
