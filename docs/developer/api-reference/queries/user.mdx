Look up a user by ID or email address.

Requires one of the following permissions: MANAGE_STAFF, MANAGE_USERS, MANAGE_ORDERS.

```graphql
user(
  id: ID
  email: String
): User
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the user.

#### `email` ([`String`](../../../developer/api-reference/scalars/string))

Email address of the user.

### Type

#### [`User`](../../../developer/api-reference/objects/user)

Represents user data.
