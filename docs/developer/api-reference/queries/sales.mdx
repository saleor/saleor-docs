List of the shop's sales.

```graphql
sales(
  filter: SaleFilterInput
  sortBy: SaleSortingInput
  query: String
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): SaleCountableConnection

```

### Arguments

#### `filter` ([`SaleFilterInput`](/docs/developer/api-reference/inputs/sale-filter-input))

Filtering options for sales.

#### `sortBy` ([`SaleSortingInput`](/docs/developer/api-reference/inputs/sale-sorting-input))

Sort sales.

#### `query` ([`String`](/docs/developer/api-reference/scalars/string))

Search sales by name, value or type.

DEPRECATED: this field will be removed in Saleor 4.0. Use `filter.search` input instead.

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`SaleCountableConnection`](/docs/developer/api-reference/objects/sale-countable-connection)
