List of the shop's sales.

```graphql
sales(
  filter: SaleFilterInput
  sortBy: SaleSortingInput
  query: String
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): SaleCountableConnection
```


### Arguments

#### `filter` ([`SaleFilterInput`](../../../developer/api-reference/inputs/sale-filter-input))

Filtering options for sales.


#### `sortBy` ([`SaleSortingInput`](../../../developer/api-reference/inputs/sale-sorting-input))

Sort sales.


#### `query` ([`String`](../../../developer/api-reference/scalars/string))

Search sales by name, value or type. 

DEPRECATED: this field will be removed in Saleor 4.0. Use `filter.search` input instead.


#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.


#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.


#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.


#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.


#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.


### Type

#### [`SaleCountableConnection`](../../../developer/api-reference/objects/sale-countable-connection)
