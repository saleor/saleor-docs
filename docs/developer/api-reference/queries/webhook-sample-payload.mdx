title: webhookSamplePayload
---

Retrieve a sample payload for a given webhook event based on real data. It can be useful for some integrations where sample payload is required.

```graphql
webhookSamplePayload(
  eventType: WebhookSampleEventTypeEnum!
): JSONString

```

### Arguments

#### `eventType` ([`WebhookSampleEventTypeEnum!`](../../../developer/api-reference/enums/webhook-sample-event-type-enum))

Name of the requested event type.

### Type

#### [`JSONString`](../../../developer/api-reference/scalars/json-string)
