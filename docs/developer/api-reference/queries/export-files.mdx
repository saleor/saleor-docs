List of export files.

```graphql
exportFiles(
  filter: ExportFileFilterInput
  sortBy: ExportFileSortingInput
  before: String
  after: String
  first: Int
  last: Int
): ExportFileCountableConnection

```

### Arguments

#### `filter` ([`ExportFileFilterInput`](/docs/developer/api-reference/inputs/export-file-filter-input))

Filtering options for export files.

#### `sortBy` ([`ExportFileSortingInput`](/docs/developer/api-reference/inputs/export-file-sorting-input))

Sort export files.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`ExportFileCountableConnection`](/docs/developer/api-reference/objects/export-file-countable-connection)
