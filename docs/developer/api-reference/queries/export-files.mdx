List of export files.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
exportFiles(
  filter: ExportFileFilterInput
  sortBy: ExportFileSortingInput
  before: String
  after: String
  first: Int
  last: Int
): ExportFileCountableConnection
```

### Arguments

#### `filter` ([`ExportFileFilterInput`](../../../developer/api-reference/inputs/export-file-filter-input))

Filtering options for export files.

#### `sortBy` ([`ExportFileSortingInput`](../../../developer/api-reference/inputs/export-file-sorting-input))

Sort export files.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`ExportFileCountableConnection`](../../../developer/api-reference/objects/export-file-countable-connection)
