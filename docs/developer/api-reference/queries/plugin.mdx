Look up a plugin by ID.

Requires one of the following permissions: MANAGE_PLUGINS.

```graphql
plugin(
  id: ID!
): Plugin
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the plugin.

### Type

#### [`Plugin`](../../../developer/api-reference/objects/plugin)

Plugin.
