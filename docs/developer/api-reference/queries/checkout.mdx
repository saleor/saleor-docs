Look up a checkout by token and slug of channel.

```graphql
checkout(
  token: UUID
): Checkout

```

### Arguments

#### `token` ([`UUID`](/docs/developer/api-reference/scalars/uuid))

The checkout's token.

### Type

#### [`Checkout`](/docs/developer/api-reference/objects/checkout)

Checkout object.
