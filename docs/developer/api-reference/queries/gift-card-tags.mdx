List of gift card tags.

Added in Saleor 3.1.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: MANAGE_GIFT_CARD.

```graphql
giftCardTags(
  filter: GiftCardTagFilterInput
  before: String
  after: String
  first: Int
  last: Int
): GiftCardTagCountableConnection
```

### Arguments

#### `filter` ([`GiftCardTagFilterInput`](../../../developer/api-reference/inputs/gift-card-tag-filter-input))

Filtering options for gift card tags.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`GiftCardTagCountableConnection`](../../../developer/api-reference/objects/gift-card-tag-countable-connection)
