Look up an attribute by ID.

```graphql
attribute(
  id: ID
  slug: String
): Attribute
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the attribute.

#### `slug` ([`String`](../../../developer/api-reference/scalars/string))

Slug of the attribute.

### Type

#### [`Attribute`](../../../developer/api-reference/objects/attribute)

Custom attribute of a product. Attributes can be assigned to products and variants at the product type level.
