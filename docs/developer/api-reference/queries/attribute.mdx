Look up an attribute by ID.

```graphql
attribute(
  id: ID
  slug: String
): Attribute

```

### Arguments

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the attribute.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of the attribute.

### Type

#### [`Attribute`](/docs/developer/api-reference/objects/attribute)

Custom attribute of a product. Attributes can be assigned to products and variants at the product type level.
