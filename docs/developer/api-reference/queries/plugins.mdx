List of plugins.

Requires one of the following permissions: MANAGE_PLUGINS.

```graphql
plugins(
  filter: PluginFilterInput
  sortBy: PluginSortingInput
  before: String
  after: String
  first: Int
  last: Int
): PluginCountableConnection
```

### Arguments

#### `filter` ([`PluginFilterInput`](../../../developer/api-reference/inputs/plugin-filter-input))

Filtering options for plugins.

#### `sortBy` ([`PluginSortingInput`](../../../developer/api-reference/inputs/plugin-sorting-input))

Sort plugins.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`PluginCountableConnection`](../../../developer/api-reference/objects/plugin-countable-connection)
