List of plugins.

```graphql
plugins(
  filter: PluginFilterInput
  sortBy: PluginSortingInput
  before: String
  after: String
  first: Int
  last: Int
): PluginCountableConnection

```

### Arguments

#### `filter` ([`PluginFilterInput`](/docs/developer/api-reference/inputs/plugin-filter-input))

Filtering options for plugins.

#### `sortBy` ([`PluginSortingInput`](/docs/developer/api-reference/inputs/plugin-sorting-input))

Sort plugins.

#### `before` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/docs/developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](/docs/developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`PluginCountableConnection`](/docs/developer/api-reference/objects/plugin-countable-connection)
