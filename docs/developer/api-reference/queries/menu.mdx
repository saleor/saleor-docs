Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID
  name: String
  slug: String
): Menu

```

### Arguments

#### `channel` ([`String`](/docs/developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `id` ([`ID`](/docs/developer/api-reference/scalars/id))

ID of the menu.

#### `name` ([`String`](/docs/developer/api-reference/scalars/string))

The menu's name.

#### `slug` ([`String`](/docs/developer/api-reference/scalars/string))

The menu's slug.

### Type

#### [`Menu`](/docs/developer/api-reference/objects/menu)

Represents a single menu - an object that is used to help navigate through the store.
