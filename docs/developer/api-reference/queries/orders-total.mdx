Return the total sales amount from a specific period.

```graphql
ordersTotal(
  period: ReportingPeriod
  channel: String
): TaxedMoney

```

### Arguments

#### `period` ([`ReportingPeriod`](/developer/api-reference/enums/reporting-period.mdx))

A period of time.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

### Type

#### [`TaxedMoney`](/developer/api-reference/objects/taxed-money.mdx)

Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.
