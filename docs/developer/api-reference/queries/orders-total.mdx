Return the total sales amount from a specific period.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
ordersTotal(
  period: ReportingPeriod
  channel: String
): TaxedMoney
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>ordersTotal.<b>period</b></code>](#)<Bullet />[`ReportingPeriod`](../../../developer/api-reference/enums/reporting-period) <Badge class="secondary" text="enum"/>
> A period of time.
> 

#### [<code style={{ fontWeight: 'normal' }}>ordersTotal.<b>channel</b></code>](#)<Bullet />[`String`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="scalar"/>
> Slug of a channel for which the data should be returned.
> 

### Type

#### [`TaxedMoney`](../../../developer/api-reference/objects/taxed-money) <Badge class="secondary" text="object"/>
> Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.
>
