Look up a export file by ID.

```graphql
exportFile(
  id: ID!
): ExportFile

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the export file job.

### Type

#### [`ExportFile`](/docs/developer/api-reference/objects/export-file)

Represents a job data of exported file.
