List of the shop's collections. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
collections(
  filter: CollectionFilterInput
  sortBy: CollectionSortingInput
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): CollectionCountableConnection
```

### Arguments

#### [`filter`](#) ([`CollectionFilterInput`](../../../developer/api-reference/inputs/collection-filter-input))

Filtering options for collections.

#### [`sortBy`](#) ([`CollectionSortingInput`](../../../developer/api-reference/inputs/collection-sorting-input))

Sort collections.

#### [`channel`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### [`before`](#) ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### [`after`](#) ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### [`first`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### [`last`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

#### [`CollectionCountableConnection`](../../../developer/api-reference/objects/collection-countable-connection)
