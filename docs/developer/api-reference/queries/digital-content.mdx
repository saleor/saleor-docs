Look up digital content by ID.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
digitalContent(
  id: ID!
): DigitalContent
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the digital content.

### Type

#### [`DigitalContent`](../../../developer/api-reference/objects/digital-content)
