Look up permission group by ID.

Requires one of the following permissions: MANAGE_STAFF.

```graphql
permissionGroup(
  id: ID!
): Group
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the group.

### Type

#### [`Group`](../../../developer/api-reference/objects/group)

Represents permission group data.
