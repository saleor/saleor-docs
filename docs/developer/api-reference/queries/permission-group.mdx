Look up permission group by ID.

```graphql
permissionGroup(
  id: ID!
): Group

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of the group.

### Type

#### [`Group`](/docs/developer/api-reference/objects/group)

Represents permission group data.
