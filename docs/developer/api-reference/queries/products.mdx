List of the shop's products. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
products(
  filter: ProductFilterInput
  sortBy: ProductOrder
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): ProductCountableConnection
```

### Arguments

#### `filter` ([`ProductFilterInput`](../../../developer/api-reference/inputs/product-filter-input))

Filtering options for products.

#### `sortBy` ([`ProductOrder`](../../../developer/api-reference/inputs/product-order))

Sort products.

#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`ProductCountableConnection`](../../../developer/api-reference/objects/product-countable-connection)
