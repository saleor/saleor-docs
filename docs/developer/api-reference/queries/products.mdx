List of the shop's products.

```graphql
products(
  filter: ProductFilterInput
  sortBy: ProductOrder
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): ProductCountableConnection

```

### Arguments

#### `filter` ([`ProductFilterInput`](/developer/api-reference/inputs/product-filter-input.mdx))

Filtering options for products.

#### `sortBy` ([`ProductOrder`](/developer/api-reference/inputs/product-order.mdx))

Sort products.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`ProductCountableConnection`](/developer/api-reference/objects/product-countable-connection.mdx)
