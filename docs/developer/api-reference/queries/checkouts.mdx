List of checkouts.

Requires one of the following permissions: MANAGE_CHECKOUTS.

```graphql
checkouts(
  sortBy: CheckoutSortingInput
  filter: CheckoutFilterInput
  channel: String
  before: String
  after: String
  first: Int
  last: Int
): CheckoutCountableConnection
```

### Arguments

#### `sortBy` ([`CheckoutSortingInput`](../../../developer/api-reference/inputs/checkout-sorting-input))

Sort checkouts.

Added in Saleor 3.1.

#### `filter` ([`CheckoutFilterInput`](../../../developer/api-reference/inputs/checkout-filter-input))

Filtering options for checkouts.

Added in Saleor 3.1.

#### `channel` ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](../../../developer/api-reference/scalars/string))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](../../../developer/api-reference/scalars/int))

Return the first n elements from the list.

#### `last` ([`Int`](../../../developer/api-reference/scalars/int))

Return the last n elements from the list.

### Type

#### [`CheckoutCountableConnection`](../../../developer/api-reference/objects/checkout-countable-connection)
