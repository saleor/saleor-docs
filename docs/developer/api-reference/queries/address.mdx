Look up an address by ID.

```graphql
address(
  id: ID!
): Address

```

### Arguments

#### `id` ([`ID!`](/docs/developer/api-reference/scalars/id))

ID of an address.

### Type

#### [`Address`](/docs/developer/api-reference/objects/address)

Represents user address data.
