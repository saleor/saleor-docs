Lookup a translatable item by ID.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
translation(
  id: ID!
  kind: TranslatableKinds!
): TranslatableItem
```

### Arguments

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

ID of the object to retrieve.

#### `kind` ([`TranslatableKinds`](../../../developer/api-reference/enums/translatable-kinds))

Kind of the object to retrieve.

### Type

#### [`TranslatableItem`](../../../developer/api-reference/unions/translatable-item)
