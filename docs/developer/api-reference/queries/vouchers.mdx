List of the shop's vouchers.

```graphql
vouchers(
  filter: VoucherFilterInput
  sortBy: VoucherSortingInput
  query: String
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): VoucherCountableConnection

```

### Arguments

#### `filter` ([`VoucherFilterInput`](/developer/api-reference/inputs/voucher-filter-input.mdx))

Filtering options for vouchers.

#### `sortBy` ([`VoucherSortingInput`](/developer/api-reference/inputs/voucher-sorting-input.mdx))

Sort voucher.

#### `query` ([`String`](/developer/api-reference/scalars/string.mdx))

Search vouchers by name or code.

DEPRECATED: this field will be removed in Saleor 4.0. Use `filter.search` input instead.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`VoucherCountableConnection`](/developer/api-reference/objects/voucher-countable-connection.mdx)
