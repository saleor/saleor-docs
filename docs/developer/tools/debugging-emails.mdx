[Saleor platform](https://github.com/mirumee/saleor-platform) use [Mailhog](https://github.com/mailhog/MailHog) for catching all the emails send by the Saleor. 

![Assign product to collection](../screenshots/mailhog.png)

Pros of using Mailhog during development:
- live preview of rendered emails.
- no need to send emails to external services.
- test emails will be caught in the local environment, so there is no possibility of sending test data to actual customers.

## Configuration

### Saleor Platform

No need for additional configuration; everything works out of the box.

### Local development environment

1. Follow [the installation guide](https://github.com/mailhog/MailHog#installation).
2. Set [SMTP server](developer/running-saleor/configuration.mdx#email_url) for Saleor.

## Access to the interface

Visit [http://localhost:8025](http://localhost:8025).
