## Spectrum

Join our [Spectrum community](https://spectrum.chat/saleor) to ask questions and propose features.

## Stack Overflow

You can ask programming-related questions on [Stack Overflow](https://stackoverflow.com/questions/tagged/saleor). Use the `saleor` tag to tag your questions.

## GitHub

Use the GitHub projects to report bugs:

- [Saleor Core](https://github.com/mirumee/saleor/issues)
- [Saleor Dashboard](https://github.com/mirumee/saleor-dashboard/issues)
- [Saleor Storefront](https://github.com/mirumee/saleor-storefront/issues)

:::note
Please don't use GitHub issues to ask questions.
:::
